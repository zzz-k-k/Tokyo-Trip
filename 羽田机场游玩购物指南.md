# 羽田机场（HND T3 国际航站楼）游玩购物指南

**航班锚点**
- **去程**：HO1385 / 6/2（周二）PVG 18:40 起飞 → **HND T3 22:30 落地**
- **返程**：HO1386 / 6/10（周三）**HND T3 00:05 起飞** → PVG 02:15

**关键事实**
- HND T3（国际航站楼）整栋楼 **24 小时开放**（与 T1 / T2 仅 05:00–翌 00:00 开放不同）。
- 但 **大多数店铺并非 24 小时**：陆侧（出关前的 1F–5F）大部分商业层在 22:00 / 22:30 关门，深夜段仅剩便利店 / 24h 餐饮 / 自动售货机；空侧（出境后的免税区）则有少数旗舰店与 TIAT DUTY FREE 系开到 00:30 甚至 24h。
- 因此本指南把可用时段切为两个截然不同的窗口：① **6/2 抵达后 22:30 → 翌 02:00**（陆侧深夜，"勉强还在营业"逻辑）；② **6/9 入境后 21:00 → 6/10 00:00 登机**（空侧出境后，"免税末班车"逻辑）。

**配套文档**：本目录另有一份 [`免税香烟品鉴指南.md`](./免税香烟品鉴指南.md)，专门展开 TIAT DUTY FREE 在售的全部香烟品牌的品牌故事 / 哲学 / 口感参数。

## 二、第一段：6/2（周二）22:30 落地 → 离开机场（陆侧深夜窗口）

### 2.1 抵达流程（22:30 落地 → 23:15 出闸）

落地 → 廊桥 → **2F 入境检查（Immigration）**：22:30 后段无大队，预计 10–20 分钟过关；走 **Visit Japan Web** 二维码通道更快。
↓
**2F 入境免税店区域**（**已过海关前最后一次"日本免税"机会，本日属"入境免税"非"出境免税"**——但本次行程正在入境而非出境，**不必采购**）。
↓
**2F 行李提取**（Baggage Claim）：22:30 国际航班行李一般 22:50–23:10 之间到位。
↓
**2F 海关 / 申报**：Visit Japan Web 提前申报 → 通常 5 分钟内出闸。
↓
**2F 入境大厅（Arrival Lobby）**：本指南第二段陆侧店铺所在区域。

## 三、第二段：6/9（周二）入境前后 → 6/10 00:05 起飞（空侧免税核心窗口）

**本段是本次 7 日行程的"购物收尾日"，所有想在日本买的免税商品（化妆品 / 香水 / 烟酒 / 食品 / 表 / 饰品 / 服装）的最后机会**。空侧的 TIAT DUTY FREE CENTRAL 是 24h 营业、SOUTH LIQUOR & TOBACCO 营业至 00:30，因此即使 22:00 才过完安检也仍有充裕时间完成核心采购。

### 3.1 6/9 抵达 / 出境时间倒推

```
航班 HO1386 起飞 00:05 / 登机口截止 23:35 / 推荐过安检 20:30 前 / 推荐抵 T3 19:30 前

19:00 — 离开两国 COGO RYOGOKU（傍晚段）
        都营浅草线 → 京急本线 直通运行（两国 → 浅草 → 押上 → 羽田机场，约 50 分钟、¥600–800）
19:50 — 抵达 HND T3 站（B1F）→ 上 3F 出境大厅
20:00 — 春秋航空 HO1386 值机柜台 / 行李托运（开柜约航班前 3 小时 = 21:00；春秋通常允许提前到柜——若柜台未开请先吃饭）
20:30 — 完成值机 + 行李托运
20:35 — 排队进入安检 + 出境检查（19:30 排队较 22:30 高峰快很多，预计 15–25 分钟）
21:00 — 抵达空侧 → 进入"免税购物 + 用晚餐 + 登机口待机"段
21:00–22:30 — 空侧购物核心 90 分钟（本指南第 3.3 节）
22:30–23:30 — 晚餐（航班无餐食，自费晚餐）+ 在 TIAT DUTY FREE CENTRAL / SOUTH LIQUOR & TOBACCO 末段补单
23:30 — 抵达登机口
00:05 — 起飞
```

**若到达 T3 时点偏晚**：① 19:50 → 推迟到 20:30 抵达 = 仍可完成全部采购，仅压缩晚餐时间到 30 分钟；② 21:30 抵达 = 跳过晚餐 + 跳过品牌精品（CHANEL / HERMES 等 22:00 已关），只能进 TIAT DUTY FREE 系做核心烟酒 / 化妆品采购；③ ≥ 22:00 抵达 = 极不推荐，已经赶不上品牌精品 + 安检高峰排队风险高。

## 四、免税购物实务要点

### 4.1 中国海关入境免税烟酒额度（HO1386 上海浦东 PVG 入境）

| 类目 | 个人免税额度 | 超额处理 |
| --- | --- | --- |
| **香烟** | **400 支（= 2 条 / 20 包）** | 超过部分按申报征税；超过 2 条不可托运分摊（同一旅客单次 400 支硬上限） |
| **雪茄** | 100 支 | 同上 |
| **烟丝** | 500 克 | 同上 |
| **烈酒（≥ 12% Vol）** | 1500 ml（= 1500 毫升 = 2 瓶 750 ml 或 1 瓶 1.5 L） | 超过部分按申报征税 |
| **总免税商品价值** | **5000 元人民币以下**（含烟酒之外所有自用商品；烟酒按上面单独额度计算） | 超过部分按申报征税 |

**关键规则**（中国海关 / 2024 起执行版）：
- **来自港澳台 / 邮寄 / 入境后 24 小时内复出境者**：烟酒减半（200 支烟 / 750 ml 酒）；本次 PVG 直飞回国按全额执行。
- **烟酒额度**与"5000 元自用商品总价值"**分别计算**——也就是说，2 条烟（约 ¥10,000 = 约 ¥500–700 元 RMB）＋ 1 瓶威士忌不算入 5000 元额度。
- **400 支 = 2 条 = 20 包**——TIAT DUTY FREE 多数香烟以"1 条 = 200 支 = 10 包"出售，故"2 条上限"对应直观清楚。
- 入境后红绿通道：**< 5000 元自用 + 烟酒额度内 → 走绿通道；≥ 5000 元 + 任何超额 → 走红通道申报**。

### 4.3 网络预约 vs 现场购买决策矩阵

| 商品类目 | 推荐方式 | 理由 |
| --- | --- | --- |
| **香烟（已知品牌 + 已知支数）** | **网络预约** | 价格便宜 5–10%、锁定库存（夜间航班易出现部分品牌断货）、节省排队 |
| **化妆品（已知 SKU）** | **网络预约** | 同上；许多日系限定（资生堂海蓝之谜 / Clé de Peau / SK-II）有"网络预约价 + 赠品"双重优惠 |
| **烈酒（已知酒款）** | **网络预约**（CENTRAL 取货）或 **现场 SOUTH LIQUOR & TOBACCO**（如想现场试品） | 山崎 / 白州 / 響 等日威经常缺货——网络预约可优先锁定 |
| **食品 / 手信（口感 / 场景试看）** | **现场** | 想现场看新品 + 口味试样 |
| **品牌精品（CHANEL / HERMES 等）** | **现场** | 大多数品牌精品不在 TIAT 网络预约目录内 |

### 4.4 支付方式

- **现金（JPY）**：全店通用
- **信用卡**：Visa / Mastercard / JCB / AmEx / Diners / UnionPay 主流通用
- **移动支付**：部分店铺支持 **Alipay / WeChat Pay / Apple Pay**（按店家告示）
- **离境退税**：免税店出售的"免税商品"已扣除消费税 + 关税，**无需额外退税办理**——出境时凭登机牌 + 护照在出口处可能被抽查

### 4.5 重要规则与注意事项

1. **液体免税额度**：免税店出售的酒类（≥ 100 ml）会装入 **STEB（Security Tamper-Evident Bag，统一密封袋）**，在 PVG 转机或最终入境前**禁止打开**——若中途有第三国转机，需确认 STEB 在该国转机机场有效（HO1386 直飞 PVG 不存在此问题）。
2. **登机前核对免税票根**：购买后店家会将免税商品装袋 + 给购物票根；登机口前可能被抽查（罕见）。
3. **离境前 1.5 小时不可购买**：部分免税商店（特别是 SOUTH LIQUOR & TOBACCO 烟酒店）按 IATA 规则在航班起飞前 1 小时停止销售酒类（22:35 后不可买酒）；**香烟无此限制**。
4. **抽烟后入境**：在中国 PVG 入境时，香烟若超过 400 支会被海关在红通道征税或没收；**严守 2 条上限**。
5. **航班无餐食**：HO1386 是廉航 / 春秋系，机上需付费购买；建议在 7-Eleven 买便当 / 三明治带上机（**便当可上机但不可分装液体超过 100 ml**）。
