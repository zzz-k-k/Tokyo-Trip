# 第一部分：日本本土品牌（JT 系，6 个）

JT（Japan Tobacco / 日本たばこ産業株式会社）由 1985 年公社民营化时成立，承袭了 1898 年日本"葉煙草専売法"以来近一个世纪的国营烟草专卖体系，是全球第三大烟草集团（次于 PMI 与 BAT），日本本土市场份额长期 ≥ 60%。在 TIAT DUTY FREE 出售的 6 个 JT 日产品牌中，**MEVIUS / SEVEN STARS / PEACE / HOPE** 构成"国民四大基石"，PIANISSIMO 与 MI-NE 是细分市场旗舰。

JT 系日产烟的共同特点：① 以 **Charcoal Filter（活性炭过滤嘴）** 为标志性技术（SEVEN STARS 1969 年首创）；② 烟叶以"日本国产烤烟（黄烟）+ 美式焙烤的轻度结合"为底味——比纯美式（Marlboro / Camel）干净、比纯英式（Dunhill / 555）温润；③ 包装设计倾向"克制 + 几何"——避免欧美品牌惯用的徽章 / 纹章感，强调字体与色块的现代主义构图。

## 1. MEVIUS（メビウス / 旧 Mild Seven）

### 基本参数
| 项目 | 内容 |
| --- | --- |
| **所属** | Japan Tobacco（JT） |
| **创立** | 1977 年 6 月 15 日（旧名 Mild Seven 时） / 2013 年 2 月起改名 Mevius |
| **产地** | 日本（神奈川 / 福岛 / 宫城 工厂为主） |
| **焦油 / 尼古丁** | 主流变体覆盖：1mg/0.1mg、3mg/0.3mg、5mg/0.4mg、6mg/0.5mg、8mg/0.6mg、10mg/0.8mg；最经典的 **Mevius Original（Soft / Box）= 10mg / 0.8mg** |
| **过滤嘴** | **活性炭三层过滤嘴**（Charcoal Filter），1977 年作为日本首款"美式混合配方 + 活性炭"组合产品上市 |
| **主要规格** | KS（85mm）/ Super Slim / 100s；Box 与 Soft Pack；普通 + 薄荷 + 限定（樱花 / 黑樱桃 / 蓝莓）系列 |

### 品牌故事
Mild Seven 于 **1977 年**作为 1969 年上市的国民王牌 "Seven Stars"（SVS / セブンスター）的"轻量版"推出。品牌名的"Mild"来自"温和（mild）"——主打入口比 Seven Stars 干净，"Seven"则承袭母品牌；上市第二年即成为日本市场销量第一，1981 年作为出口版进入海外市场，到 1980 年代末成为**全球销量最高的烟草品牌**之一（与 Marlboro 长期并列前三）。它是日本第一款采用**美式混合配方 + 活性炭过滤嘴**组合的香烟，这种工艺让美式烟原本的甜腻被活性炭过滤后变得清爽，是 1970s–1990s 一代日本"轻烟时代"的真正定义者。

**2013 年 2 月**，JT 因 WHO 烟草框架公约 + 欧盟新规禁止使用"Mild / Light / Low"等可能让消费者误解为"减害"的字样，决定全球同步把 Mild Seven 改名为 **Mevius**——新名取自拉丁语"meus"（"我的"）+ "vius"（"路 / 道路"）的合成，意为"我的道路"，由著名设计公司 Pentagram 与日本本土平面设计团队共同操刀。改名后保留了原配方与几乎所有视觉元素（蓝白配色、抽象波浪线 logo），只是把字母从"Mild Seven"换成"Mevius"——这是当代品牌史上最被研究的"非毁灭式更名"案例之一。

### 烟叶配方与口感档案
- **配方**：美式混合（American Blend），以美国 Burley + Virginia 烤烟为主，混入少量日本国产黄烟、土耳其烟叶（Oriental），三层混合压制后切丝
- **入口**：极轻盈的甜烟焦油气，几乎闻不到生烟草味；活性炭层会把入口的"生味"压低到几乎察觉不到
- **中段**：微糖 + 烤焙香（类似浅烤咖啡的香气），烟雾呈中等密度、不刺喉
- **余韵**：5–10 秒内消散，留下一点点甜后味，不挂喉
- **代表性变体**：
  - **Mevius Original（Box）10mg**：经典款，平衡甜度与烤焙
  - **Mevius Option Yellow / Purple（爆珠系）**：滤嘴含一颗压破式果味胶囊（柠檬 / 黑加仑 / 葡萄等），按一下出薄荷或果香——是 2010s 年代日本 Mevius 系列最突出的设计创新
  - **Mevius Premium Menthol Option**：薄荷爆珠 + 浓厚薄荷烟叶，是日本薄荷烟基准
  - **Mevius E Series**：1mg / 3mg 超低焦油变体，加倍活性炭

### 设计与哲学
- **包装**：底色"Mevius Blue"（介于群青与深蓝之间）+ 白底 logo + 一道横向白色波纹（象征"道路 / 流动"），由 Pentagram 重新设计
- **核心哲学**："**Be Yourself, Anytime, Anywhere**"——2013 年改名时确立的品牌口号，强调"在任何时候做自己"的当代个人主义；其本质上是对 1970s 那种"集体上班族吸 Mild Seven"的国民形象的解构与重构
- **视觉符号**：直线感的字体 + 单色几何 + 几乎没有任何"传统"装饰元素——这种"克制"恰是 JT 系日产烟与欧美烟最大的视觉分野

### 经典意义与推荐变体
Mild Seven / Mevius 是**1980s–2010s 全球烟草市场最重要的"温和派"代表**。在 TIAT 选购时建议：
- **首选**：**Mevius Original 10mg（Box）**——本款是品牌识别度最高、最能体现"美式 + 活性炭"配方哲学的版本
- **进阶**：**Mevius Option Purple（黑加仑爆珠）**——日本独有的爆珠工艺代表
- **避免**：1mg 极轻款（口感被淡化到几乎只有空气感，对于希望体验日产烟工艺的人不合算）

## 2. SEVEN STARS（セブンスター / SVS）

### 基本参数
| 项目 | 内容 |
| --- | --- |
| **所属** | Japan Tobacco（JT） |
| **创立** | **1969 年 2 月 1 日** |
| **产地** | 日本 |
| **焦油 / 尼古丁** | **14mg / 1.2mg**（标准款，焦油是 Mevius 的 1.4 倍——是 JT 在售日产烟中**焦油最高、口感最浓**的一支） |
| **过滤嘴** | **世界首款采用活性炭过滤嘴的香烟**（1969 年首创）；此后 Mevius 等 JT 系全部采用同样工艺 |
| **主要规格** | KS（85mm）Box / Soft；近年推出 Super Slim 与 D-spec（低烟臭）变体 |

### 品牌故事
Seven Stars 于 **1969 年 2 月 1 日**由当时还属于"日本専売公社"（Japan Tobacco and Salt Public Corporation，1949–1985 国营烟草专卖体系）推出，是世界上**第一款采用活性炭过滤嘴**的香烟——这一创新比 Marlboro 1971 年的"low-tar"项目早 2 年。品牌名"Seven Stars / 七颗星"来自包装上排列成 V 字形的 7 颗星徽（最初寓意为"日本专卖公社的 7 个目标"，后逐渐被解读为"北斗七星 / 引路星座"等更诗意的意象）。

上市后立即取代 PEACE 成为日本市场销量第一的品牌，并保持这一位置长达 7 年（直到 1977 年被 Mild Seven 反超）。**Seven Stars 至今仍是日本焦油最高的主流国民品牌之一**——一些资深的日本作家、漫画家、爵士乐手会说他们"始终抽 Seven Stars"，这成了一种带怀旧色彩的品味宣示。手冢治虫、野坂昭如等昭和大家都是 Seven Stars 的长期使用者。

### 烟叶配方与口感档案
- **配方**：以日本国产黄烟（黄色种 / Bright Yellow）+ Virginia 烤烟为主，**国产烟叶比例显著高于 Mevius**——这是其更"扎实"口感的源头
- **入口**：深沉的烤焙烟气，几乎没有甜味，是直接的烟草本味
- **中段**：浓密烟雾（每口烟体积感比 Mevius 大约多 30%），强劲的喉感
- **余韵**：30–60 秒持续的烟焦底香 + 烤焙余韵——这是与 Mevius 最大的口感差异
- **代表性变体**：
  - **Seven Stars Original（Box / Soft）14mg**：旗舰款
  - **Seven Stars Black Charcoal Menthol**：黑色包装薄荷版，烟叶更浓
  - **Seven Stars 8mg / 5mg**：减焦油版本，但仍保留较强的烟草本味
  - **Seven Stars D-spec**：D = "Deodorant"，低烟臭工艺，**适合不愿让衣物 / 室内长时间残留烟味的吸烟者**

### 设计与哲学
- **包装**：白底 + 黑色 V 字 7 星徽 + 经典 Helvetica-like 字体；近年版本只在徽章下增加了细线"7 STARS"小字。整套设计自 1969 年至今几乎未变（仅微调字距与图标比例），**是日本现存最长寿的视觉资产之一**
- **核心哲学**：内敛 / 持守 / 不需要解释——Seven Stars 几乎没有过任何形式的电视广告或形象代言；它的影响力完全来自烟本身的口感与"老烟民世代相传"的口口相传
- **视觉符号**：7 颗星 + 白底 + 黑字——这种极简至极的视觉与它"焦油最高 + 口感最浓"的内在形成了反差感，是日本设计美学中"見えないことは見えること"（看不见即是看见）的经典体现

### 经典意义与推荐变体
**这是 TIAT 免税柜中最值得买的一支日产烟**——理由是：
1. 中国大陆几乎不流通，是绝佳的"日本独有"礼物
2. 焦油 14mg 的浓郁口感是其他品牌无法复刻的
3. 1969 年的视觉资产已是日本现代设计史的一部分，盒子本身具有收藏价值
4. 老一辈日本知识分子"不假修饰的男性气质"的具象——抽 Seven Stars 在日本被视为某种文化身份的宣示

**TIAT 推荐**：**Seven Stars Original Box 14mg**——务必买原版，不要被 5mg / 8mg 减焦油版误导。

## 3. PEACE（ピース）

### 基本参数
| 项目 | 内容 |
| --- | --- |
| **所属** | Japan Tobacco（JT） |
| **创立** | **1946 年 1 月 13 日**（战后第一款日本烟） |
| **产地** | 日本 |
| **焦油 / 尼古丁** | **Peace（无过滤嘴）= 28mg / 2.3mg**（**全日本焦油最高**！）；**Peace Filter / Light Tap 8mg = 8mg / 0.7mg**；**Peace Aroma 10 = 10mg / 0.9mg** |
| **过滤嘴** | 经典款无过滤嘴（Plain）；Filter / Aroma 系列加过滤嘴 |
| **主要规格** | 罐装 50 支（**Can Peace**，日本唯一仍量产的金属罐装烟）+ 10 支盒装（Peace Aroma）+ 20 支软包（Peace Light） |

### 品牌故事
Peace 是日本战后历史的"在场证人"——**1946 年 1 月 13 日**，由当时还在 GHQ 占领下的"日本专売公社"作为战后日本第一款新发售烟推出。品牌名"Peace / 平和"直接呼应 1945 年战败后日本"再不要战争"的国家意志；而**让 Peace 成为 20 世纪日本设计史里程碑**的关键事件，是 **1952 年 4 月**邀请到法裔美籍工业设计师 **雷蒙德·罗维（Raymond Loewy / 1893–1986）** 重新设计包装。

雷蒙德·罗维是 20 世纪美国"工业设计之父"——他设计了可口可乐玻璃瓶（标准苗条曲线版）、Lucky Strike 红圆白底、宾州铁路 GG1 / S1 机车、肯尼迪总统专机 Air Force One 涂装。1952 年应日本专卖公社邀请来日本设计 Peace 时，他在 6 周内完成方案，**收取了 1 万 5 千美元设计费**（按 1952 年汇率约 540 万日元），是当时日本任何设计师收费的数十倍——但这一笔投入直接让 Peace 销量在新包装上市后暴增数倍，成为后来日本企业开始重视"工业设计"的转折点。

罗维的设计极简：**白底 + 蓝色翅膀展开的鸽子 + 三色橄榄枝**——白底象征日本国旗的纯洁，蓝色（"Peace Blue"）介于群青与海军蓝之间，鸽子衔橄榄枝是基督教 + 古希腊神话双重传统下的"和平"原型符号。这套视觉元素在 70 多年后仍在使用，是**世界上最长寿的烟草品牌包装设计**之一。

### 烟叶配方与口感档案
- **配方**：以美国 Virginia 烤烟为主，混入少量 Burley 和东方烟叶；**未采用活性炭过滤嘴**（与 SEVEN STARS / Mevius 不同）——保留烟叶最朴素的本味
- **入口**（无过滤嘴 Peace 28mg 经典款）：极强烈的烤焙烟气一瞬间充满口腔，是其他任何主流烟难以复刻的强度——**这是真正"老烟枪"的入门门槛**
- **中段**：浓郁的甘草 + 烤焙烟焦混合香气，烟雾密度极高
- **余韵**：60 秒以上持续的烟焦底香，**喉感强烈**——经常出现"咳一声"的体验，这是焦油浓度高的特征
- **代表性变体**：
  - **Peace（无过滤嘴 / Plain）28mg**：**绝对经典**，是日本目前在售的唯一焦油 28mg 的国民品牌香烟
  - **Peace Filter Light Tap 8mg**：加过滤嘴的轻量版本，焦油大幅降低，但保留 Peace 配方的烟焦底香
  - **Peace Aroma 10mg**：含微量香料调味，10 支小盒
  - **Can Peace（罐装 50 支）**：金属罐包装，是日本现存唯一量产罐装烟，常被作为礼物，是 TIAT 免税柜中最具仪式感的伴手礼

### 设计与哲学
- **包装**：罗维 1952 年定稿至今几乎未变——白底 + 蓝色鸽子 + 橄榄枝。**鸽子的蓝色色相（Peace Blue）已被 JT 注册为商标**，与可口可乐红同等的"色彩商标权"等级
- **核心哲学**：**"为和平而设计"** —— 罗维著名格言"美的设计源于简洁（Simplicity is the keynote of all true elegance）"在 Peace 上得到了完美体现
- **文化角色**：在战后 1940s–1950s 日本，PEACE 一支烟的价格相当于一份普通工人三餐之一，是普通家庭节日才舍得买的奢侈品；至今 Peace 仍承载着"昭和工人阶级"+"战后重建"+"克制美学"的多重文化记忆

### 经典意义与推荐变体
**Peace（罐装 50 支）= TIAT 免税柜的"必买名片"**。理由：

## 4. HOPE（ホープ）

### 基本参数
| 项目 | 内容 |
| --- | --- |
| **所属** | Japan Tobacco（JT） |
| **创立** | **1957 年 7 月 1 日**（**日本第一款带过滤嘴的香烟**） |
| **产地** | 日本 |
| **焦油 / 尼古丁** | Hope（短杆 / Short）= 14mg / 1.0mg；Hope Light = 6mg / 0.5mg；Hope 1 = 1mg / 0.1mg |
| **过滤嘴** | 普通过滤嘴（1957 年首创）；非活性炭 |
| **主要规格** | **70mm 短杆**（"Short Hope"）20 支盒 + 标准长 KS 85mm |

### 品牌故事
Hope 是**日本第一款带过滤嘴的香烟**——1957 年 7 月 1 日由日本専売公社推出。在此之前日本国内市场全是无过滤嘴的"光烟"（Plain），Peace（1946）是其中代表。Hope 的过滤嘴当时用的是 acetate 醋酸纤维素，最初目的是降低焦油摄入并解决"嘴部苦味"问题。

**值得指出**：1931 年也有一款"Hope"（同名旧版）发售过，但因 1940 年 9 月昭和天皇下令"不许日本香烟使用外国名"，旧 Hope 在二战中被废止；战后 1957 年的新 Hope 是从零起算的全新品牌，与战前同名烟无关联。

最具识别度的特征是 **70mm 的"Short Hope" 短杆设计**——比标准 85mm（KS）短约 18%；这种短杆既来自战后物资短缺、希望"用更少烟叶提供同等满足感"的工艺尝试，也成为之后 Hope 区别于其他品牌的视觉记忆。短杆对应的口感是"密度大、燃烧快、单口尼古丁更集中"——是出租车司机 / 体力劳动者 / 短时段抽烟需求者的偏爱。

### 烟叶配方与口感档案
- **配方**：以美国 Virginia + Burley 为主，混入少量日本国产黄烟；非活性炭，仅醋酸纤维素普通过滤嘴
- **入口**：直接而厚实的烟焦气，与 Peace 风格相近但稍轻
- **中段**：烤焙烟气持续，烟雾浓但比 Peace 更"利落"
- **余韵**：约 30 秒消散，比 Seven Stars 更短促但比 Mevius 更深沉
- **代表性变体**：
  - **Hope Short 14mg（70mm）**：短杆经典，是 Hope 最具识别度的版本
  - **Hope Light 6mg**：减焦油但保留短杆设计
  - **Hope 1（1mg）**：超低焦油版本
  - **Hope Menthol 6mg**：薄荷版

### 设计与哲学
- **包装**：白底 + 黑色"HOPE" 字样 + 一道蓝色或绿色斜带（依变体而异）；70mm 短杆配套设计的小尺寸盒装是其视觉记号
- **核心哲学**：**"低调的希望"**——Hope 没有 Peace 罗维那种宏大设计语汇，也没有 Mevius 那种当代品牌口号；它是"更朴素、更工人阶级"的存在
- **文化角色**：在日本被视为"出租车司机的烟"——短杆方便在堵车间隙快速抽完一支；这种"短时间满足"的工艺哲学使它成为日本"出租车文化"的一部分

### 经典意义与推荐变体
Hope 是**日本烟草过滤嘴时代的开创者**，与 Peace（无过滤嘴时代的代表）+ Seven Stars（活性炭时代的开创者）+ Mevius（轻烟时代的代表）共同构成 JT 旗下的"四代表性"。**TIAT 选购建议**：
- **Hope Short 14mg**：买经典短杆款；是 1957 年至今的活化石
- **避免**：Hope 1mg —— 过度淡化到几乎只剩气流感

Pianissimo 名字来自意大利语音乐术语"pp"（极弱）——表达"极轻量"的产品定位。**2002 年 11 月**由 JT 与 RJ Reynolds（当时的日本合资）联合推出，是日本首款明确以"**clean cigarette / 干净烟**"为概念诉求的产品。所谓"clean"指三个层面：① 焦油 1mg 是当时市场上最低之一；② 烟臭明显比标准烟低；③ 烟雾稀薄、不易让旁人察觉——这套话术精准地命中 2000s 初日本年轻女性 + 都市职业女性"想抽烟但不希望被人发现 / 不希望气味残留"的细分需求。

- **余韵**：5 秒内消散，几乎不残留
- **代表性变体**：

- **包装**：浅色调粉彩 + 极简字体 + 没有任何视觉嘈杂——是日本"少女系包装"在烟草界的代表
- **核心哲学**：**"听不到的音"**——pianissimo 在乐谱上是接近无声的极弱音；引申为这支烟"几乎不被察觉"

## 7. MARLBORO（万宝路）

### 基本参数
| 项目 | 内容 |
| --- | --- |
| **所属** | Philip Morris International（PMI） |
| **创立** | **1924 年**（最初是女士向品牌） / **1954 年**重新定位为男性向品牌 |
| **产地** | 美国 / 日本 / 瑞士（免税版多为瑞士 / 荷兰生产） |
| **焦油 / 尼古丁** | Red 10mg / 0.8mg；Gold（旧名 Lights）6mg / 0.5mg；Silver 1mg / 0.1mg；Black Menthol 8mg / 0.6mg |
| **过滤嘴** | 普通醋酸纤维素 + 部分变体加活性炭（依市场） |
| **主要规格** | KS 85mm Box / Soft；100s；Super Slim |

### 品牌故事
Marlboro 是**世界销量第一的香烟品牌**——已连续超过 50 年（1972 至今）位列全球首位，在 2024 年仍占全球市场约 9%。它的故事是**香烟史上最经典的"品牌重塑"案例**。

Marlboro 由 Philip Morris **1924 年**作为"女士向"品牌推出（口号："Mild as May / 五月般温和"），包装是浅米色 + 红色滤嘴（红色滤嘴是为了"让女士不被口红印记尴尬"）。但 1924 至 1950s 间销量平平，1953 年 Philip Morris 决定彻底重塑——找到芝加哥广告公司 Leo Burnett 操刀，由广告人 **Leo Burnett 本人** 提出"**The Marlboro Man / 万宝路牛仔**"概念：把这支女士烟改造为"美国西部牛仔"形象的男性烟——硬汉、独立、户外、无拘无束。

**1954 年 Marlboro Man 广告首发**，是 20 世纪广告史第一次在烟草上引入"原型人物（archetype）"叙事：广告中始终是一位戴着 Stetson 牛仔帽的中年牛仔骑马 / 扎篱笆 / 烧饭 / 抽烟，没有任何卖点话术。这种"诉诸图像而非话术"的广告方法在之后的 30 年间让 Marlboro 销量翻了 30 倍，1972 年首次成为世界第一。"Marlboro Man" 也被《Advertising Age》评为"20 世纪最有影响的广告形象"。包装设计上采用红白对角组合的"屋顶（roof）设计"——这是工业设计师 Frank Gianninoto 的杰作，至今几乎未变。

### 烟叶配方与口感档案
- **配方**：标准美式混合（Burley + Virginia + Oriental），其中 Burley 比例最高
- **入口**：经典美式烟焦气，明显的甜烟感（Marlboro 加入了**甘草 + 可可**等调味，是其与 Camel 不同的关键）
- **中段**：浓郁烤焙 + 甘草余韵
- **余韵**：30 秒持续
- **代表性变体**：
  - **Marlboro Red（万宝路红）10mg**：经典款，完整配方
  - **Marlboro Gold（万宝路金）6mg**：旧名 Lights，减焦油但保留原配方风格
  - **Marlboro Black Menthol 8mg**：黑色包装薄荷
  - **Marlboro Ice Blast / Double Burst（爆珠系）**：薄荷爆珠 + 双胶囊
  - **Marlboro Gold Touch（金麦秆 / 紫金）**：日本 / 韩国限定，金色优雅版

### 设计与哲学
- **包装**：红 / 金 / 白 / 黑配色 + 经典屋顶设计 + Marlboro 哥特字体（注：哥特体的选择是为了与"传统欧洲"做出区分，传达"美国新大陆"的现代感）
- **核心哲学**："**Come to Marlboro Country / 来到万宝路国度**"——这是一种"想象中的美国西部 = 自由 + 男性气质 + 户外 + 工作伦理"
- **文化角色**：与 Levi's 牛仔裤、可口可乐、Coca-Cola 一道，是 20 世纪美国"软实力 / soft power"的代表

### 经典意义与推荐变体
**TIAT 推荐**：**Marlboro Red 10mg Box**——这是世界销量第一品牌的旗舰款；中国大陆虽有销售但配方不同（更轻），免税版才是"原始美式"。如喜欢爆珠：**Marlboro Ice Blast / Double Burst**——日韩限定，国内不流通。

## 8. CAMEL（骆驼）

### 基本参数
| 项目 | 内容 |
| --- | --- |
| **所属** | 美国市场：Reynolds American Inc.（RAI / RJR）；日本市场：JTI |
| **创立** | **1913 年**（美国第一支"美式混合配方"商业香烟） |
| **产地** | 美国 / 日本 / 瑞士 |
| **焦油 / 尼古丁** | Camel Filter（黄）= 9mg / 0.7mg；Camel Light（金）= 6mg / 0.5mg；Camel Menthol = 9mg / 0.7mg |
| **过滤嘴** | 醋酸纤维素 |
| **主要规格** | KS 85mm Box / Soft；近年推出 Camel Crush 爆珠系列 |

### 品牌故事
Camel 由 **R.J. Reynolds Tobacco Company** 在 **1913 年**推出，是**美国市场上第一支"美式混合配方（American Blend）"商业香烟**——把美国本土 Burley + 进口土耳其（Oriental）+ Virginia 烤烟混合，比当时主流的"纯 Virginia"或"纯土耳其"配方更平衡。这套美式混合方法之后被 Marlboro / Lucky Strike 等无数品牌承袭，**奠定了 20 世纪世界主流香烟的口感范式**。

Camel 的视觉设计在 1913 年由 Reynolds 内部设计师完成——包装上的"骆驼"原型来自一只名为 **Old Joe** 的动物园骆驼（Barnum & Bailey 马戏团的成员），1913 年 9 月时该马戏团在 Winston-Salem（Reynolds 总部所在地）演出，被 Reynolds 设计师抓拍，画在包装上。这只骆驼站立在金黄色背景的金字塔与棕榈树前，整套视觉自 1913 年至今 110 多年几乎完全未变——是世界上最古老仍在使用的烟草品牌包装设计。

Camel 在 20 世纪上半叶是美国销量第一的烟草品牌（直到 1955 年被 Marlboro 反超）。"**I'd walk a mile for a Camel / 我愿走一英里也要抽一包 Camel**"是 1921 年起的经典口号，20 世纪知识界（海明威、福克纳、庞德等）都是 Camel 的忠实使用者，使其与"美国文学"具有特殊联系。

### 烟叶配方与口感档案
- **配方**：Burley + Virginia + Oriental 三段混合；与 Marlboro 不同的是 Camel 不加甘草调味，烟叶味更"原始"
- **入口**：略带土耳其烟特有的辛香气 + 干果调
- **中段**：扎实的烟焦烤焙
- **余韵**：余韵层次比 Marlboro 更复杂——会出现明显的"烟草树脂感"
- **代表性变体**：
  - **Camel Filter 9mg（黄）**：经典款，原配方
  - **Camel Light 6mg（金）**：减焦油但保留原烟叶比例
  - **Camel Crush（爆珠）**：薄荷爆珠
  - **Camel Turkish Royal**：使用更高比例土耳其烟，复刻 1913 年原型

### 设计与哲学
- **包装**：金黄色底 + Old Joe 骆驼 + 金字塔与棕榈树 + 经典 sans-serif "CAMEL" 字体——整套设计保留 1913 年原貌
- **核心哲学**：**"原始美国"**——比 Marlboro 更"真实"的美式叙事，强调"不依赖西部牛仔形象的美国"
- **文化角色**：知识分子的烟、文学的烟、爵士时代的烟

### 经典意义与推荐变体

- **代表性变体**：
  - **Lucky Strike Original Red 10mg**：经典款
  - **Lucky Strike Filters Lights 8mg**：减焦油

### 设计与哲学
- **包装**：罗维 1942 年定稿的白底 + 红圆 + 黑字至今保持不变；红圆在视觉上有强烈的"靶心"感——是 20 世纪最具识别度的烟草包装之一
- **核心哲学**："**It's Toasted（被烤焙过的）**"——强调烟叶工艺
- **文化角色**：与海明威、Mad Men（电视剧 Mad Men 中虚构的 Lucky Strike 广告战），是"二战 + 战后美国"的视觉记忆代表

### 经典意义与推荐变体
**TIAT 推荐**：**Lucky Strike Original Red 10mg**——罗维设计的白底红圆是必收的设计史经典。

## 16. AMERICAN SPIRIT（美国精神）

### 基本参数
| 项目 | 内容 |
| --- | --- |
| **所属** | Santa Fe Natural Tobacco Company（**RJ Reynolds 旗下**，2002 年起）；日本市场由 JTI 销售 |
| **创立** | **1982 年**（"Natural American Spirit"原品牌） |
| **产地** | 美国 New Mexico（圣塔菲）；日本市场销售版本由 JTI 在德国 / 瑞士工厂生产 |
| **焦油 / 尼古丁** | American Spirit Light Yellow（黄）6mg / 0.5mg；American Spirit Original Blue（蓝）11mg / 0.9mg；American Spirit Menthol Green（绿）9mg / 0.8mg |
| **过滤嘴** | 醋酸纤维素 |
| **主要规格** | KS 85mm Box；近年推出 Sky 系列（含活性炭） |

### 品牌故事
Natural American Spirit 由 **Bill Drake**（《The Cultivator's Handbook of Natural Tobacco》作者）+ Robert Marion + Chris Webster 三位**新墨西哥州 Santa Fe 当地的烟草自然主义者** 1982 年创立。他们的初心是反对当时主流卷烟里普遍添加的甘草、可可粉、糖、丙二醇等约 600 种化学添加剂——主张"100% 无添加 / 100% 全叶烟草（whole leaf）"，这意味着烟卷里只有烟叶碎屑，**不含 reconstituted tobacco（再造烟叶 / 烟梗重组的烟纸）**——这是与几乎所有主流品牌的根本区别。

1991 年起 Santa Fe 开始尝试 **100% certified organic（认证有机）烟叶**——是世界上第一个认证有机烟草项目。**2002 年 RJ Reynolds 收购 Santa Fe Natural Tobacco**，但保留了独立运营与"无添加 / 有机"的产品定位。

2017 年美国 FDA 要求 RJ Reynolds 不能再在 American Spirit 包装上使用"Natural"和"Additive-free"等可能误导的词汇——但产品配方与"100% 无添加 / 100% 全叶 / 部分认证有机"的工艺标准并未改变。

### 烟叶配方与口感档案
- **配方**：100% 全叶烟草（whole leaf）+ 无添加；多数变体使用 Virginia + Burley 美式混合，部分（Sky 系列）含活性炭
