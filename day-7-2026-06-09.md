## 子区域 1：浅草 / 浅草桥 / 两国（Asakusa / Asakusabashi / Ryogoku）

**段内导览**：本段将「两国本地的江户原乡午市」与「浅草寺—雷门—仲见世商业街」缝合为同一打卡环。**当日动线建议**：10:00 从 COGO RYOGOKU 退房后，先把行李按文档头三方案择一处理（方案 A 寄存前台 / 方案 B 浅草站投币柜 / 方案 C 拖行）；10:15 步行 8 分钟到**江户东京博物馆**（09:30 开门，本次 10:15–11:30 入馆 75 分钟，含常设展核心 + 战争证言映像新展）；11:30 出馆步行 6 分钟到 **江戸蕎麦 ほそ川**门口排队（开门 11:30，本店两国本地代表，现金优先），用餐 60 分钟左右；12:30 步行回 COGO RYOGOKU 取行李（若选方案 A）→ 13:00 都营浅草线两国 → 浅草（1 站、¥180、TST ❌）→ 13:30–15:00 **雷门 / 仲见世通 / 浅草寺**做打卡（避开正午 12:00 后的本地与外国旅客双重峰值）；15:30 押上 → 晴空塔双展望。

**本段地理说明（江戸蕎麦 ほそ川 ≠ 浅草）**：江户荞麦 细川（江戸蕎麦 ほそ川）的实际地址在**墨田区龟泽 1-6-5**，属**两国 / 押上 / 晴空塔** major_area（与 COGO RYOGOKU 同区、步行约 6 分钟可达），并不位于「浅草—雷门—仲见世」的浅草寺核心区。本段把它纳入"浅草段"是因为它构成了 Day 7 退房日的**「江户原乡」开场**：上午先回到两国本地吃完一星十割荞麦 → 再去浅草做最后的雷门打卡 → 押上晴空塔，整体上是「两国（住宿区原乡）→ 浅草（江户下町）→ 押上（次区）」的顺时针半圈，地理上是连续的「东京东部下町带」，不构成回头。

## 子区域 2：押上 / 晴空塔（Oshiage / Skytree）

**段内导览**：本子区域是 Day 7 的**收尾段**与本次 7 日行程的**最后一处保留地点**。从子区域 1 浅草段（雷门 / 仲见世通 / 浅草寺 / 江户荞麦 细川）折回后，**乘都营浅草线**一站到押上（**浅草 → 押上 1 站 / 约 4 分钟 / ¥180、TST ❌**），出押上站 B3 出口直通晴空塔下层 **SOLAMACHI 商业楼**（東京ソラマチ / TOKYO Solamachi，约 300 间店铺 + 餐饮层 + 水族馆 + 天文馆 1F–7F + 30F–31F）—— 先在商业楼层逛 30–60 分钟（避雨 / 简餐 / 伴手礼最后补给），再上塔做**双展望**（350 米「天望甲板」+ 450 米「天望回廊」）。

**双展望机制**：晴空塔展望分两层 —— **350 米的「天望甲板」（天望デッキ / Tembo Deck）**（位于第 1 号塔身、由三层 350 / 345 / 340 米组成、含玻璃地板）+ **450 米的「天望回廊」（天望回廊 / Tembo Galleria）**（位于第 2 号塔身、是从 445 米斜向回旋上升至 451.2 米天望シャトル展望点的玻璃管廊）；只买基本票 ¥3,100 仅可到 350 米天望甲板，加购或直接买**双展望套票 ¥4,400** 才能继续上 450 米。本日强烈推荐**直接买双展望套票**——既然到了押上不上 450 米的回廊就等于错过晴空塔最高视野。

**航班时刻协调**（按文档头硬约束 ① 10:00 退房 + ② 当晚航班时刻决策）：
- **方案 X — 中午 / 下午机场**（航班 13:00–17:00 起飞）：当日**跳过晴空塔段**，10:00 退房后只打卡浅草雷门，11:30 直奔押上换京成 Skyliner / 都营浅草线京急直通去机场。本子区域**不进入**。
- **方案 Y — 傍晚机场**（航班 17:00–20:00 起飞）：缩短在塔时间，**16:00 入场**做傍晚段（350 米一层 + 450 米一层各停 30 分钟），17:00 出塔后即去机场（押上京成 Skyliner 41 分钟到 NRT）；此时段非日落 / 夜景双享，仅做"日间展望 + 关东轮廓"打卡。
- **方案 Z — 夜间航班 / 次日航班**（航班 ≥ 21:00 或 6/10 早晨）：**18:00–21:00 段**为最佳，进塔时仍是日落黄金 1 小时（6 月夏至前夕东京日落约 18:55，黄昏 19:30，全暗 20:00）→ 夜景灯光 → 双展望充分赏味；21:00 是天望回廊最终入场，21:30 是天望甲板最终入场，22:00 闭馆，离场后再去机场。**本任务文档默认按方案 Z 描述推荐玩法**，方案 X / Y 的时段调整在 Place_Profile 内同步给出。

**主导交通方式**：**都营浅草线** + **东武晴空塔线**（東武スカイツリーライン，原东武伊势崎线浅草—押上段 6/9 当日 TST 已失效，按实际票价计费）。押上站 B3 出口直接连通晴空塔基座，不需要额外的地面通行。从晴空塔出来去机场两套备选 —— **NRT 方向**走京成押上线 → 京成 Skyliner（押上 41 分钟、¥2,580），**HND 方向**走都营浅草线京急直通（押上 → 羽田机场约 50 分钟、¥820），具体由文档尾任务 8.4 处理。
