# Nihonbashi Takashimaya Shopping Center（日本桥高岛屋）（日本橋髙島屋ショッピングセンター / Nihonbashi Takashimaya Shopping Center）

- **类型**：购物（百货）+ **历史建筑（高桥贞太郎 1933，重要文化财）**
- **位置**：东京都中央区日本桥 2-4-1（日本桥中央通沿线，与日本桥三井大楼 / COREDO 室町隔街相对）；最近车站 **东京 Metro 银座线 / 东西线 / 都营浅草线「日本桥」站 B2 / B4 出口直结**（TST ✅，本次推荐）；**东京 Metro 银座线 / 半蔵门线「三越前」站 B5 出口步行 5 分钟**（TST ✅）；JR「东京」站丸之内中央口步行 8 分钟（TST ❌ JR 段，但本次行程不乘 JR）。
- **历史与背景**：日本桥高岛屋（**日本橋髙島屋**，1933 年开馆原称"高岛屋东京店"）是 1831 年（天保 2 年）京都创业的"**髙島屋（Takashimaya）**"百货公司在东京开设的第一家直营店，**也是日本第一家被指定为「重要文化财（重文）」的百货建筑**。本馆建筑由 **高橋貞太郎（Takahashi Sadatarō，1892–1970）**主持设计，**1933 年（昭和 8 年）3 月 20 日**开馆，地上 8 层 + 地下 2 层 + 屋上 1 层（开馆时含屋顶游园 + 儿童乐园 + 室内运动场）。高橋貞太郎是大正—昭和初期日本"第三世代"建筑家代表（与渡邊節 / 内藤多仲 / 村野藤吾同期），师从伊东忠太，以"古典样式 + 钢筋混凝土"的现代古典主义为代表，其代表作还含日生剧场 / 学士会馆 / 神田旧河出书房等。本馆建筑为"和洋折衷"+ 古典样式 + 现代百货功能的结合：外观为白色花岗岩 + 红色瓷砖 + 绿色瓦顶（部分设半圆拱窗 + 古典柱式），内部含开馆当时日本最大规模的中央吹き抜け（atrium）+ 大理石内装 + 1933 年开馆时设的"日本第一台百货店扶梯"。1937 年高橋貞太郎进一步主导**第一期增筑工程**（增筑南馆 5 层），战后 1952 年与 1965 年由"村野・森建築事務所（村野藤吾，1891–1984）"主导第二 / 三期增筑——把南馆扩至 7 层 + 北馆增筑 + 内部装修升级；**村野藤吾是昭和期日本最具代表性的建筑家之一**（代表作含日生剧场 / 大阪新歌舞伎座 / 早稻田大学演剧博物馆等），他主持的高岛屋增筑是"百货建筑现代化与古典样式延续"的典型案例。**2009 年 6 月 30 日**本馆 + 1937 年增筑南馆 + 1952 / 1965 年增筑部分整体被指定为**国家重要文化财**（"高岛屋东京店"作为名义），**是日本第一家被指定为重文的百货建筑**——这一指定背后是文化厅 + 高岛屋集团合作的"百货建筑文化保护"先例，也是 2010 年代后续 多家昭和初期百货（如三越本店等）保护运动的范本。**2017 年起**高岛屋集团与三井不動産共同主导"日本橋髙島屋ショッピングセンター"再整备工程，**2018 年 9 月 25 日**新馆（**日本橋髙島屋 SC 新館**，三井不動産 + 大成建設）+ **东馆 / 西馆改修**完成、改名"**日本桥高岛屋购物中心（Nihonbashi Takashimaya Shopping Center）**"——本馆（重文）+ 新馆 + 东馆 + 西馆四栋作为同一商业综合体运营，但本馆作为重文保护核心保留 1933 年高橋貞太郎原立面 + 1937 / 1952 / 1965 年村野藤吾增筑形态。
- **看点**：
  1. **1933 年高橋貞太郎本馆白色花岗岩 + 红色瓷砖立面**：日本第一家重文百货建筑的代表外观，日本桥中央通沿线打卡点。
  2. **本馆 1F 中央吹き抜け（atrium）+ 大理石内装**：开馆当时日本最大规模的百货中央 atrium，至今保留 1933 年 + 1952 / 1965 年村野藤吾增筑后的混合内装；**本馆 1–2F 公共空间在营业时间内可外观参观**。
  3. **村野藤吾 1952 / 1965 年增筑部分**：南馆 / 北馆增筑部分的内部楼梯 + 装饰（含村野藤吾标志性的"流线型扶手 + 弧形门廊"），是昭和后期百货建筑现代化的代表内装。
  4. **屋上日本庭园（本馆屋顶）**：1933 年开馆时即设的屋顶游园，至今仍开放（按本馆营业时间），是日本最早的"百货屋顶日本庭园"之一。
  5. **B1F 食品层 + 1F 化妆品 + 2F 国际品牌**：现代百货标准布局；本日**百货物販层 20:00 闭馆**，本次行程仅外观打卡。
  6. **新馆（2018，三井 / 大成建設）+ 东馆 / 西馆**：日本桥 1 丁目街区扩展，含 LANVIN / United Arrows / 高岛屋食品馆等品牌；本日仅外观打卡。
