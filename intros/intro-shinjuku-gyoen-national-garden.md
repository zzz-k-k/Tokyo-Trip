- **类型**：庭园 / 公园（国营 / 国民公园；日 + 英 + 法三式庭园复合体）
- **位置**：东京都新宿区内藤町 11；最近车站 **东京 Metro 丸之内线「新宿御苑前」站 1 / 2 出口步行 5 分钟**（约 350 m）至新宿门 / 大木户门；亦可东京 Metro 丸之内线 / 副都心线 / 都营新宿线「新宿三丁目」站 E5 出口步行 5 分钟（约 400 m）至新宿门；JR / 私铁 / 地铁 各线「新宿」站南口步行 10 分钟（约 800 m）至新宿门；都营新宿线「新宿御苑前」（与丸之内线同站名不同位置 —— 待用户实地确认换乘出口）。园区南北约 1.1 公里、东西约 700 米、**总面积 58.3 公顷（583,000 m²）**，是新宿最大的连续绿地。

## 历史与背景

新宿御苑的土地起源于 **江户时代初期 1590 年代**——德川家康入府关东后将此地（旧名「内藤町」）赐给徳川家家臣 **内藤清成**（信州高远藩内藤家初代）作为下屋敷领地，因此整片土地最初是**信州高远藩内藤家的江户下屋敷**（约 22 万坪 / 72 公顷），是江户外缘最大的大名屋敷之一；**1698 年（元禄 11 年）** 内藤家把屋敷北缘的甲州街道宿场让出供幕府新设宿驿，即「**内藤新宿**」（今"新宿"地名的由来——"新设的宿场"）。**明治维新后 1872 年（明治 5 年）** 内藤家把屋敷上缴政府，**1879 年（明治 12 年）** 改为「**内藤新宿试验场**」（农林水产部门管辖、引种欧美果树 / 蔬菜的农业试验场）；**1906 年（明治 39 年）5 月 26 日** 经天皇敕令改建为「**新宿御苑**」对皇室开放——以法国凡尔赛宫为参考的"皇室庭园"——由法国造园家 **Henri Martinet（亨利・马丁内）** 主导设计西式整型庭园（西芝生）+ 英式风景庭园（中央芝生）+ 日本庭园（旧御凉亭一带）三式合一布局，**这是日本第一座"日 + 西"复合式皇家庭园**。**1949 年（昭和 24 年）5 月 21 日** 战后由 GHQ 命令开放为「**国民公园**」对公众开放，由 **环境省**（前身厚生省）管辖至今；**1958 年（昭和 33 年）** 园内大温室（旧温室）完工、**2012 年** 现代温室（新温室）开放，是现今日本最具规模的兰花 / 热带花卉温室之一。园内**樱花约 1,100 株 / 65 个品种**（日本最丰富的樱花品种园之一），从 2 月「寒樱」到 4 月下旬「关山 / 八重樱」连续开放近 3 个月，是东京都心赏樱代表地（与上野公园 / 千鸟之渊并列）；6 月梅雨季虽不在樱花期，但温室玫瑰 / 兰花 / 蜘蛛兰、母与子之森 / 玉藻池畔的紫阳花仍构成可观花景。一句话：新宿御苑是**江户大名下屋敷 → 明治皇家庭园 → 昭和国民公园**这一三层叠加的活样本，本质上不是"普通公园"，而是日本近代庭园史的代表作。

## 看点

1. **法国式整型庭园（Jardin à la française / 西芝生）**：园西侧约 6 公顷的对称式法式整型庭园 —— **巴黎凡尔赛 + 杜伊勒里宫**为参考，由 Henri Martinet 1906 年设计的"日本第一座法式整型庭园"，中央喷水池 + 两侧法国梧桐林荫道 + 修剪整齐的草坪几何块；秋季银杏 / 法国梧桐变色时是关东最美的落叶街道之一。
2. **英国式风景庭园（English landscape garden / 中央芝生）**：园中央约 12 公顷的开阔草坪 + 不规则水池 + 自然式树丛 —— 英国 18 世纪 Capability Brown 风格，是东京少数允许"草地野餐 + 摊毯子睡午觉"的大型公共绿地（**禁止酒精 / 烟火 / 球类**），周末家庭群聚的画面是新宿地表的"和平象征"。
3. **日本庭园（旧御凉亭 + 玉藻池）**：园东侧约 8 公顷的池泉回游式日本庭园 —— 中央有 1928 年（昭和 3 年）昭和天皇结婚祝贺时由台湾总督府寄赠的「**旧御凉亭**」（中国闽南风格的木造凉亭，亦称"台湾阁"），是园内唯一登录有形文化财；周边玉藻池畔 6 月有紫阳花、9 月有彼岸花。
4. **大温室（新温室）**：园南侧的现代玻璃温室（2012 年开放）—— 收藏约 2,700 种热带 / 亚热带植物（兰花 / 食虫植物 / 椰子 / 香蕉 / 日本绝灭危惧种），常年 25–30°C，6 月梅雨季是最舒服的避雨场所。
5. **樱花约 1,100 株 / 65 品种**：从 2 月「**寒樱 / 河津樱**」、3 月下旬「**染井吉野**」、到 4 月「**关山 / 一叶 / 八重樱**」连续 3 个月——本次行程 6 月不在樱花期，但**樱花树本身**（树形 + 树皮 + 嫁接标牌）仍可作为"日本樱花博物馆"性质的植物学打卡。
6. **母与子之森**：园北端的杂木林散步道 —— 1968 年昭和天皇 / 香淳皇后为纪念皇太子结婚而命名，林中保留江户时代内藤家屋敷遗树（武藏野落叶林原貌），是园内最少游客、最幽深的一段。

## 推荐玩法

1. **从新宿门 / 大木户门进园**（不要从千驮谷门——千驮谷门人少但距 JR 千驮谷站较远）→ 购票 ¥500（成人）/ ¥250（学生）/ 65 岁以上 ¥250 / 中学生以下免费；
2. **沿"新宿门 → 法式整型庭园 → 英式风景庭园 → 日本庭园 → 大温室 → 母与子之森"** 顺时针走一圈，全程约 60–90 分钟（含温室停留）；
3. **6 月推荐定点**：**英式风景庭园中央芝生**摊毯子野餐（**禁止酒精 / 烟火**，可带便当 + 茶 + 水）30 分钟 → 大温室避雨 20 分钟 → 日本庭园旧御凉亭拍照 15 分钟 → 母与子之森散步 15 分钟；
4. **不推荐**：园内不允许跑步 / 骑车 / 球类 / 飞盘 / 无人机；不要带宠物入园（**禁止携带宠物**）。

## 注意事项

1. **禁止携带酒精饮料 / 烟火 / 球类 / 飞盘 / 无人机 / 宠物**：园方在新宿门 / 大木户门入口处会随机抽查包内物品；带便当 + 茶 + 水 + 咖啡均可，但**绝不可带啤酒 / 清酒 / 红酒**进园（这是日本国营庭园的明确硬规则，与上野公园 / 代代木公园允许酒精的做法不同）。
2. **禁止跑步 / 骑车**：园内全程**禁止跑步 / 慢跑 / 骑车 / 滑板 / 滑轮鞋**；只允许散步 + 摊毯子静坐。这是与"代代木公园 / 皇居外苑"的最大差别。
3. **草坪允许摊毯子**：英式风景庭园中央芝生**允许摊毯子野餐 + 静坐 + 午睡**，但禁止"球类 / 飞盘 / 跑动"；草坪边缘的"芝生养生中"红绳围栏区域不可踩踏。
4. **拍照许可**：园内**允许手持手机 / 相机静音拍照**，禁止三脚架 / 闪光灯 / 自拍杆（园方规则与博物馆同步）；商业拍摄（婚纱照 / 短片 / 模特拍摄）需提前申请许可（[https://fng.or.jp/shinjuku/](https://fng.or.jp/shinjuku/) 查询）。
5. **6 月梅雨季**：园内大温室（南端）+ 旧御凉亭（中段日本庭园）+ 母与子之森（北端杂木林）三处可避雨；摊毯子野餐请看天气预报，雨天改为"温室 + 凉亭"室内动线；雨后日本庭园池畔石阶湿滑。
6. **周一定休**：每周一定休（周一为国定假日时翌平日休）+ 12/29–1/3 年末年始；**6/6（周六）正常开馆**，本日不冲突；但若因故改期请避开周一。
7. **闭园时间提醒**：3/15–6/30 期间 **18:00 闭园 / 17:30 最终入园**；本段动线建议 13:25 入园 → 14:55 出园（不超 18:00），**绝不要把御苑安排到 17:00 后**——临近闭园园方会用园内广播 + 巡逻员从北端母与子之森开始驱赶，体验受影响。
8. **现金 / IC 卡 / 在线 QR 三选**：园门窗口接受现金 + Suica / PASMO + 部分国际信用卡；**最快通道是官网在线购票打印 / 截屏 QR**，QR 通道无排队。

- **最佳季节 / 最佳时段**：**3 月下旬樱花期（染井吉野盛花）**+ **4 月上旬–下旬八重樱期**（晚樱品种丰富）+ **11 月银杏 / 法国梧桐 / 红叶期**（西芝生法式整型庭园是关东最美落叶街道之一）+ **2 月寒樱期**（少人 + 早开品种）；本次行程 6 月**不在樱花期**，但温室玫瑰 / 兰花 + 母与子之森 / 玉藻池畔紫阳花 + 草坪野餐仍是 6 月主流玩法；最佳时段 09:00 开园后或 15:00 后人少光线柔和。
