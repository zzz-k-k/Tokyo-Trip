## 历史与背景

「東京豚骨拉麺ばんから / Tokyo Tonkotsu Ramen Bankara」是由 **株式会社花研** 于 **2002 年（平成 14 年）3 月** 在东京板桥创业的拉面连锁品牌——创业初衷是把九州博多 / 久留米的浓厚豚骨拉面传统**改造为东京口味**（汤底浓度比博多减 30%、面条改为东京流行的中粗卷曲面、加入东京关东煮风的"角煮"作为招牌配料），形成"**东京式豚骨**"的独立流派；**ばんから / Bankara** 一词源自日语「**蛮殻 / 蛮ハイカラ**」（"野蛮 + 西式时髦"的合成词）。**2012 年新宿歌舞伎町店开业** 后因 24 小时营业 + 位于歌舞伎町一番街入口位置成为品牌**最具象征性的旗舰店**。**2018 年起店外人行道上方设置 YouTube 直播摄像头**（"**新宿歌舞伎町ライブカメラ**"24 小时面向歌舞伎町一番街直播），疫情期间成为全球网友"云观察东京"的代表镜头之一——**单日全球观看人数最高峰超 100 万**，CSV 备注「YouTube 直播摄像头所在地」即指此。**2024 年品牌全国扩张** 至 30+ 分店（关东 + 关西 + 中京 + 九州 + 海外），新宿歌舞伎町店仍为旗舰。

## 推荐玩法

1. **入店前在店外 YouTube 摄像头镜头前停留 30 秒挥手**（提前给国内朋友发直播链接）；
2. **店门口左侧食券机点单** → 推荐**「角煮拉面 ¥1,200」+「煮玉子 ¥150」+「替え玉 ¥150」** = ¥1,500 / 人；
3. **入座出示食券给店员** → 等 5–8 分钟出餐 → 店员问「お好みは？」回答「普通 / Futsu / Normal」最稳；
4. **吃面礼仪**：日本拉面**允许大声吸面**；**汤底不必喝完** + **不剩面**；吃完说「ご馳走様でした」即可；
5. **离店前再次到店外 YouTube 摄像头前挥手** → 进入歌舞伎町一番街向北夜游或东行入 Golden-Gai。

## 注意事项

1. **食券机点单 + 现金 + IC 卡 + VISA 三选**：食券机接受 **现金 + Suica / PASMO IC 卡 + VISA / Mastercard / JCB 信用卡**，但**部分国际信用卡读卡可能失败**—— 备 ¥2,000 现金为后备。
2. **24 小时营业但深夜客层复杂**：凌晨 02:00–05:00 是歌舞伎町夜班族 + 醉客混杂时段；**通常和平**，不与醉客对视即可。
3. **店内吸烟可（部分席位）**：吧台前 6 席为吸烟席、其余为禁烟席；**待用户实地确认** 现行政策。
4. **不剩面 + 不必喝完汤**：**「不剩面」**（失礼）+ **「不必喝完汤」**。
5. **「お好みは？」回答**：国际客直接说「普通 / Futsu / Normal」最稳。
6. **YouTube 直播摄像头隐私**：店外摄像头**24 小时全球直播**——**避免在镜头前做不雅动作**。
7. **梅雨季 + 24 小时室内避雨**：店面 24 小时营业 + 室内空调，是歌舞伎町夜段最稳定的避雨点。
8. **末班车提醒**：**末班都营大江户线 新宿西口 → 两国 约 24:30 前发车**；如错过末班可在本店坐到 05:00 首班。
