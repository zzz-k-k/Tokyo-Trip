- **类型**：购物 / 御宅（中古漫画 / 同人志 / 模型 / 复古玩具 / 偶像周边等综合店；涩谷 BEAM 旁地下店）
- **位置**：东京都涩谷区宇田川町 31-2 涩谷 BEAM 大厦 B2F；最近车站 JR 山手线 / 东急东横线 / 东京 Metro 半蔵门线 / 副都心线 / 银座线「涩谷」站 A4 / B1 / 8 号出口步行 5 分钟（TST ✅ 部分覆盖：半蔵门 / 副都心 / 银座线段 ✅、JR 山手线 / 东急段 ❌）。
- **历史与背景**：Mandarake（まんだらけ）是 1980 年由古物商古川益藏（Furukawa Masuzo）在中野百老汇创立的中古漫画 / 御宅商品综合连锁，经四十多年发展现已是日本最大的"二手 ACGN（Anime / Comic / Game / Novel）+ 同人志 + 复古玩具 + 偶像周边"流通体系，全国设 9 大城市直营店与多家分店。涩谷店于 1995 年开业，初期设在涩谷 BEAM 大厦的 B2F 地下层（一直延续至今），是 Mandarake 系统中"涩谷 / 表参道 / 原宿"亚文化客层的窗口店，也是 Mandarake 品牌在中野以外的第二家直营店。涩谷 BEAM 大厦本身于 1992 年开业（设计 / 隈研吾、山下泰郎），是涩谷宇田川町亚文化街区（含 BEAM、Tower Records、Loft 等）的标志性建筑之一；BEAM 大厦的"凹凸交错的金属外立面 + 顶部尖塔"在 1990 年代初是日本"后现代"建筑的代表，让 Mandarake 涩谷店从开业起就嵌入了这条"日本平成期 ACGN 商业街"的核心带。涩谷店与中野店、秋叶原 Complex 店在选品定位上有明显分工：中野店是"全主题分店组合（古董玩具 / 怪兽软胶 / 怀旧游戏卡带）"、秋叶原 Complex 店是"现代御宅 + 国际游客 + 偶像 / 同人"、涩谷店则定位"涩谷亚文化客层"——主销 1990–2010 年代漫画 / 同人志 / 偶像周边 + V-kei / 视觉系音乐周边 + 涩谷系亚文化遗产。
- **看点**：
  1. **B2F 地下店空间**：BEAM 大厦地下两层的中古漫画 + 同人志 + 偶像周边综合卖场，单层约 700 平方米。
  2. **涩谷亚文化遗产选品**：1990–2010 年代涩谷系亚文化（V-kei 视觉系 / J-POP 偶像 / 涩谷系流行音乐）周边、稀有 LP / CD、绝版同人志。
  3. **BEAM 大厦本身（隈研吾 / 山下泰郎 1992 设计）**：地下店外墙 + 大厦立面是平成初期"后现代"建筑遗产；可在地面层拍 BEAM 立面。
  4. **「特价台 / 限定 100 円台」**：B2F 入口处的特价台是涩谷店招牌，每月轮换不同主题特价货。
- **推荐玩法**：13:35 从涩谷 Hikarie B3 通道 / 东口出 → 步行 5 分钟到 BEAM 大厦 → B2F 地下店速逛 25 分钟（重点：漫画区 + 同人志区 + 偶像周边特价台）→ 14:00 出发去 Shibuya Sky。
- **票务 / 营业时间**：
  - 入场料金：免费进入；商品消费另计。
  - 营业时间：12:00–20:00；定休日 不定休（原则年中无休）。
  - 6/8（周一）正常营业。票价 / 营业时段如有调整以官网为准 —— 待用户实地确认（官网：https://www.mandarake.co.jp/shop/sby/）。
- **交通到达**：
  - 从前一地点（代官山蔦屋书店）：东急东横线 代官山 → 涩谷（¥140、2 分钟、TST ❌）+ 出涩谷站 A4 / B1 / 8 号出口步行 5 分钟到 BEAM 大厦。合计 ≈ 12 分钟、¥140、TST ❌。
  - 从两国 COGO RYOGOKU 出发：都营大江户线 → 半蔵门线 涩谷（约 25 分钟、TST ✅）+ 步行 5 分钟，0 元票价（TST 已覆盖）。
- **周边联动**：步行 3 分钟内可达 **Shibuya Sky**（涩谷 Scramble Square）；步行 5 分钟可达 **LINE CUBE SHIBUYA**（公园通顶端）；步行 5 分钟可达 **Euro Live**（道玄坂中段，已结束仅外观）；步行 8 分钟可达 **d47 Museum**（涩谷 Hikarie 8F）。
- **消费档次**：B2F 中古漫画 ¥100–500 / 册、同人志 ¥500–2,000 / 册、偶像周边 ¥500–8,000 / 件；本次速逛预算约 ¥0–3,000（按机动选购）。
- **注意事项**：
  - 店内允许拍商品 / 货架，但不允许拍他人客户 / 收银台。
  - 部分稀有 LP / 同人志为玻璃柜陈列，不允许直接触碰，需店员协助取出。
  - 现金 / 主要电子支付（Suica / PASMO / 主要信用卡）皆可。
  - 6 月梅雨季 BEAM 大厦地下层空气湿热，建议把外套寄存到入口前（B2F 入口处有免费小型投币柜）。
- **购物类别专属字段**：
  - **主营商品类目**：中古漫画 + 同人志 + 偶像周边 + V-kei / 视觉系音乐周边 + 复古模型 + 偶像写真集 + 稀有 LP / CD。
  - **价格区间**：¥100–8,000 / 件（多数 ¥500–2,500）。
  - **是否支持免税**：单店消费 ¥5,000+ 凭护照可申请免税；当场扣除消费税约 10%。
  - **是否支持寄国际**：店内不直接寄国际，但 Mandarake 有官方代购 + 海外发送服务（官网在线下单 → 海运 / 空运），费用按尺寸计；建议大件 / 多件购物用此途径。
