# 涩谷 SKY（渋谷スカイ / Shibuya Sky）

- **类型**：观景（涩谷 Scramble Square 商业楼顶 14F + 45F + 46F + 屋顶 SKY STAGE 的展望复合设施）
- **位置**：东京都涩谷区涩谷 2-24-12（涩谷 Scramble Square 大厦顶部）；最近车站 JR 山手线 / 东急东横线 / 东急田园都市线 / 东京 Metro 半蔵门线 / 副都心线 / 银座线「涩谷」站东口直结（B6 / B7 / 16b 出口直通 Scramble Square 入口大厅，TST ✅ 半蔵门 / 副都心 / 银座线段）。
- **历史与背景**：涩谷 SKY 于 2019 年 11 月 1 日与涩谷 Scramble Square 大厦同期开业，是日建设计 + KPF + SANAA 联合设计的 230 米高复合大厦的"展望主体"，由东急 / JR 东日本 / 东京 Metro 共同开发——大厦本体作为「涩谷站百年再开发计划（涩谷站桜丘口地区第一种市街地再开発事业等）」的核心节点，把过去 1956 年开业的东横百货店东馆原址完全置换为 47 层的写字楼 + 商业 + 展望复合体。涩谷 SKY 的展望区分三层布置：14F 入场口（购票 / 检票 / 寄存大件物品）→ 45F「SKY GALLERY」（半室内的展望长廊 + 多媒体艺术展示）→ 46F（屋顶通道）→ 屋顶「SKY STAGE」（260 米海拔高度的露天屋顶展望平台，约 2,500 平方米的开放屋顶 + 中央草坪 + 透明玻璃围栏 + Scramble Square 标志性的"虫眼镜（CLOUD HAMMOCK）"躺椅）。屋顶高度 230 米 + 大厦本体海拔 25 米 + 屋顶围栏顶端 = 海拔 260 米左右，是涩谷一带（被环状的 JR 山手线包围的"碗底"地形）能取得的最高展望视角。屋顶 SKY STAGE 的视野以南西—东三个方向为主：西北是新宿副都心摩天楼群（含都厅 / 东京 Mode 学园塔 / Park Tower 等）；正南是恵比寿—大崎—品川一带；正西是富士山方向（晴天时可见富士山轮廓）；东北是东京塔 + 六本木 Hills + 晴空塔的"东京中心三塔"组合。SKY STAGE 是 2019 年起东京新增最热门的"360 度无遮挡屋顶展望"，2022 年 3 月起调整为时段预约制（每 30 分钟一段，每段限流约 800 人）以控制屋顶人流密度。
- **看点**：
  1. **屋顶 SKY STAGE 360 度展望**：260 米海拔的露天屋顶 + 透明玻璃围栏 + 中央草坪，是涩谷区内的最高公开展望点。
  2. **CLOUD HAMMOCK（云吊床）躺椅**：屋顶南端的 18 张透明玻璃围栏旁的"虫眼镜"躺椅，可仰躺看天。
  3. **45F SKY GALLERY 多媒体艺术装置**：teamLab 系或多媒体艺术家驻场的环形展示空间，按月度更替；本月（6 月）展品以官网为准 —— 待用户实地确认。
  4. **正西富士山方向（晴天）**：从屋顶 SKY STAGE 西侧透明围栏向西看，晴天可看到富士山轮廓（6 月梅雨季稀少，可遇不可求）。
  5. **东北东京三塔组合**：东京塔 + 六本木 Hills + 晴空塔的横向并列，这种视角在东京都心仅涩谷 SKY + 六本木 Hills 屋顶 + 晴空塔本身可见。
