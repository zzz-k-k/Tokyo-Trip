- **类型**：商业街 / 夜游（亚洲最大规模的红灯 / 娱乐街区；现代国际化亚文化夜场代表）
- **位置**：东京都新宿区歌舞伎町 1–2 丁目；最近车站 **JR / 私铁 / 地铁各线「新宿」站东口步行 5 分钟**（约 400 m）至歌舞伎町一番街南口；亦可 西武新宿线「西武新宿」站正面口出闸即入歌舞伎町北端步行 2 分钟、东京 Metro 副都心线 / 都营大江户线 / 丸之内线「新宿三丁目」站 B11 / B13 出口步行 5 分钟、都营大江户线「东新宿」站 A1 出口步行 5 分钟。街区南北约 600 米、东西约 400 米，**约 0.24 km² 的紧凑街区集中了约 3,000 家餐饮 / 风俗 / 娱乐设施**（日本最高密度），是新宿东口的核心商业街区。

## 历史与背景

歌舞伎町的土地起源于 **江户时代** 内藤新宿宿场的外围低洼地带（旧名「角筈」），是周围村落的稻田 + 沼泽地带；**明治末期至昭和初期** 此地为新宿西北边的"水稻地区 + 杂居住宅区"，没有商业街性格。**1945 年（昭和 20 年）东京大空袭** 中此地被彻底烧毁，战后 GHQ 占领期由当地町会会长 **铃木喜兵卫** 主导战后复兴计划——**1948 年（昭和 23 年）4 月** 町会决定在此**新建一座大型歌舞伎剧场**作为街区核心，并据此把街区命名为「**歌舞伎町**」（"歌舞伎座"街），但**剧场建设最终因资金 + 用地问题终未实现**——歌舞伎町因此成为**日本唯一一个"以从未建成的剧场命名"的街区**。**1950 年代** 街区因新宿站东口的人流红利迅速发展为新宿首屈一指的商业街，但**红线指定区**（売春防止法施行前的合法红灯区）+ 战后黑市 + **韩国华侨街** + **黑社会**（山口组 / 住吉会 / 稻川会三大组的关东支部均在歌舞伎町设事务所）+ 风俗营业等多重力量交织，在 **1960–80 年代** 形成"日本最复杂的城市生态"。**1990 年代起** 街区开始国际化转型——**亚洲（中 / 韩 / 越 / 泰）+ 欧美**留学生与劳动者大量涌入开店，传统风俗业转向**主持俱乐部（ホストクラブ）/ 牛郎店 / 二次元主题餐饮（女仆咖啡 / 执事咖啡）/ 电竞酒吧 / Karaoke**等更现代的形态。**2015 年「新宿东宝大厦」（Hotel Gracery 新宿 + TOHO 影院 + 屋顶哥斯拉雕像）** 开业、**2023 年 4 月「东急歌舞伎町塔」（Tokyu Kabukicho Tower）** 开业（隈研吾设计的 48 层 225 米地标摩天楼），是 21 世纪歌舞伎町从"红灯街"向"国际化娱乐综合街区"转型的代表项目。今天歌舞伎町俗称「**眠らない街 / 不眠之街**」（"Neverland"或"Sleepless Town"），24 小时人流不断，是东京夜文化的视觉地标。

## 看点

1. **歌舞伎町一番街拱门（Kabukichō Ichibangai Gate）**：街区南端入口的红色拱形招牌"歌舞伎町一番街"+ 黑底霓虹字 —— 是歌舞伎町最易识别的视觉地标。
2. **东宝大厦哥斯拉雕像（Hotel Gracery 新宿屋顶）**：街区中心的新宿东宝大厦 8 楼露台上**全身高 12 米的哥斯拉头部雕像**（Godzilla Head）—— 2015 年设置，**每整点哥斯拉头会咆哮 + 喷雾**（仅日间 12:00–20:00）。
3. **东急歌舞伎町塔（Tokyu Kabukicho Tower / 2023）**：隈研吾 + Yuko Nagayama 设计的 48 层 225 米摩天楼 —— 含 NAMCO LAND / 109 Cinemas Premium / Bellustar Tokyo / Hotel Groove Shinjuku 等设施。
4. **新宿黄金街（Shinjuku Golden-Gai）**：歌舞伎町东侧、花园神社西侧的 6 条窄巷木造酒场群（约 280 间小酒场）。
5. **花园神社（Hanazono Shrine）**：街区东侧的江户期勧请神社 —— 新宿总镇守。
6. **二次元主题街（女仆咖啡 / 执事咖啡 / 电竞酒吧）**：街区东侧靠近 Golden-Gai 的小巷里集中了 20+ 家主题餐饮。

## 推荐玩法

1. **从 JR 新宿站东口出闸 → 步行 5 分钟到歌舞伎町一番街拱门** → 在拱门下拍 1 张正面入街全景（夜晚最佳）→ 进街向北沿一番街走 100 米看哥斯拉雕像 → 街中段右转到东宝大厦露台（持餐饮收据上 8F）→ 街北端看东急歌舞伎町塔；
2. **吃晚饭**：歌舞伎町一番街南端的 **Bankara 拉面新宿歌舞伎町店** 24 小时营业，本日推荐作为晚饭（19:00 后段）；
3. **夜游街区** 60–90 分钟（不入风俗店 / 主持俱乐部）→ 22:00 后东进 Golden-Gai。

## 票务 / 营业时间

街区**免费 + 24 小时开放**（公共道路）；个体店家营业时间各异，但**多数夜场从 11:00–翌日 05:00 营业**，部分 24 小时；**街区从晚 19:00 开始进入夜场高峰**，22:00–翌日 02:00 是人流最密集时段。

## 交通到达

- **从前一地点 新宿御苑（新宿门）**：新宿门 → 步行约 12 分钟（约 1 km，向西北沿新宿通走至歌舞伎町一番街南口）；亦可乘东京 Metro 丸之内线 新宿御苑前 → 新宿（一站、¥180、**TST ✅**）+ 步行 5 分钟。**推荐步行**穿过新宿三丁目商业街感受新宿地表，**全段免费**。
- **从两国 COGO RYOGOKU 出发（参考路线）**：COGO 步行 6 分钟 → 都营大江户线 两国站 → 都营大江户线（两国 → 新宿西口，约 25 分钟、¥280、**TST ✅**）→ 新宿西口站 D5 出口步行 5 分钟到歌舞伎町西端；约 35 分钟一程，**TST ✅** 实付 ¥0。

## 周边联动

步行 5 分钟内 **新宿黄金街 / Shinjuku Golden-Gai**（一番街东侧 200 m）、**Bankara 拉面新宿歌舞伎町店**（一番街南端入口处）、**花园神社**（Golden-Gai 东侧）、**新宿东宝大厦 + 哥斯拉雕像**（街区中心）、**东急歌舞伎町塔**（街区西北）；构成 Day 4 子区域 3 新宿夜段的完整步行链。

## 消费档次

街区免费；如吃 Bankara 拉面 ¥850–1,200 + 进 1 家 Karaoke 1 小时 ¥1,500 + 哥斯拉雕像近距离观览（Bonsalute Café ¥800 咖啡）= 人均 ¥3,000–5,000；不入风俗店 / 主持俱乐部 / 牛郎店。

## 注意事项

1. **风俗店 / 主持俱乐部 / 牛郎店谨慎**：**绝不可跟随客引き入店**——一旦进店"无菜单 + 无价目表"消费，结账时可被收 ¥50,000–500,000 / 人不等的"任意定价"，是歌舞伎町**最常见的国际游客陷阱**。本次行程**严禁跟随任何街边招客员**，仅入有明确菜单 + 价目表的连锁店。
2. **黑社会事务所避让**：**不要拍摄黑色西装持续聚集的店面 / 大厦入口**；遇到肢体冲突 / 街头骚动**立刻退出街区到 JR 新宿站东口**。
3. **歌舞伎町一番街南端「东京・新宿・歌舞伎町案内所」**：一番街拱门内 50 米处有官方设置的"案内所"+ 警视厅新宿警察署歌舞伎町交番（24 小时）—— 遇到任何问题**立即去交番报警**。
4. **未成年携带禁止**：歌舞伎町**绝不可带未成年人入街**（警视厅条例：22:00 后 18 岁以下不可在街区内停留）。
5. **禁止外摆酒精**：街区公共道路**禁止外摆喝酒**。
6. **6 月梅雨季**：街区一番街拱门 + 主要小巷有檐廊覆盖，雨天可在 Bankara 拉面 / 唐吉诃德 / 东宝大厦内避雨。
7. **拍照礼仪**：拱门 + 哥斯拉雕像 + 东急歌舞伎町塔可自由拍摄；**不要把客引き / 风俗店招牌 / 黑社会事务所 / 醉客 / 警员近景拍入**。
8. **回程末班车提醒**：**末班都营大江户线 新宿 → 两国 约 24:30 前发车**；如错过末班可乘出租车从歌舞伎町到两国约 ¥3,500–5,000 / 25 分钟。

## 商业街 / 夜游类别专属字段

- **是否收费**：**街区免费 + 24 小时开放**（公共道路）；个体店家按各自菜单 / 票价计费。
- **建成 / 命名年代**：街区**1948 年（昭和 23 年）4 月** 由町会主导命名为「歌舞伎町」；**2023 年 4 月 14 日** 东急歌舞伎町塔开业。
- **代表性店家 / 设施**：**Bankara 拉面新宿歌舞伎町店**（24 小时）、**Hotel Gracery 新宿 + 哥斯拉雕像**（2015）、**东急歌舞伎町塔**（2023 / 隈研吾）、**Don Quijote 歌舞伎町店**（24 小时）、**TOHO Cinemas 新宿**、**Shinjuku Golden-Gai**（约 280 间小酒场）、**花园神社**。
- **营业时间高峰段**：**晚 19:00 进入夜场高峰，22:00–翌日 02:00 是人流最密集时段**；24 小时不眠，俗称「眠らない街」。
- **消费档次**：人均 ¥3,000–5,000（拉面 + Karaoke + 咖啡），不入风俗 / 主持 / 牛郎店。
- **国际游客禁忌**：**严禁跟随街边客引き入店**；不要拍摄风俗店 / 黑社会事务所 / 醉客 / 警员；公共道路禁止外摆喝酒。
- **取景影视**：《新宿事件》（2009 / 成龙）、《唐人街探案 3》（2021）、Netflix《全裸监督》（2019）。
