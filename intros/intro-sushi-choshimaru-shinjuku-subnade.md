## 历史与背景

**銚子丸**：1976 年创业的千叶县銚子港直送回转寿司连锁，本部位于千叶市，目前在千叶 / 东京 / 埼玉 / 神奈川一都三县共约 94 家店。品牌定位"家庭式回转寿司"，主打**銚子港朝採水揚的当日鲜鱼直送**——銚子港是日本三大渔港之一（千叶县东端，黑潮 + 亲潮交汇）。

**「劇団員」概念**：銚子丸将寿司师傅称作「劇団員」（剧团员），把店内吧台称作「舞台」（舞台），把寿司制作过程视作"现场表演"——这种品牌哲学源自创业者堀地速男「家族で楽しめる寿司劇場」（全家可享的寿司剧场）的初衷，是日本回转寿司行业里少见的剧场化品牌叙事。

**新宿サブナード店**：2026 年 2 月开业，是品牌**首次进入新宿核心商圈**。サブナード是新宿东口地下街的标志性步行系统，1973 年开通，连接 JR 新宿站 / 西武新宿站 / 都营 / Metro 多线，年客流量超 2,000 万人——选择这一立地点是品牌"从家庭餐厅向都市快食"的战略调整。

## 看点

### 1. 生本まぐろ解体ショー（金枪鱼解体秀）

**「銚子丸 感謝祭」2026 年期间（5/23 ~ 6/21）每周末固定开催**：
- **第 1 場**：11:30
- **第 2 場**：17:30

新宿サブナード店参加全店同时开催。解体秀本身约 **15–20 分钟**，从去鳃 → 切头 → 三枚卸 → 部位分割（赤身 / 中とろ / 大とろ / 中落ち / かま）现场完成，资深劇団員讲解部位特征 + 处理技法。

**当日限定**：生本まぐろ 3 カン组合 **¥830（含税 ¥913）**，含赤身 / 中とろ / 大とろ 三种部位，刚切下来即握，**仅限解体秀当日数量限定**。

### 2. 平日半額キャンペーン（周间半价）

「銚子丸 感謝祭」期间每周一到周五特定品种半价，**6/8（一）~ 6/12（五）的指定半价品种为「びんちょうまぐろ（长鳍金枪鱼）」¥180（含税 ¥198）**。

### 3. 銚子港直送的当日鲜度

銚子港朝 4–5 时水揚 → 8–10 时筑地中央卸売市场 → 11 时前送达店内 → 12 时午市开始。**中等价位回转寿司能做到「当日水揚」就送上桌的极少**，銚子丸是少数从产地到店内全链条 8 小时以内的品牌之一。

### 4. 「劇団員」舞台体验

寿司吧台前两排座位是观赏「劇団員」操作的最佳位，观察握寿司、切片、装盘的连续动作约 30–60 秒一贯（标准回转寿司店通常工厂化预制 1–2 秒就上盘；銚子丸坚持现握使其单贯单价比 100 円系列高一档）。

## Navigation_Essentials

1. **从 WITH HARAJUKU HALL（短片节会场）到本店**：JR 山手线 原宿 → 新宿（1 站、约 3 分钟、¥160、TST ❌）→ 出 JR 新宿站东口 M8 / E8 出口 → サブナード地下街直入（步行 5 分钟）
2. **预订状态**：**不接受预约**，到店受付制；推荐 **17:00 前抵达取号**，参加 17:30 解体秀场次
3. **缘 App**：到店前可下载使用（IOS / Android），查看排队状态 + 累积积分
4. **推荐入店时段**：解体秀前 30–45 分钟到店取号最稳；过 17:25 后基本不可能赶上 17:30 場
5. **支付方式**：现金 + IC 卡 / Apple Pay / 信用卡（Visa / Master / JCB / Amex / 银联）
6. **重要提醒**：**App 等级别优惠券 + おすし集め割引券在新宿サブナード店和アリオ橋本店不可使用**——**仅解体秀和半价菜单全国统一适用**
