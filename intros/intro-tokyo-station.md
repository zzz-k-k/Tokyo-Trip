- **类型**：交通节点 / **历史建筑（辰野金吾 1914，重要文化财）**
- **位置**：东京都千代田区丸之内 1-9-1（东京站丸之内中央口正面）；最近车站 **JR 东日本 / JR 东海「东京」站丸之内中央口直通**（TST ❌ JR 段、但本次仅外观打卡不乘 JR）；**东京 Metro 丸之内线「东京」站直结**（TST ✅，本次推荐）；东京 Metro 半蔵门线 / 都营三田线「大手町」站步行 5 分钟（TST ✅）；东京 Metro 千代田线「二重桥前」站步行 5 分钟（TST ✅）。
- **历史与背景**：**东京站丸之内站舍**是日本铁道史与近代建筑史上最重要的文物级建筑之一——1914 年（大正 3 年）12 月 20 日开业，由"日本近代建筑之父" **辰野金吾（Tatsuno Kingo，1854–1919）**主持设计、葛西万司辅助、铁道院主导兴建，是辰野金吾"红砖样式（辰野式 / Tatsuno-style）"的代表作。辰野金吾是日本第一代留学英国（伦敦大学，师从 Josiah Conder）回国的建筑家，明治末期—大正初期在日本本土设计了大量公共建筑（含日本银行本店、奈良酒店、滋贺县厅、九州大学初期建筑等），其"赤煉瓦（红砖）+ 白色花岗岩水平腰带 + 八角形穹顶"的组合被称为"辰野式"，东京站丸之内站舍是这一样式的集大成者。**站舍长 335 米**，由南北两端的"南翼 / 北翼"两个**八角形穹顶大厅（南北圆顶，Dome）** + 中央部"中央口正面玄关"组合，外墙采用红砖 + 白色花岗岩水平腰带，是亚洲最大的红砖站舍。1923 年关东大地震时站舍受轻微损伤但主体未塌；**1945 年 5 月东京大空袭中南北两个八角形穹顶被烧毁**，1947 年战后由铁道省主持简易复原（穹顶被替换为简化的方形屋顶 + 仅 2 层规模），战后约 60 年间一直以"简化版"运营。**2007–2012 年**由 JR 东日本主导的"丸之内站舍保存 + 复原工程"（设计：JR 东日本建筑设计事务所 + 田村明日香等顾问；施工：鹿島建設 / 大林组 / 清水建設 三社共同体；总造价约 500 亿日元）以辰野金吾 1914 年原设计图为蓝本，**完全复原南北两个八角形穹顶 + 第三层 + 内部装饰**——**2012 年 10 月 1 日**完成复原后开馆，是日本"近代建筑战后复原"的最大规模工程之一。**1949 年指定为铁道纪念物 + 2003 年指定为国家重要文化财（重文）**——丸之内站舍 + 站前的丸之内 Tokyo Station Square 广场（下一项）+ 行幸通绿带组合，是丸之内—东京站—皇居前的"皇居前正面玄关轴"的核心节点，与新宿副都心的"高层都市"主义并列，是"日本传统首都正面玄关"的代表景观。
- **看点**：
  1. **辰野金吾 1914 年原设计 + 2012 年复原的红砖站舍正面**：335 米长的红砖立面 + 白色花岗岩水平腰带 + 中央"丸之内中央口"正面玄关，是亚洲最大的红砖站舍，傍晚至夜景时段（19:00 后）打灯效果最佳。
  2. **南北两个八角形穹顶（南北圆顶 / Dome）**：辰野式的标志元素，1945 年烧毁后 2012 年完全复原；穹顶下方为"南北圆顶大厅"，**进入 JR 站内可仰看穹顶内侧的浮雕装饰**（含八方位 / 干支动物 / 凤凰等浮雕，免费）。
  3. **东京站画廊（Tokyo Station Gallery）**：站舍北翼（北圆顶）3 楼的小型美术馆（按月度更替企划展，¥1,000–1,500），是"在国家重要文化财内的美术馆"日本独例。
  4. **2012 年复原后的"第三层"**：原 1914 年站舍为 3 层，1945 年烧毁后战后简化为 2 层；2012 年复原后回归 3 层，是"完全复原"的关键标志。
  5. **正面对望行幸通 + 皇居**：从站舍中央口正面西望，正前方是行幸通绿带（皇室行幸时使用的中央大道）+ 远处的皇居正门石桥 + 二重桥；这是日本"皇居前正面玄关轴"的代表视角。
- **推荐玩法**：19:40 出东京站丸之内中央口 → 步行到广场中央 → 回望站舍正面（中央口 + 南北圆顶 + 第三层），拍 5 分钟红砖夜景立面 → 沿广场南北方向走，分别拍南圆顶 + 北圆顶八角形穹顶外观 → 如时间允许从中央口入站内仰看南北圆顶内部浮雕 5 分钟（**注意：进入 JR 站内仅作仰看不乘 JR，免费**）→ 20:00 衔接 Marunouchi Tokyo Station Square 段。

  - **建筑师**：**辰野金吾（Tatsuno Kingo，1854–1919）+ 葛西万司**（1914 年原始设计）；**JR 东日本建筑设计事务所 + 田村明日香等**（2012 年完全复原工程主持）；施工：鹿島建設 / 大林组 / 清水建設 三社共同体（2007–2012）。

  - **设计要点**：335 米长红砖站舍 + 白色花岗岩水平腰带（"辰野式 / 赤煉瓦样式"）+ 南北两个八角形穹顶（南北圆顶 / Dome）+ 中央部正面玄关 + 第三层；亚洲最大的红砖站舍；2003 年指定为国家重要文化财；2012 年战后完全复原工程为日本"近代建筑战后复原"代表作。
