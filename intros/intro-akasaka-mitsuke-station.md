# 赤坂见附站（赤坂見附駅 / Akasaka-Mitsuke Station）

- **类型**：交通节点（东京 Metro 银座线 + 丸之内线 双线交汇地下车站；与永田町站构内连通可换乘有乐町线 / 半蔵门线 / 南北线，事实上联通 Tokyo Metro 5 条线）
- **位置**：东京都千代田区永田町 2-11-1（11 番出口）/ 港区赤坂 3-1（10 番出口）；最近车站即本站，10 / 11 番出口位于外堀通沿线，与日枝神社山门步行约 8 分钟、与赤坂青野和菓子本店步行约 5 分钟。
- **历史与背景**：赤坂见附站是东京 Metro 银座线（1939 年虎之门 → 涩谷段开通时设站）与丸之内线（1959 年荻洼 → 池袋段于此交汇）的双线换乘站，是战前帝都高速度交通营团（帝都高速度交通営団，营団地下铁）→ 战后东京 Metro 体系下最早的几个十字交汇枢纽之一。**本站名「赤坂见附」由江户城外郭见附（外郭门）而来**：江户时代的江户城以本丸为中心向外有「内郭」与「外郭」两层城堡门防御体系，「见附」（みつけ）为外郭城门，36 处见附中赤坂见附是西南方向通往甲州街道（甲府城下町方向）的重要城门，1636 年由筑前黑田家修筑石垣，1872 年明治维新拆除大部分见附时仅留少量石垣（现存于站东北侧「赤坂见附跡」公园）。地下铁站名沿用旧城门地名，1939 年银座线开通时定名「赤坂见附」沿用至今。1959 年丸之内线扩建至此，与原银座线站台呈上下交叉立体换乘结构，是东京最早的几条「同台换乘」地下铁站之一（与日比谷站、池袋站并列）。1974 年同站构内被半蔵门线 / 有乐町线 / 南北线的「永田町站」连通，至此一站可换 5 条 Tokyo Metro 线。地理上本站邻接「山王下御旅所」（山王祭神舆中转点）与日枝神社入口，每年 6 月山王祭神舆通过外堀通时本站 11 番出口为主要观礼出口之一。CSV 备注里"我之间住过的地方"指的是用户在过往东京之行中曾住宿赤坂见附站附近，对该站本身的城市肌理与高密度餐饮 / 商务环境留有印象。
- **看点**：
  1. **银座线 + 丸之内线立体同台换乘结构**：上层银座线（A 月台、向涩谷方向 / B 月台、向浅草方向）+ 下层丸之内线（C 月台、向池袋方向 / D 月台、向荻洼 / 中野富士见町方向），可在同站台对面换乘；是东京地铁最高效的几个换乘节点之一。
  2. **「赤坂见附跡」石垣**：站东北侧（11 番出口出地面后向皇居方向步行约 3 分钟）保留的江户外郭见附石垣遗迹，是少有可在地表直接触摸的江户城门遗迹。
  3. **山王下御旅所**：站 11 番出口正对外堀通对面 200 米，山王祭神幸祭日（具体日期以官方公告为准）神舆中转点，平日仅为小型空地。
  4. **「Belle Vie 赤坂」站直商业楼**：与本站 B1 直通的购物 / 餐饮综合楼，B1 美食街 + 1F 杂货 + 2–6F 服装餐饮，午餐时段（11:30–13:30）为周边永田町商务客覆盖的实用餐饮带。
  5. **赤坂见附 KIOSK / Metro 商店**：站内月台与改札外的 KIOSK 是东京 Metro 限定周边（毛绒玩偶 / 文具）出售点之一。
- **推荐玩法**：从日枝神社结束开祭典礼神事后下山，沿西南侧稻荷参道走到外堀通，向北步行约 8 分钟即可到达赤坂见附站 11 番出口。在站内可拍摄一组「银座线 + 丸之内线立体同台换乘」机位（上下层月台同框，是东京 Metro 经典构图之一），再在 11 番出口外的赤坂见附跡石垣处停留 5 分钟回溯江户城外郭历史；若时间允许可绕到山王下御旅所对侧拍一张神舆中转点空景作纪念。本站不需要购入车票（TST 已覆盖），完成参观后可步行 5 分钟接赤坂青野和菓子本店。
- **票务 / 营业时间**：
  - 站内首班车 / 末班车时间：约 05:00–24:30（具体以东京 Metro 时刻表为准）。
  - 改札 / 月台 24 小时通行（首末班车时段除外）。
  - 票价（按段计）：东京 Metro 单程 ¥180–290（160–290 ICOCA / Suica）；本日已激活 TST 72 小时券，**TST ✅ 已覆盖**，无需另行购票。
  - Belle Vie 赤坂商业楼营业时间约 10:00–21:00（餐饮区延长至 23:00），定休按各店家。
- **交通到达**：
  - **从前一地点（日枝神社）**：从神社南参道下山后沿外堀通向北步行约 8 分钟可达 11 番出口；或乘 Metro：日枝神社最近赤坂站乘东京 Metro 千代田线 → 国会议事堂前 → 半蔵门线一站到永田町站构内换乘抵达赤坂见附站（约 5 分钟、TST ✅）。**首推步行**（同段更快、可顺路看见附跡石垣）。
  - **从两国 COGO RYOGOKU 出发（参考，本日不直接适用，参拝完日枝神社后才到达本站）**：都营大江户线 两国 → 青山一丁目（约 22 分钟、TST ✅）→ 换东京 Metro 银座线 青山一丁目 → 赤坂见附（一站、约 2 分钟、TST ✅）；合计约 25 分钟、TST ✅ 全段覆盖。
- **周边联动**：步行 5 分钟内可达 `Akasaka Aonono Wagashi Main Store`（赤坂青野和菓子本店，本日下一站，沿外堀通南向 5 分钟）；步行 8 分钟可达 `Hie Shrine`（日枝神社）；步行 10 分钟内可达「迎宾馆赤坂离宫」（参观需在线预约 / 周三 / 周六 / 周日开放，本日周日预约可能可入但需事先在线预约）、「TBS 放送中心赤坂 BIZ TOWER」、「赤坂 ACT Theater」。
- **消费档次**：站内通行免费（TST ✅）；若在站内 KIOSK 购买饮料 / 周边 ¥200–800、在 Belle Vie 赤坂吃午餐 ¥1,000–2,000；本站作为「住宿过的怀旧打卡点」单独消费几乎为零。
- **注意事项**：
  - 站内月台 / 换乘通道为东京最繁忙站之一（早晚高峰人流密集），拍照请避开人流主线、勿在月台边缘停留。
  - 11 番出口出地面是外堀通主车道，过马路请走斑马线 / 信号灯（Akasaka 街道车流较快）。
  - 见附跡石垣处无围栏，可触摸但请勿攀爬；周边为政府机关核心区，若见警备人员请保持距离勿干扰。
  - 山王祭神幸祭日（具体日期以官方公告为准）外堀通将临时交通管制，此时本站 11 番出口可能改作神舆观礼专用通道。
