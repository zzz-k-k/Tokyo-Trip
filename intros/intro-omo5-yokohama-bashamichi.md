- **类型**：观景 / 酒店公共空间（46F 环状展望走廊「OMO Base」+ THE YOKOHAMA BAY 餐厅）
- **位置**：神奈川县横浜市中区北仲通 5-57-2（旧 The Tower Yokohama Kitanaka）；最近车站 港未来线「馬車道」站 **2a 出口直結**（地下通路直接接入 1 楼大堂），步行 0 分钟。

## 历史与背景

OMO5 横滨马车道 by 星野集团于 **2026 年 1 月 15 日开业**（行程 6/3 时已开业约 4.5 个月），是星野集团旗下都市观光酒店品牌「OMO」系列在横滨的首家据点。酒店入驻的建筑为「The Tower Yokohama Kitanaka」（北仲ノット，地上 58 层 / 高约 200 米），是港未来地区仅次于横滨地标大厦（296 米）的第二高建筑。星野集团将 46 楼整层改造为酒店大堂 + 公共空间「OMO Base」——一条环绕整层的 **360 度环状展望走廊**（海拔约 154 米），对**非住客完全开放**且**免费**，是横滨地标大厦 Sky Garden（69F）2026 年起暂闭后，港未来地区**唯一免费 + 对公众开放的高层展望空间**。46F 同层还设有 THE YOKOHAMA BAY 餐厅（午餐 / 晚餐 / 酒吧）、台球厅与公共休息区，空间宽敞，本地人多在此观景，i 人无社交压力。

**导航注意**：搜「OMO5 横滨马车道」（搜旧名 Oakwood / The Tower Yokohama Kitanaka 会显示永久歇业误导）→ 一楼**靠大马路 + 马车道地铁站方向**有一处隐蔽的玻璃感应门 → 入门左转即电梯口 → 升 46F。

## 看点

1. **46F 环状展望走廊「OMO Base」（360 度全景 / 海拔 154 米 / 免费）**：环绕整层的公共展望走廊，可一览横滨港、港未来摩天楼群、红砖仓库、大栈桥、远处富士山轮廓（晴日）与东京晴空塔方向。是 Sky Garden 暂闭后港未来地区的**展望首选**。
2. **THE YOKOHAMA BAY 餐厅 / 酒吧**：46F 同层，面朝横滨港的高层餐饮，午餐 / 晚餐 / 鸡尾酒均可（人均 ¥2,000–6,000）；非住客可直接入座。
3. **台球厅 + 公共休息区**：46F 同层的休闲空间，适合观景间隙休息。
4. **港未来第二高建筑外观**：从地面仰望 The Tower Yokohama Kitanaka（58 层 / 约 200 米），是港未来天际线的重要组成。
5. **馬車道站 2a 出口直結**：地下通路直接接入 1 楼，全程不出地面，雨天最佳展望选择。

## 推荐玩法

从红砖仓库 / HAMMERHEAD 步行 8 分钟到马车道站方向，或乘港未来线馬車道站 2a 出口直结 → 1 楼玻璃感应门入内 → 左转电梯口 → 升 46F → 环状展望走廊 360 度慢走一圈（15–20 分钟）→ 如时间允许在 THE YOKOHAMA BAY 点一杯饮品面海休息（20 分钟）→ 下楼出站。整段约 30–45 分钟。**本日地标塔 Sky Garden 暂闭，展望需求全数在此完成。**

## 票务 / 营业时间

- 46F「OMO Base」展望走廊：**免费、对非住客开放**。
- 营业时间：参考北仲ノット运营惯例，约 **09:00–22:00**（**待用户实地确认**）。
- THE YOKOHAMA BAY 餐厅：午餐 11:30–14:00 / 晚餐 17:30–22:00 / 酒吧 14:00–22:00（具体以现场为准）。
- 官网：https://hoshinoresorts.com/en/hotels/omo5yokohamabashamichi/（**待用户实地确认**）。

## 交通到达

- **从前一地点 横滨红砖仓库 / HAMMERHEAD 出发**：步行约 8 分钟到馬車道站方向，或乘港未来线馬車道站 2a 出口直结（¥0 步行 / 已在站内），TST ❌。
- **从两国 COGO RYOGOKU 出发（参考，本日不适用）**：JR 总武线快速 两国 → 东京 → 京滨东北线 → 横滨 → 港未来线横浜 → 馬車道（1 站、¥190）→ 2a 出口直结；合计 ≈ 50 分钟、¥850（无 TST 覆盖）。

## 周边联动

步行 8 分钟内可达「横滨红砖仓库」「横滨 HAMMERHEAD」；步行 5 分钟可达「横滨地标大厦」（地下通道经 Queen's Square）；港未来线馬車道站直结，一站可达横滨站（¥190）或元町・中華街站（¥190）。

## 消费档次

展望走廊免费；THE YOKOHAMA BAY 餐厅人均 ¥2,000–6,000（午餐 / 晚餐 / 鸡尾酒）；不消费仅观景 ¥0。
