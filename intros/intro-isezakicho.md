- **位置**：神奈川县横浜市中区伊勢佐木町 1–7 丁目（南北纵贯约 1.5 公里）；**北端入口**：横滨市中区伊勢佐木町 1 丁目 1（伊勢佐木モール北口，紧邻 JR 关内站南口）；**南端**：横滨市中区伊勢佐木町 7 丁目（接近黄金町 / 阪東橋）。最近车站：JR 京滨东北线 / 根岸线「**关内**」站南口步行约 5 分钟到 1 丁目北端入口（伊勢佐木モール），或横滨市营地铁蓝线「**伊势佐木长者町（伊勢佐木長者町）**」站步行约 3 分钟到 3 丁目中段，或京急本线「**日ノ出町**」站步行约 5 分钟到 7 丁目南端；港未来线「**馬車道**」站步行约 8 分钟亦可。

## 历史与背景

伊势佐木大街是横滨开港（1859 年）后最早形成的商业街之一，被广泛认为是**日本第一条引入西式店铺布局的商店街**。1859 年横滨港开港后，外国商人在山下町与元町一带的居留地开设洋行，紧邻居留地外侧的伊势佐木町则承担起"日本人面向外国货的购物前线"角色 —— 1872 年（明治 5 年）伊势佐木町被横滨府正式指定为商业街，街区规划首次引入欧美的"沿街连排店铺 + 玻璃陈列窗"形制，被称为"日本商店街的鼻祖"。**大正期（1912–1926）**伊势佐木町与东京的银座并称为日本"**关东两大繁华街**"，成为关东地区最重要的购物 + 夜生活集散地：松屋（1903 年开业，伊势佐木町分店）、不二家（伊勢佐木店）、有邻堂（1909 年伊势佐木町本店）、勝烈庵（1927 年伊势佐木町本店）等老铺均在此期形成，至今多家仍在原地营业。**昭和初期（1926–1945）**伊势佐木町进一步扩张到 7 丁目，是横滨电影院 / 剧场最密集的街区。**战后（1945–1950）**因横滨港被驻日盟军征用、伊势佐木町亦被划入美军占领区，整段商店街曾被改造为美军娱乐街，文化身份一度断裂。1950 年代起在横滨市与商店街振兴会主导下逐步恢复，1978 年伊势佐木モール（1-2 丁目北段）正式形成"歩行者天国"步行街区（车辆禁入），是日本最早的拱廊式步行商店街之一。**1968 年青江三奈演唱、川内康範作词、铃木庸一作曲的『伊勢佐木町ブルース』**（伊势佐木町布鲁斯）发行后席卷全日本，成为昭和歌谣黄金期的代表作之一 —— 那句"あなた知ってる 港ヨコハマ 伊勢佐木あたりに"让这条街区在战后日本流行文化中具有了特殊的"昭和港湾夜色"符号位置；至今北端 1 丁目入口处仍立有**「伊勢佐木町ブルース歌碑」**（按按钮可播放原曲），是商店街的核心打卡点。进入 21 世纪后，随港未来 21 号街区的新港湾商业崛起，伊势佐木町相对沉降，但仍是横滨港湾区**最完整的"昭和拱廊商店街 + 现存大正 / 昭和老铺街区"**。

## 看点

1. **「伊勢佐木町ブルース」歌碑（伊勢佐木モール北口 / 1 丁目入口）**：1991 年立，正面镌刻青江三奈 1968 年代表曲的开头歌词、按钮按下播放约 30 秒原曲片段；是商店街的核心打卡点 + 昭和歌谣文化遗产。
2. **伊勢佐木モール 1-2 丁目（北端）拱顶老街景**：1978 年形成的步行者天国，拱顶上方挂着各时代灯笼 + LED 招牌，1909 年开业的有邻堂伊势佐木町本店、1903 年松屋伊勢佐木店旧址、1927 年勝烈庵伊势佐木町本店等老铺集中于此段；夜晚 18:00–20:00 灯光最完整。
3. **3-7 丁目（南端）昭和老电影院 + 风俗街区**：相比 1-2 丁目的"光鲜步行街"，3 丁目以南是更原始的横滨夜街 —— 横滨ジャック&ベティ（旧横浜日劇，1952 年开业、现仍在营业的独立电影院）等共存；本次行程**不深入南端**。
4. **与中华街联动（步行 8–10 分钟可达 善邻门 + 关帝庙 + 妈祖庙 + 横滨大世界）**：伊势佐木町北端与中华街南端之间步行约 8–10 分钟，是 Day 1 收尾段最自然的"晚餐 + 商店街漫步"组合。
5. **昭和老铺餐饮（不二家伊勢佐木店 / 勝烈庵 1927）**：大正末期至昭和初期的老餐厅至今在原地营业，可作晚餐选择（人均 ¥1,200–3,000）—— 勝烈庵的炸猪排是横滨当地"昭和洋食"代表。

## 推荐玩法

从 GuestHouse FUTARENO 卸完行李后约 18:30 出门 → 步行 / 横滨市营地铁 / JR 京滨东北线一站到关内站（5–15 分钟）→ 关内站南口出站步行 5 分钟到伊势佐木モール 1 丁目北口 → **先在「伊勢佐木町ブルース歌碑」前拍 5 分钟（按下按钮听 30 秒原曲）** → 沿伊势佐木モール 1-2 丁目向南漫步 15 分钟（拱顶灯光 + 老铺外墙 + 拱廊商店街拍夜景）→ 在 1-2 丁目内或顺道折回中华街选一家晚餐（不二家伊勢佐木店 / 勝烈庵 / 中华街老店三选一，人均 ¥1,500–3,000，约 60 分钟）→ 晚餐后回到伊势佐木モール 拱顶下买一瓶崎阳轩烧卖（1928 年创业的横滨地产）作为 Day 2 早餐 / 行李途中的零食 → 21:00 回关内站 → 21:30 回 FUTARENO。整段约 2.5–3 小时（含晚餐），不含晚餐则约 1 小时。

## 票务 / 营业时间

- 商店街本体**全年 24 小时开放、不收费**；伊勢佐木モール（1-2 丁目）实施"歩行者天国"（步行者优先），平日 11:00–20:00 / 周末 11:00–18:00 时段车辆禁入、其余时段允许配送车辆通行。
- 各店铺营业时段不同：日间商店多为 10:00–20:00、餐饮店多为 11:00–22:00、Live House / Bar 多为 18:00–24:00 / 翌 02:00。
- 「伊勢佐木町ブルース歌碑」按钮播放服务白天 + 夜间均可使用，无收费。
- 商店街官网：https://www.isezakimall.or.jp/（**待用户实地确认**当期活动 / 临时关店信息）。

## 注意事项

- **6 月梅雨季**：伊勢佐木モール（1-2 丁目）全段为有顶拱廊（アーケード商店街），即使大雨亦可全程步行；仅"关内站 → モール 1 丁目"5 分钟过渡段无拱顶覆盖，请带折伞。
- **现金优先**：商店街内多数大店支持主要电子支付，但**老铺与 3 丁目以南的小店多数仅收现金**；建议夜游前在便利店 ATM 取出 ¥10,000 现金备用。
- **3 丁目以南风俗店避让**：本次行程**仅游览 1-2 丁目北端**，不深入 3 丁目以南。如不慎走到 3 丁目以南遇到拉客，礼貌摇头说「いいえ、結構です」即可离开。**夜晚 22:00 后避免单独深入南端**。
- 拍照：商店街内允许个人手持摄影（拱顶 / 招牌 / 歌碑），但**不允许对店内陈列 / 工作人员 / 顾客面部正面拍摄**。
- **夜归节奏**：本子区域作为 Day 1 收尾，21:00–21:30 应主动结束并返回 FUTARENO。

## 类别专属字段

### 商业街 / 夜游专属字段

- **街区规模 / 开放时段**：南北纵贯约 **1.5 公里**（1–7 丁目），**全年 24 小时开放、不收费**；夜晚 22:00 后 1-2 丁目（北端）部分店铺逐渐关闭。
- **街区分段**：
  - **1-2 丁目（北端，伊勢佐木モール）**：本次行程**核心游览段**；歩行者天国步行街 + 拱顶 + 老铺 + 「伊勢佐木町ブルース」歌碑；夜晚 18:00–21:00 氛围最佳。
  - **3-4 丁目（中段）**：连锁店 + 横滨ジャック&ベティ独立电影院 + 部分老式 Cafe；可作过渡段。
  - **5-7 丁目（南端）**：昭和后期遗存的 Bar / Cabaret / 风俗店 / 小型 Live House 集中段；本次行程**不深入**。
- **主营商品类目**：日间商店（百货 / 服饰 / 杂货 / 文具 / 化妆品）+ 餐饮（昭和洋食 / 拉面 / 寿司 / 中华 / 咖啡馆）+ 老铺特产（崎阳轩烧卖 / 不二家洋果子 / 有邻堂书籍）+ 夜间 Bar / Live House / 风俗（仅 5-7 丁目）。价格区间 ¥500–10,000。
- **夜游建议时段**：**18:00–21:00**（灯光全开 + 商业氛围 + 安全 + 拱廊步行街便利）。
