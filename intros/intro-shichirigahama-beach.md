# 七里滨（七里ヶ浜 / Shichirigahama Beach）

- **类型**：海岸 / 观景（湘南海岸代表段；葛饰北斎《富岳三十六景・相州七里濱》取景地；江之电沿海段标志性视野）
- **位置**：神奈川県鎌倉市七里ガ浜东 / 七里ガ浜；最近车站 江之电 **七里ヶ浜駅** 出站即正对海岸（步行 0–2 分钟，跨过国道 134 号即下沙滩）
- **历史与背景**：七里ヶ浜（七里滨）是相模湾北岸自 **稲村ヶ崎** 至 **小动岬（こゆるぎみさき）** 之间长约 2.9 公里的弧形海岸，名称来自江户期"七里 ≈ 2.7 公里"的旧里数计法（实测约 2.9 公里），是镰仓海岸最具辨识度的一段。**江户后期天保元年（1830 年）前后**，浮世绘大家 **葛饰北斎** 在其代表作《富岳三十六景》系列中以 **「相州七里濱」** 为题，描绘从稲村ヶ崎西望——海面、江之岛剪影、远处富士山三层叠加——的构图，使七里滨成为日本浮世绘"远眺富士"画系列中除 神奈川冲浪里 / 凯风快晴 / 山下白雨 之外最具地标识别度的一景。**江户末期至明治末年**，随着横须贺线 1889 年开通、江之电 1902 年开通，七里滨从渔村转型为 **湘南海岸观光休闲带的代表段**，被一并称为「湘南」（Shōnan）的核心地理符号；大正—昭和期与镰仓文人圈交叠，川端康成、芥川龙之介等都曾留下从镰仓海岸远眺富士的笔记。**战后 1960 年代起**，受美军驻日带来的加州冲浪文化影响，湘南海岸（特别是七里ヶ浜与稲村ヶ崎之间这段弧线）成为日本本土冲浪文化的发祥点之一，至今每周末仍是关东冲浪人最密集的练习场。**1990 年代以降**，随《SLAM DUNK》（井上雄彦，1990–1996 年连载）动画版（1993 年东映动画）以片头曲「君が好きだと叫びたい」镜头切到江之电沿线道口而成动漫圣地巡礼地标后，七里滨 → 镰仓高校前 一线晋级为关东国际游客密度最高的海岸段之一；2024 年 The First Slam Dunk 剧场版上映后，海外（特别是中港台 / 东南亚 / 欧美）粉丝持续涌入，至今平日下午沙滩外侧国道 134 号"江之电 + 海岸 + 江之岛"机位常有 50+ 人同框拍摄。**当地居民周末野餐与冲浪文化**——周六周日清晨 5:00–7:00 与黄昏 17:00–19:00 是本地家庭在沙滩铺野餐垫、儿童放风筝、冲浪者下海的高峰时段，与拍照游客互不相扰，已成为湘南海岸的固有日常。
- **看点**：
  1. **江之电 + 海岸 + 江之岛 三层叠景**：沿国道 134 号自七里ヶ浜駅向西步行 200–400 米可达多个无障碍机位，江之电单线沿海岸路面行驶，正好与海岸线 + 江之岛剪影构成横向三层；午后斜光（15:00–17:30）最佳
  2. **富岳三十六景「相州七里濱」原构图视角**：从七里ヶ浜駅向东 600–800 米至 稲村ヶ崎 公园高地，可在晴空万里 + 风向适合的极少数日子里同框看到「海面 / 江之岛 / 富士山」三层（**6 月梅雨季能见度低，看到富士的概率约 5–10%**，请将此项作为彩蛋而非保留期待）
  3. **沙滩本身（无入场限制）**：黑灰色火山砂质沙滩，长约 2.9 公里，可全长漫步；海水温度 6 月初约 18–20°C，未到泳期，但适合赤脚踩沙线
  4. **冲浪练习场视野**：七里ヶ浜中段（江之电 七里ヶ浜駅 与 鎌倉高校前駅 之间约 1.5 公里）是关东最常见的冲浪点之一，午后通常有 20–50 名冲浪者在浪线外侧排队等浪
  5. **WAVES 七里ヶ浜车站旁的「七里ヶ浜海岸公園」**：站前下沙滩之间有一段木栈步道与沿岸 ParkingLot，上方 2 楼有「Pacific DRIVE-IN」（自 1968 年开业的湘南海岸观光餐厅，本日不入店但可作为下午茶备援）
- **推荐玩法**：自 七里ヶ浜駅 出站后 → 跨国道 134 号步行 1 分钟下沙滩 → 沿海岸西行 300–500 米寻找"江之电 + 海 + 江之岛"三层构图机位（推荐自驾 / 步行至「七里ヶ浜海岸公園」西侧的栈道护栏处）→ 在沙滩上停留 15–20 分钟（坐下、看浪、拍冲浪者剪影）→ 折回七里ヶ浜駅站台再上江之电（藤沢方向 1 站约 2 分钟）→ 鎌倉高校前駅。整段建议 **40–60 分钟**；若要等夕阳金光 + 富士山 + 江之岛三联只需在站前栈道护栏前停留至 17:00 之后。
- **票务 / 营业时间**：
  - **沙滩入场**：**完全免费 + 24 小时开放**（无入园门 / 无关闭时间，但夜间无照明，6 月日落约 18:53 后建议撤离）
  - **泳场期**：**仅每年 7 月–8 月**为正式开放泳期，期间设临时海之家与救生员；**其他月份（含 6 月）仅可进入沙滩与浅水区，禁止下水游泳**（无救生员，且本段海域离岸暗流强劲，居民会主动劝阻外来游客下水）
  - **6 月不开泳但可冲浪**：冲浪者可全年使用沙滩与海面，但需自带板与穿防寒服（6 月初海水 18–20°C，需 3 mm 短袖或全防寒服）
  - **官方告示**：镰仓市观光协会 https://trip-kamakura.com/place/179.html / 七里ヶ浜地区告示，**待用户实地确认**最新泳场开放期与海象告示
- **交通到达**：
  - **从前一地点 高德院（长谷段收尾）**：长谷駅 → 江之电（藤泽方向 3 站）→ 七里ヶ浜駅，约 **8 分钟、¥260（TST ❌）**；下車後跨过国道 134 号即下沙滩，**步行 0–2 分钟**
  - **从两国 COGO RYOGOKU 出发（参考线，非本日动线）**：JR 总武线快速 → 东京站 → JR 横须贺线 → 鎌倉駅 → 江之电 鎌倉駅（藤泽方向 5 站）→ 七里ヶ浜駅，约 1 小时 25 分钟、¥1,430（TST ❌）
- **周边联动**：
  - **江之电 1 站约 2 分钟、¥200** 即可达 **鎌倉高校前駅**（本子区域下一站，SLAM DUNK 道口）
  - **江之电 2 站、向东（鎌倉方向）** 即可达 **稲村ヶ崎**（葛饰北斎「相州七里濱」原构图视角的高地公园，本次行程未列入保留地点，可作为夕阳富士山观景的备援加点）
  - 步行 / 江之电 6 站约 18 分钟可达 **江之岛站（江ノ島駅）**（江之岛参道 / 江岛神社，本次行程未列入保留地点）
- **消费档次**：**沙滩入场免费 + 江之电单段 ¥200–260 / 一日券 ¥800**；沙滩本身无消费点，海岸公园上方"Pacific DRIVE-IN"等观光餐厅人均 ¥1,500–2,500（本日不入店）；建议自带饮水与小食（站前 7-Eleven 在站东侧 100 米可补给）
- **注意事项**：
  - **6 月不下水**：本日（6 月 4 日）非泳期，海域无救生员配置且离岸暗流较强，**禁止脱鞋下水深及膝盖以上**；可赤脚踩沙线 / 浅水线
  - **国道 134 号过路**：从七里ヶ浜駅下沙滩需跨越国道 134 号（双向 4 车道），**仅在站前正规人行斑马线过路**，国道 134 号在午后冲浪 / 观光车流密集，禁止跨栏 / 跳栏过路
  - **拍摄机位与道路安全**：在国道 134 号沿岸拍江之电 + 海 + 江之岛三层构图时，**站位严格限制在人行道与栈道护栏内侧**，**禁止站到机动车道或中央分隔带上拍摄**——与下一站「鎌倉高校前道口」执行严格度等同（详见下一条目）
  - **6 月梅雨季现实提示**：① 阴天反而是「相州七里濱」版画意境最强的天气（北斎原画即为乌云 + 浪面 + 江之岛剪影）；② 雨天沙滩湿沉但仍可走，但木栈步道湿滑需谨慎；③ 富士山可见率 6 月梅雨季约 5–10%（远低于冬季 30–50%），不要把"看到富士"作为本子区域的保留期待
  - **海风强劲**：海岸常年北风 / 西南风 5–10 m/s，6 月平均风速更高，需带可压制头发的小帽 / 头巾，三脚架在海岸防风性能不足时易倒（且站前栈道禁止架三脚架）
  - **现金优先**：江之电站台仅有自动售票机（接受现金 + Suica / Pasmo / ICOCA 等全国互通 IC 卡），日本本土 IC 卡可直接刷闸；外国发行的非接触卡（含 Apple Pay 海外版）部分情况下不可用，**请以 Suica / Pasmo 为主**
  - **垃圾自带**：沙滩与栈道无垃圾桶（日本海岸普遍政策），所有垃圾须自带回；7-Eleven 出口处的回收箱亦不接受外带食品包装
- **海岸 / 观景类专属字段**：
  - **是否收费**：**完全不收费**（沙滩 + 海岸公园 + 栈道全开放）
  - **开闭时间**：**24 小时开放**（无门 / 无关闭时间，但夜间无照明且海岸暗流，建议日出至日落 30 分钟内活动）
  - **泳场期**：**仅 7 月–8 月**正式开放游泳；**其他月份（含本日 6 月）禁止下水游泳**，但可在沙滩漫步、冲浪、踩浅水线
  - **最佳季节 / 时段**：**全年可游**；**最佳时段 15:00–17:30 午后斜光**（江之电 + 海 + 江之岛 三层构图最具立体感）；**梅雨季阴天**反而最贴近葛饰北斎《富岳三十六景・相州七里濱》原构图意境；**冬季晴日**远眺富士山可见率最高（30–50%）
  - **是否允许带饮食 / 摄影**：允许带饮食（自带垃圾回）；**摄影允许**含手机 + 相机；**禁止三脚架架设在车站栈道与国道 134 号沿岸**（道路安全 + 行人通行优先）
  - **特别意义**：**葛饰北斎《富岳三十六景》第二十二景「相州七里濱」取景地**——视野越过江之岛能看到富士山的经典构图原点
