- **类型**：购物（山本耀司旗下无性别 / 街头支线 Ground Y 直营店；S'YTE 的线下替代方案）
- **位置**：东京都涩谷区宇田川町 15-1 **涩谷 PARCO**（Ground Y 为涩谷 PARCO 直营店之一；楼层以现场为准）；最近车站「**涩谷**」站步行约 5–7 分钟（Metro 段 TST ✅、JR 段 TST ❌）。
- **关于 S'YTE 的重要说明**：你提到想逛「涩谷 PARCO 的 S'YTE」—— 需要澄清：**S'YTE 是山本耀司的「线上限定（online-only）」品牌，没有独立实体店**，仅在官方电商 THE SHOP YOHJI YAMAMOTO 销售。涩谷 PARCO 内可实际逛到的山本耀司支线实体店是 **Ground Y**（无性别 / 无年龄界限的当代街头线，黑色基底 + 大廓形 + 大量动漫·艺术家联名）。因此本条目以 **Ground Y 涩谷 PARCO 店** 作为「S'YTE 需求」的现实落地替代；若你执意要 S'YTE 单品，只能在官方线上购买（可在日期间下单寄到酒店）。
- **历史与背景**：Ground Y 是 Yohji Yamamoto 社推出的「ジェンダーレス・エイジレス（无性别·无年龄）」支线，以黑色为基底、长廓形 / 不对称剪裁为标志，价格远低于巴黎主线，是年轻客群进入「山本耀司世界」的主要入口。Ground Y 直营店覆盖 GINZA SIX / 拉福莱原宿 / **涩谷 PARCO** / 心斋桥 PARCO / 名古屋 PARCO，常与《咒术回战》《鬼灭之刃》《EVA》《新世纪福音战士》及多位艺术家（田名网敬一等）推出联名。
- **看点**：
  1. **「一眼山本」的平价正品**：黑色长衬衫 / 大廓形外套 / 解构剪裁，保留主线基因但价格亲民。
  2. **动漫·艺术家联名**：本店 / PARCO 限定的联名 T 恤 / 单品，是 Ground Y 最具收藏性的品类。
  3. **无性别剪裁**：适合各种体型自由尝试 oversize 廓形。
- **推荐玩法**：与同在涩谷 PARCO 6F 的 [THE★JOJO WORLD](intro-the-jojo-world-shibuya-parco.md) 一并逛。预留 20–30 分钟试穿 + 选购；T 恤 / Cut&Sewn ¥10,000–20,000、衬衫 ¥20,000–40,000、外套 ¥40,000–80,000，是「全新正品又不太贵」的最佳平衡。
- **票务 / 营业时间**：
  - 入场免费；营业时间随涩谷 PARCO（通常 11:00–21:00）。
  - 免税：护照单店 ¥5,000+ 可办免税。
  - 具体楼层 / 当日营业以涩谷 PARCO 楼层导览为准 —— 待用户实地确认。
- **交通到达**：涩谷站步行 5–7 分钟到涩谷 PARCO。
- **周边联动**：涩谷 PARCO 6F THE★JOJO WORLD + 动漫 IP 层；Day 6 涩谷段的涩谷 SKY / Mandarake / RAGTAG。
- **消费档次**：¥10,000–80,000 / 件。
- **注意事项**：
  - **S'YTE 无实体店**，请勿在涩谷 PARCO 找 S'YTE 专柜；线下能买的是 Ground Y。
  - 联名款常售罄 / 限购，热门款建议开店即到。
