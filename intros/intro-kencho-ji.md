# 建长寺（建長寺 / Kencho-ji）

- **类型**：寺院（临济宗建长寺派大本山，镰仓五山第一位 / 日本第一座纯禅宗专门道场）
- **位置**：神奈川県鎌倉市山ノ内 8；最近车站 JR 横须贺线 **北鎌倉駅 南口** 步行 15 分钟（约 1.2 公里，沿県道 21 号「鎌倉街道」南行）；从明月院出门后继续沿鎌倉街道南行约 800 米 / 步行 12 分钟即至总门
- **历史与背景**：建长寺由镰仓幕府第五代执权 **北条时赖** 于 **建长五年（1253 年）** 创建，是日本临济宗建长寺派大本山、**镰仓五山第一位**（圆觉寺为第二位）。开山祖师为南宋来日禅僧 **兰溪道隆（中国南宋来日禅僧 / 大觉禅师）**——其受北条时赖之招、由中国南宋杭州径山一系直接将禅林「七堂伽蓝制」与五山修行体系移植入日本，建长寺即为 **日本第一座纯禅宗专门道场**。境内伽蓝按南宋禅林「七堂伽蓝制」**南北中轴线一线**布置：**总门 → 三门 → 佛殿 → 法堂 → 唐门 → 方丈**，是 **日本现存最早的禅宗式七堂伽蓝完整样本**，对后世的圆觉寺、建仁寺、东福寺等五山禅刹伽蓝构成产生决定性影响。本寺亦是 **日本最早的禅宗式庭园** 的所在地——方丈背后的「**蘸碧池（さんぺきち）**」由开山兰溪道隆亲自设计的池泉式庭园，国指定名胜，是日本园林史上禅宗样庭园的开山实例。寺址原是镰仓「**地狱谷（じごくがやつ）**」——镰仓幕府的处刑场，北条时赖以镇魂为愿在此立寺，故 **建长寺本尊为地藏菩萨而非释迦如来**（关东禅刹中罕见的独特身份）。本寺历经 **1293 年永仁地震、1426 年应永大火、1923 年关东大震灾** 等多次受灾重建，现存伽蓝大半为江户期再建：**三门** 为安永四年（1775）二层楼门再建；**佛殿** 是宽永十九年（1647）由三代将军 **德川家光** 从增上寺移筑而来的旧崇源院（二代将军秀忠正室阿江）灵屋（江户德川家祈愿所）；**法堂** 是文化十一年（1814）再建，关东最大木造法堂。境内保存 **国宝梵钟**（建长七年 / 1255 年铸，关东最古之一）+ 多件重要文化财（佛殿、唐门、梵钟、兰溪道隆顶相图等）。境内最深处的 **半僧坊（はんそうぼう）** 是明治期奉迁入境的山中神社、建长寺镇守神，登 245 级石阶可俯瞰镰仓海岸 + 远眺富士山，是镰仓内陆段唯一的高地展望点。
- **看点**：
  1. **三门（山门，1775 安永四年再建）**：二层楼门，悬「**建長興国禅寺**」扁额；二层供奉五百罗汉与释迦三尊，平时不开放（特别公开期可登），是镰仓五山最具仪式感的中轴线起点
  2. **佛殿（1647 移筑 / 重要文化财）**：宽永十九年（1647）由德川家光从增上寺移筑的旧崇源院灵屋，桃山样式厚重屋顶 + 内部细密漆彩，**内安本尊地藏菩萨像**——建长寺寺址原为镰仓「地狱谷」处刑场，北条时赖以镇魂为愿建立本寺，故本尊非禅宗常见的释迦如来而为地藏菩萨，是本寺最具独特身份的核心
  3. **法堂（はっとう，1814 再建）**：**关东最大木造法堂**；天井「**云龙图**」由小泉淳作于 **2003 年描绘完成**（建寺 750 周年纪念），仰看双龙绕梁、笔触细密，是镰仓现代寺院艺术的代表，**全室内不受天气影响**
  4. **唐门（からもん，1647 桃山样式 / 重要文化财）**：与佛殿一同从增上寺移筑的桃山样式四足门，旧崇源院灵屋附属，金箔与漆彩残存丰富，2011 年完成大规模修复后重现近世初期的华丽风貌
  5. **方丈 / 龙王殿与蘸碧池庭园（国名胜）**：方丈是法事 / 接客的主厅，背后池泉「**蘸碧池**」由开山兰溪道隆亲手设计，是 **日本最早的禅宗式庭园** 之一；可在方丈缘廊静坐 15 分钟观池水与后山新緑相映，是建长寺最静谧的精神视角
  6. **半僧坊**：境内最深处的山中神社，登 245 级石阶至展望台可俯瞰镰仓街道与相模湾、远眺富士山（晴日可见率冬季 30–50%，6 月梅雨季 5–10%）；天狗像群与朱色社殿掩映杉林之中，是镰仓内陆段唯一的高地展望点
  7. **国宝梵钟「洪钟」**：钟楼内悬建长七年（1255 年）铸造的国宝梵钟，是 **关东最古的梵钟之一**，钟体铭文「建長禅寺」由开山兰溪道隆亲笔
- **推荐玩法**：从明月院出门后沿鎌倉街道南行 12 分钟（约 800 米）至建长寺总门 → 入门后 **沿南北中轴线一线参观**（总门 → 三门 → 佛殿地藏菩萨参拝 → 法堂仰看天井云龙图 → 唐门外观 → 方丈缘廊与蘸碧池庭园静坐 15 分钟）→ 在方丈购朱印 / 御守 → 若体力允许从方丈左侧山道登 **半僧坊** 245 级石阶（来回约 40 分钟，展望台可俯瞰镰仓海与镰仓内陆山线）→ 自总门出，沿鎌倉街道继续南行约 800 米 / 步行 12 分钟到 **鶴岡八幡宮** 三之鸟居衔接镰仓中心段。整段建议预留 **60–90 分钟**（不含半僧坊）/ **90–120 分钟**（含半僧坊）。
