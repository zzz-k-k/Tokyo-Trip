- **类型**：商业街 / 御宅文化街区
- **位置**：东京都千代田区外神田 / 神田佐久间町 / 神田练塀町一带（以"秋叶原中央通"为南北主轴，东达昭和通、西至昌平桥通、南达万世桥、北至末广町交差点，约 600 × 400 米街区）；最近车站 JR 山手线 / 京浜东北线 / 总武线「秋叶原」站电器街口出站即入、东京 Metro 日比谷线「秋叶原」站步行 1 分钟、つくばエクスプレス「秋叶原」站直通、东京 Metro 银座线「末广町」站步行 3 分钟（中央通北端）、都营新宿线「岩本町」站步行 5 分钟。

## 历史与背景

秋叶原（俗称"アキバ"/ Akiba）所在的外神田一带在江户时代是商人与町人聚居的"神田佐久间町"，1869 年（明治 2 年）冬旧幕府旧领发生大火，明治政府在此划出"火除地"作防火带，并在内修建"秋叶大権現"（秋叶神社）祈求镇火 ——"秋叶原"地名即源自该社。明治后期火除地被改造为青果市场（神田青果市场），秋叶原成为东京三大青物集散地之一。1945 年 3 月 10 日东京大空袭后整片市街化为焦土；战后 1949 年随着"东京电机大学"（前身电气工业讲习所）与神田周边收音机零件商人聚集，神田青果市场旁的高架下空间被改造为"秋叶原ラジオセンター"（无线电中心，1949 年开业），是秋叶原"电器街"的物质起点。1950 年代神田青果市场迁出（市场用地后来成为秋叶原 UDX / Yodobashi-Akiba 旧址），整片街区彻底转型为家电 / 电子零件批发市场，1960–70 年代是日本"白电黑电"消费爆炸期的最大单点市场。1980 年代 PC 自作热潮把秋叶原从"白电"推向"电脑零件"，1990 年代《新世纪福音战士》《セーラームーン》等动画 + ガレージキット（GK 模型）+ 同人志的兴起使秋叶原成为"御宅族（Otaku）"亚文化圣地——1998 年 Animate 秋叶原店开业、2001 年第一家女仆咖啡「Cure Maid Café」诞生。2000 年代 AKB48（2005 年于秋叶原 UDX 内设专属剧场，至今每日公演）+ 偶像握手会 + 2005 年中野独史《電車男》改编电视剧 + 2011 年《Steins;Gate》游戏化使秋叶原从"日本国内御宅圣地"升格为"国际游客必访的日本流行文化地标"。今日秋叶原是混合"电器街 + 御宅街 + 偶像 / 女仆咖啡街 + 国际游客购物街"的复合街区，每周日 13:00–18:00（夏季至 18:00 / 冬季至 17:00）中央通实施"步行者天国"封路，是日本最具识别度的青年亚文化都市空间。

## 看点

1. **秋叶原中央通**：南起 JR 高架万世桥、北至末广町交差点，约 600 米的御宅街主轴；两侧密布动漫店 / 模型店 / 中古游戏店 / 偶像 / 女仆咖啡，是秋叶原的"门面"。
2. **三大无线塔（ラジオ会館 / ラジオセンター / ラジオデパート）**：秋叶原战后电器街的三大物质起点，详见下文 Akihabara Electric Town 单独条目。
3. **AKB48 剧场（秋叶原 UDX 内）**：日本最大女子偶像团体的常驻剧场，每日 18:30 公演（演出门票需事先抽选）；外观打卡 + 周边商店随时可访。
4. **女仆 / 偶像 / 主题咖啡群**：以"@home cafe"（中央通 Don Quijote 4F）为代表的女仆咖啡 + 各种动漫主题咖啡（Gundam Café 已闭店；现存 Pokemon Café 浅草外延 / Capcom Café 池袋等）；秋叶原街内现存约 30 家女仆 / 偶像咖啡。
5. **唐吉诃德 秋叶原店（驚安の殿堂 ドン・キホーテ）**：中央通中段 8 层综合杂货 + 4F"@home cafe"+ 8F AKB48 剧场，**国际游客购买免税商品的主力据点之一**。
6. **神田明神（外延，步行 5 分钟）**：730 年创建的江户总镇守，因近年御宅文化授予"IT / 动漫绘马"而成为"御宅圣地"；时间允许可外延打卡 30 分钟。

## 推荐玩法

从 JR 秋叶原站电器街口出闸，先在站前广场拍秋叶原 UDX + 电器街口对面 Yodobashi-Akiba 一镜；沿中央通北行约 5 分钟到 Sofmap 总店 / Animate 秋叶原 / Don Quijote 入口处，**热身一圈 30–45 分钟**（看店面 + 拍街景但不深入购物）；接着进入电器街段（下文）→ Mandarake → Pop Life M's。**14:00–16:00 为最佳访问时段**（避开早上 11:00 前部分店铺未开 + 避开 17:00 后人流爆炸），**16:00 准时撤离前往水道桥**。

## 票务 / 营业时间

街区本身全天开放、全年免费、无入场限制；**步行者天国**（中央通封路）每周日 13:00–18:00（夏季至 18:00 / 冬季缩短至 17:00），本次 6/5（周五）不在步行者天国时段；街区内各家店铺营业时间各异，多数 10:00–22:00 或 11:00–20:00；女仆咖啡 / 主题咖啡 11:00–22:00；多数店铺**全年无休**或仅 1/1 休业。

## 交通到达

- 从前一地点 东京国立博物馆（上野公园北端）：步行 5 分钟到 JR「上野」站不忍口 → JR 山手线 / 京浜东北线 上野 → 秋叶原一站、约 2 分钟、¥150（TST ❌，短距 JR）；或全程步行 15 分钟（沿中央通南下，路过昌平桥）。**推荐 JR 山手线一站**（最直接）。
- 从两国 COGO RYOGOKU 出发：都营大江户线 两国 → 上野御徒町 → 步行 8 分钟南下到秋叶原中央通北端；或 JR 总武线 两国 → 秋叶原（约 5 分钟、¥160、TST ❌）。**推荐 JR 总武线一站**。

## 周边联动

步行 5 分钟内 `Akihabara Electric Town`（电器街，本身在秋叶原街区内）、`Mandarake Complex`（外神田 3 丁目，北行 6 分钟）、`Pop Life Department M's`（外神田 1-15，电器街口步行 2 分钟）；步行 15 分钟内 `Ueno Park`（北行 1.2 公里）。秋叶原 → 水道桥（衔接 Tokyo Dome）：JR 总武线 → 御茶水 + 中央线一站、6 分钟，或都营新宿线 + 三田线（TST ✅）12 分钟。

## 消费档次

街区本身免费；女仆咖啡入场费 ¥500 + 饮品 ¥800–1,200 + 拍照套餐 ¥1,000 / 张；主题咖啡套餐 ¥2,000–3,500；唐吉诃德 / Yodobashi 商品价格不一；街内简餐 ¥800–1,500 / 餐；整段（不含 Mandarake / M's 购物）人均 ¥1,000–4,000。

## 注意事项

- **街拍 / 摄影规则**：中央通公共空间允许拍照，但**对女仆 / 偶像（含发传单的店员）拍照前必须征得同意**——不少街头店员胸前会挂"撮影禁止"牌；女仆咖啡 / 主题咖啡内部禁止对女仆 / 食客拍照（仅可在指定背景前付费拍合照）。违反将被店员当场制止 / 被请离街区。
- **拉客（キャッチ）应对**：电器街口 / 中央通会有"AV / 高额ぼったくり"店铺的拉客（多为黑人 / 东欧人男性 + 看似"游客向" 的看板），**一律忽略 + 直行；切勿跟随到楼上店铺**。正规店铺都在街边一楼且有店牌。
- **现金 / IC 卡**：电器街老店与小店仍以现金为主；秋叶原中央通连锁店与免税柜支持 Suica / PASMO / Apple Pay / 主流国际信用卡；建议出门前换好 ¥10,000–20,000 现金。
- **6 月梅雨季**：街区有大量高架下 / 商店街顶棚，但 Mandarake / Pop Life M's 之间需露天步行 3–5 分钟，**带折叠伞**。
- **18:00 棒球场硬锚点**：本街区是 Day 3 下午的核心，但**严禁滞留至 17:00 之后**——从秋叶原到水道桥的 JR / 三田线段虽短（≤ 12 分钟），但 17:00 后 Tokyo Dome 周边地铁口 / 检票口会出现球迷大潮，安检 + 入场 ≥ 30 分钟，故**16:00 必须离开秋叶原**。
