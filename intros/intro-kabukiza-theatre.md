- **类型**：演出 / 剧场（"六月大歌舞伎"单幕席硬锚点）+ **建筑（隈研吾 + 三代目设计事务所，2013 第五代复原）**
- **位置**：东京都中央区银座 4-12-15 GINZA KABUKIZA（晴海通与昭和通交叉口东南角，与歌舞伎座大楼上层 KABUKIZA TOWER 共同构成的复合大厦）；最近车站 **东京 Metro 日比谷线 / 都营浅草线「东银座」站 3 号出口直结**（TST ✅，**单幕见席专用入口**为本出口直结向地面层北侧绕到楼东侧）；东京 Metro 银座线 / 丸之内线 / 日比谷线「银座」站 A7 出口步行 5 分钟（TST ✅）；JR 山手线 / 京浜东北线「有乐町」站中央口步行 8 分钟（TST ❌）。
- **历史与背景**：歌舞伎座（Kabuki-za）是 **1889 年（明治 22 年）11 月 21 日**由前 大阪每日新闻 / 时事新报社主笔 **福地源一郎（Fukuchi Gen'ichirō，号桜痴，1841–1906）** 提倡建设的"近代日本第一座专演歌舞伎的西洋式剧场"，第一代由建筑家 **高原弘造** 主持设计、千葉勝五郎主导兴建，在银座木挽町（现银座 4 丁目）以"和洋折衷"白漆木造样式开馆，开创了日本传统歌舞伎从江户期常演的"芝居小屋（市村座 / 中村座）"向"近代剧场"的过渡。1911 年（明治 44 年）第一代毁于火灾，**1911–1924 年第二代**由 **岡田信一郎（Okada Shinichirō，1883–1932）** 设计、以钢筋混凝土 + 桃山样式（破風唐破風 + 千鸟破风的城郭式屋顶）建成，是日本"和风钢筋混凝土"建筑的早期代表。1923 年关东大地震损毁第二代屋顶，1924–1945 年第三代（仍由岡田信一郎主持复原）作为昭和初期"东京最大剧场"运营到太平洋战争。**1945 年 5 月 25 日东京大空袭中第三代被烧毁**，1950 年代由 **吉田五十八（Yoshida Isoya，1894–1974）** 主持设计的 **第四代（1951 年开馆）**以战后版"桃山复古样式"重新开馆，承办了 20 世纪后半叶大量歌舞伎名角（市川團十郎 / 中村勘三郎 / 坂東玉三郎 等）的常演与"襲名披露"等核心仪式。第四代历经 60 年使用后于 2010 年闭馆，**2013 年 4 月 2 日第五代**（**现行**）开馆——**第五代由建筑家隈研吾（Kengo Kuma，1954– ）担任主创设计 + 三代目（吉田五十八）设计语言的继承团队 + 清水建設施工**，整体保留第四代桃山破风样式的正面主立面 + 大屋根，但在原剧场上方叠加 29 层的 KABUKIZA TOWER（写字楼 + 歌舞伎座向次代传承的资料馆 / 演员练习场 / 商铺）成为复合大厦；隈研吾主导的"传统正面 + 上方现代塔楼"组合获得日本建筑学会作品赏（2014），是平成末期"传统建筑现代复原"的代表作。剧场本体保留 **1,956 席**（一等 / 二等 / 三等席 + 4 楼单幕见席），上演按月度公演计划组织——**6 月公演称为「六月大歌舞伎」，按惯例分昼之部（11:00 起约 4 幕）+ 夜之部（16:00 起约 4 幕，**第五场即夜之部最后一幕 / 大切**）**；本次行程 6/8 选定的"第五场"按 KABUKI WEB 当周公告为准，演目梗概可能为「**勧進帳**」「**身替座禅**」「**棒しばり**」「**藤娘**」「**鏡獅子**」等通俗易懂、上演时长 60–90 分钟的代表演目（具体演目以官方当周公布为准 —— 待用户实地确认）。
- **看点**：
  1. **第五代（2013 年）正面主立面**：保留岡田信一郎 / 吉田五十八第二代 / 第四代的桃山破风样式 + 唐破风 + 千鸟破风组合 + 银座 4 丁目晴海通正面对望——东京都心最具识别度的"和风正面"建筑。
  2. **隈研吾的 KABUKIZA TOWER（29 层楼上叠加）**：原剧场上方叠加现代写字楼 / 资料馆，内部含歌舞伎博物馆 + 演员练习室 + 屋顶日本庭园（地下 2F 木挽町广场免费开放）；是隈研吾"在传统建筑上叠加现代塔楼"的代表作。
  3. **4 楼单幕见席（一幕見席 / Hitomakumi-seki）**：银座 4 丁目歌舞伎座大楼东侧的**専用エレベーター**（与 1–3 楼正席入口完全分离）直升 4 楼独立席区，约 96 座位（含立见区），**仅可观看一幕、不可中途换座、不可中途退场后再返场**——是日本传统歌舞伎"廉价短观"的代表制度，1923 年关东大地震后期由第三代歌舞伎座主导回归延续至今。
  4. **第五场（夜之部大切）演目**：按 KABUKI WEB 当周公告为准；典型为「勧進帳」「身替座禅」「棒しばり」「藤娘」「鏡獅子」等通俗代表作中的一齣，上演 **60–90 分钟**。
  5. **B2F 木挽町广场（免费开放）**：歌舞伎座大楼地下 2F 与东银座站 3 号出口直结的免费开放空间，含歌舞伎相关展示 + 物販 + 餐饮 + 屋顶日本庭园（5F），适合散场后顺路看。
  6. **歌舞伎博物馆 + 屋顶日本庭园**：5F 歌舞伎博物馆（¥600）与同层屋顶日本庭园（免费）展示历代歌舞伎座照片 + 演员遗物 + 道具样本。
