- **类型**：购物（浮世绘 / 新版画版画专门店；百年老铺）
- **位置**：东京都中央区银座 8-6-19 银座 KK ビル 1F（並木通沿线，银座 8 丁目交叉口附近）；最近车站 东京 Metro 银座线「新桥」站 1 号出口步行 3 分钟（TST ✅）、东京 Metro 银座线 / 丸之内线 / 日比谷线「银座」站 A2 / B5 出口步行 5 分钟（TST ✅）、JR 山手线 / 京浜东北线「新桥」站银座口步行 5 分钟（TST ❌）。
- **历史与背景**：渡辺木版美術画舗（S. Watanabe Woodcut Prints）由 **渡邊庄三郎（Watanabe Shōzaburō，1885–1962）** 于 **1909 年（明治 42 年）**在京桥（后迁银座 8 丁目並木通现址）创立，是日本"**新版画运动（新版画 / Shin-hanga Movement）**"的核心**版元（Hanmoto，出版商兼制作总监）**与至今仍在营业的同名百年老铺。19 世纪末日本传统浮世绘随明治维新后印刷技术西化而逐渐式微，渡邊庄三郎从 1900 年代起以"复兴 + 现代化"为目标，与画师 **川瀬巴水（Kawase Hasui，1883–1957）**、**伊東深水（Itō Shinsui，1898–1972）**、**吉田博（Yoshida Hiroshi，1876–1950）** 等签下专属版元契约，把传统"绘师 / 雕师 / 摺师"三分工的木版画技法与西洋远近法 / 写生主义 / 大判尺寸结合，催生了 1910–1950 年代日本近代版画的代表流派"新版画"。1923 年关东大地震烧毁了渡邊店内储存的全部川瀬巴水原版与未售版画，渡邊在地震后率领川瀬重新雕版、再版了"东京二十景"等代表作品（这批"震灾后再版"今日是新版画市场最稀缺的版本之一）。**Steve Jobs（1955–2011）** 在 1980 年代多次访日期间通过古董商介绍接触到川瀬巴水的新版画，被川瀬"日本风景的静谧感"深深打动，**生前曾多次亲自走进银座的渡邊木版美術画舗挑选版画**，成为店铺最知名的国际收藏者；据 Lisa Brennan-Jobs（Jobs 长女）后来在回忆录与访谈中提到的细节，**Jobs 去世前三个月（2011 年 7 月）丽莎去探望他时，他在加州 Palo Alto 家中与医院的病房里挂着的依然是川瀬巴水的木版画**——这段"乔布斯 + 渡邊木版 + 川瀬巴水"的典故让本店在 2010 年代后成为国际苹果迷 / 设计师 / 收藏者的"圣地巡礼"目的地。今日店铺仍由渡邊家族第三代 / 第四代经营，店内陈列川瀬巴水 / 伊東深水 / 吉田博 / 笠松紫浪 等约 100 位昭和期版画作家的现代摺新作 + 复刻版 + 老版（古版）+ 文献，是日本"绘师—雕师—摺师"三分工活态传承的代表小铺。
- **看点**：
  1. **川瀬巴水木版画陈列**：店内主厅陈列川瀬巴水"东京二十景"「東京駅」「日本橋」「眼鏡橋」「大根河岸」+「旅みやげ」系列+「日本风景集（東日本 / 西日本）」+「东京风景」+ 各地名所风景画的现代摺（仍由渡邊家在保存的原版上现代摺出）+ 古版（昭和期初版）。
  2. **「乔布斯典故」相关版画**：Steve Jobs 生前所购川瀬巴水代表作「東京駅」「夕暮の池泉（明治神宫）」「猿沢の池の月（奈良）」等品种长期有现代摺库存；店员可提供"乔布斯曾选购此款"的具体确认（按到货为准）。
  3. **新版画三分工现场**：店内展示传统"绘师 / 雕师（彫師）/ 摺师"分工的木版工具 + 雕版样本，以及现代摺师从同一原版手工摺出的多色彩印的工艺细节。
  4. **吉田博 / 伊東深水 / 笠松紫浪等同期作家**：店内不仅有川瀬巴水，还有同代新版画大师吉田博「光と影」「帆船」系列、伊東深水的美人画、笠松紫浪的城市风景等约 100 位作家约数千张作品的库存。
  5. **国际寄送服务**：因新版画为收藏级商品，店内长期提供专业的木板裱框 + 防潮包装 + 国际航空寄送（日邮 EMS / 黑猫宅配国际服务），适合海外收藏者购买后直接寄国际。
