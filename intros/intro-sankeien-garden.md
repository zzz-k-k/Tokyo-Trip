## 历史与背景

三溪园由日本明治至大正期间的横滨生丝商**原三溪（本名原富太郎，1868–1939）** 建造并于 1906 年（明治 39 年）正式向公众开放，是关东地区**最完整的"建筑迁建型"日本庭园**——也就是说，原三溪并不是在本牧本地就地建造庭园，而是亲赴京都、镰仓、岐阜、和歌山、白川乡等日本各地，以个人私财买下当时即将被拆除或失去维护的**濒危古建筑**，整体拆解、按编号运回横滨本牧的这片约 17.5 公顷的山水之间，再以传统木造工艺**原样复建**。这种"以个人之力抢救国家级建筑遗产"的实践在日本园林史上是空前绝后的孤例。园内现存 **17 栋历史建筑**，其中 **10 栋被指定为「国家重要文化财（重要文化遺産）」**、3 栋为「横滨市指定有形文化财」，包括：来自京都旧灯明寺的**三重塔（旧燈明寺三重塔，1457 年室町时代）**——也是**关东地区最古老的木造塔建筑**之一；旧紀州德川家別邸的**临春阁（臨春閣，1649 年江户初期，桃山样式）**；原三溪自宅**鹤翔阁（鶴翔閣，1902 年，原三溪起居与文人沙龙场所）**；旧灯明寺**本堂（旧燈明寺本堂，1457 年室町时代）**；旧東慶寺**佛殿**（来自镰仓东庆寺，1634 年）；**月华殿（旧伏见城内殿一部分，1603 年桃山时代）**；**天授院（来自镰仓心平寺地藏堂，1651 年）**；**春草庐（旧織田信長庵，桃山时代）**；**听秋阁（聴秋閣，1623 年由德川家光建于二条城内的茶屋，江户初期）**；以及若干石塔、石灯、墓塔与古井。原三溪本人还是日本近代最重要的**艺术赞助人**之一，与涩泽荣一同时代但走的是"实业 + 艺术"的双轨路线：他出资资助横山大观（1868–1958）、下村观山（1873–1930）、菱田春草（1874–1911）、前田青邨（1885–1977）等近代日本画家在三溪园内创作（园内的「鹤翔阁 / 三溪记念馆」即此沙龙的现场），帮助他们度过 1907–1930 年代日本画从困境走向 "近代日本画之黄金期" 的关键二十年；横山大观甚至称原三溪为「明治 / 大正期日本画的恩人」。三溪园因此既是**建筑博物馆**（17 栋古建筑现场）、也是**日本近代美术史的现场**（横山大观 / 下村观山的"原家圈"沙龙）、还是**关东地区最完整的池泉回游式日本庭园**（外苑 + 内苑双园结构 + 大池 + 小山 + 茶亭）。1953 年原家把三溪园捐赠给横滨市，由公益财团法人三溪园保胜会运营至今；2007 年被指定为日本**「国家史迹」+「国家名胜」双重指定**——这两个层级在日本文化财保护体系中是顶级别名（仅次于"特别史迹 / 特别名胜"）。

## 看点

1. **三重塔（旧灯明寺三重塔，1457 年）**：从大池对岸（外苑南端）的"莲池前石桥"是经典拍摄机位，三层塔影倒映入水，是关东最古老木塔的标志性场景；塔本体不开放登塔，但下层基座可近距离观察室町时代木造工艺。
2. **临春阁（旧纪州德川家别邸 / 1649 年桃山样式）**：内部不对外开放（内苑的内苑），但回廊外观工整，可绕一周观察"书院造 + 数寄屋造"过渡时期的木结构与遣户。
3. **鹤翔阁（原三溪自宅，1902）+ 三溪记念馆**：可入内参观，记念馆陈列原三溪的书信、横山大观与下村观山的题字与小品、以及三溪园建造过程的历史照片；旺季可预约抹茶体验（事前预约制，¥1,500–2,500）。
4. **听秋阁（聴秋閣 / 1623 年德川家光二条城茶屋，江户初期）**：移建自京都二条城内的茶屋亭阁，国家重要文化财；春秋两季有限期内部公开。
5. **月华殿（伏见城内殿一部分 / 1603 年桃山时代）**：移建自伏见城（丰臣秀吉所建、后被德川家康利用），园内"桃山遗构"代表。
6. **6 月限定景观**：6 月初的**绣球花（紫阳花 / アジサイ）尾声 + 花菖蒲（ハナショウブ）盛期**——园内的"睡莲池 + 花菖蒲田"在 6 月中旬到 6 月下旬迎来最佳观赏期，6/3 当日处于花菖蒲开始至盛期的过渡阶段（早开品种已开），可见。
7. **大池 + 内苑回游路径**：池畔的古梅、古松、苔藓与三重塔影构成一条 90 分钟回游路径，是日本"池泉回游式庭园"的关东地区现存最大规模实例。

## 推荐玩法

从正门入园后**逆时针沿大池外周行进**：先到莲池前石桥拍三重塔机位（10 分钟）→ 沿大池北岸过桥进入**内苑**——临春阁外观（10 分钟）→ 月华殿（5 分钟）→ 听秋阁外观（5 分钟）→ 天授院 + 春草庐（10 分钟）→ 出内苑回到外苑南端的旧灯明寺本堂 + 三重塔基座（15 分钟）→ 最后回到正门旁的**鹤翔阁 + 三溪记念馆**喝一份抹茶（30 分钟，¥600–800）。整段建议留 90–120 分钟（核心保留）；时间紧时压缩抹茶体验到 15 分钟，整段最低 60 分钟可走完核心三处（三重塔 + 临春阁 + 鹤翔阁记念馆）。

## 周边联动

本牧地区位于横浜港湾东南侧的工业 + 居住混合地带，**步行 10 分钟内无其他保留地点**；步行 15 分钟可达本牧海釣施設 / 本牧市民公园（不在保留清单内）。建议返程时把山下公园 / 红砖仓库段已游览的内容串成"港未来 → 山下 → 本牧"完整线，回横滨站取行李（约 50 分钟）。

## 注意事项

- 园内允许拍照、**不允许使用三脚架**（即使个人非商业用途也禁止；仅持照相机或手机的手持拍摄可以）；商业拍摄 / 婚纱拍摄需事前向三溪园保胜会申请并支付场地费。
- 6 月梅雨季园区步道湿滑（青苔 + 石板路 + 内苑石阶），建议带**防滑鞋 + 折伞 / 雨衣**；轮椅 / 婴儿车在内苑石阶处需绕行（外苑大池外周路径相对平坦）。
- 园内允许**带瓶装水 + 简餐**，但**不允许 BBQ / 明火 / 大型野餐席布**；中央广场附近有"待春軒"茶屋（轻食 + 抹茶）以及池畔的"三溪园茶屋"提供日式简餐 + 关东煮 + 抹茶（人均 ¥800–1,500）。
- 临春阁、听秋阁等内苑古建筑**多数仅外观可观，内部不开放**（每年春秋限期特别公开除外）；进入内苑前请遵守"勿碰木结构 / 勿坐栏杆"等基本礼仪。
- 园内手机信号弱（本牧海湾遮蔽）；建议提前下载好官方 PDF 园区图（官网提供日 / 英 / 中三语版）。
- 主要电子支付（Suica / PASMO / 主要信用卡）皆可在正门售票口使用；茶屋部分仅收现金 + Suica。
- 闭园时刻 17:00 严格执行，16:30 后不再售票；本次行程必须**16:00 前出园**才能赶上 16:45 横滨站取行李 → 17:00 入住 FUTARENO 的时点。

## CSV 原始备注

> 经典日式传统庭园，园内最神奇的地方在于，原三溪从京都、镰仓等日本各地，将多栋极具历史价值的古建筑整体拆解并搬运到这里重新组装。整个园内共有 17 栋历史建筑，其中有 10 栋被日本政府指定为"国家重要文化财"（重要文化遗产），比如来自京都旧灯明寺的三层宝塔，它是关东地区最古老的木塔。
