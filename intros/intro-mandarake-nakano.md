## 历史与背景

Mandarake（まんだらけ）的创业故事是日本御宅文化史最重要的章节之一——**1980 年（昭和 55 年）2 月**，**漫画家古川益藏**在**中野百老汇 4 楼**开设了一家仅 8 平米的中古漫画店，命名为「**まんだらけ**」（源自佛教用语「曼陀羅華」+ 「だらけ」"满是…"的合成词），是日本**第一家专营中古漫画 + 同人志 + ACG 周边的连锁前身**。古川益藏出身手塚治虫主宰的「虫プロ商事」，在 70 年代末发现了"**中古漫画 + 绝版同人志 + 70 年代昭和动漫周边**"被市场严重低估的商机；**1987 年**首次在中野百老汇内扩张为 3 间分店，形成**"主题分店化经营"**的雏形。**2000 年代起在中野百老汇内进一步细分**为 30+ 主题分店——包括 **「门（モン / Mon）」店**（古董玩具专门）、**「大怪兽（大怪獣 / Daikaiju）」店**（怪兽软胶专门）、**「Galaxy（ギャラクシー）」店**（怀旧游戏机原装卡带专门）。**与秋叶原 Complex 的选品分化**：秋叶原店以"通俗 ACG / 当代同人志 / 现役漫画"为主，面向 10–25 岁当代御宅；**中野店本店**则以"**昭和怀旧 / 古董玩具 / 怪兽软胶 / 怀旧游戏机原装卡带 / 绝版漫画初版本**"为主，面向 30–60 岁"**核心收藏家 / 资深御宅**"。**国际游客占比约 40%**——BBC / NHK World / 法国 Arte 频道等多次将中野百老汇与秋叶原电气街并列为"东京御宅双圣地"。今日 Mandarake 中野店是**Mandarake 系全国 30+ 分店的发源地与精神图腾**。

## 看点

1. **「门（モン / Mon）」店**：中野百老汇 4F 的古董玩具专门店—— 专卖战前 / 战后昭和 30–50 年代的**铁皮玩具 + 软胶玩具 + 早期万代 / 田宫 / Takara** 等老厂初期产品；**单件 ¥3,000–500,000+**。
2. **「大怪兽（大怪獣 / Daikaiju）」店**：中野百老汇 3F 的怪兽软胶专门店——专卖**东宝哥斯拉 / 圆谷奥特曼 / 大映卡美拉 / 东映幪面超人**等特摄系列的怪兽软胶玩偶；**单件 ¥1,500–80,000**。
3. **「Galaxy（ギャラクシー）」店**：中野百老汇 4F 的怀旧游戏机原装卡带专门店 —— 专卖**FC / SFC / N64 / GameBoy / MegaDrive / NeoGeo** 等 1980–2000 年代家用机的原装卡带 + 完盒 + 主机；**单件 ¥500–200,000+**。
4. **「LIVE（同人志 / 限定本）」店**：中野百老汇 3F 的同人志专门店 —— 专卖绝版同人志 + Comiket 限定本；**单本 ¥300–30,000**。
5. **「SF / 少女漫画」店**：中野百老汇 2F 的科幻单行本 + 战前 / 战后少女漫画绝版本专门店；**单本 ¥500–100,000+**。
6. **「Cosplay / コスチュームコーナー」店**：中野百老汇 2F 的 Cosplay 服装 / 道具中古店；**单件 ¥1,000–50,000**。
7. **中野百老汇商城本身**：**1966 年开业的 5 层综合商城**；商城内除 Mandarake 系外还有**Daily Chico**（B1F 的 8 色冰激凌店）、**TACO ché**（4F 的同人 / 独立出版书店）等。

## 推荐玩法

1. **15:10 抵达 JR 中野站北口** → 直通「中野 SUN MALL」拱廊街 → 步行 5 分钟至商城南入口 → 进入中野百老汇 1F 大厅（**先看 1F 入口处的指引图**）；
2. **乘自动扶梯上 2F** → **3F** → **4F** —— 三层逐层扫店，**每层停留 30–40 分钟**，总计 90–120 分钟；
3. **CSV 备注三家分店重点驻足**：
   - **4F「门」店** 看古董铁皮玩具墙 20–30 分钟；
   - **3F「大怪兽」店** 看哥斯拉 / 奥特曼软胶墙 20 分钟；
   - **4F「Galaxy」店** 看 FC / SFC / N64 卡带墙 + 任选 1 卷昭和原装卡带（**预算 ¥1,500–5,000 / 卷**）20–30 分钟；
4. **B1F「Daily Chico」吃 8 色冰激凌**（**¥520 / 8 色叠 25 cm 高**）10 分钟；
5. **17:30 离开中野百老汇** → 沿 SUN MALL 折返中野站北口 → JR 中央线快速 1 站返新宿。

## 注意事项

1. **现金 + IC 卡 + VISA 三选**：多数分店接受**现金 + Suica / PASMO IC 卡 + VISA / Mastercard / JCB / 银联 / Amex**；**少数小型分店仅收现金**——建议备 ¥10,000–20,000 现金。
2. **中古商品**：**绝大多数商品为中古品（二手）**，按品相 / 完盒度 / 罕见度分级定价。
3. **不支持免税（中古商品法规限制）**：**中古品被排除在 Tax Free 范畴外**；**仅店内少数全新商品支持免税**——整体免税比例极低。
4. **国际寄送**：各分店**支持国际寄送（EMS / 海外宅急便 / 船便）**——结账时告知"**国際発送 / International Shipping**"即可下单。
5. **散点店铺需走动多**：30+ 分店散布于 2F、3F、4F 三层，需频繁穿梭——**穿轻便鞋 + 不带大件行李**最舒适。
6. **拍照许可**：公共空间允许自由拍照；**店内拍照需先征得店员许可**——**禁止拍摄商品标价 + 罕见商品近景**。
7. **闭店时点 20:00**：多数分店 **20:00 关店**——本日 17:30 离场远早于闭店。
8. **不在 TST 覆盖**：**JR 中央线 新宿 ↔ 中野 段 TST ❌**（单程 ¥160 / 往返 ¥320 实付）。
9. **6 月梅雨季**：商城 + SUN MALL 拱廊街**全段室内 + 有顶**，**即使大雨也可全程不打伞**。
10. **店家专精度极高**：可与店员请教商品历史 / 来源，但**不要装内行 / 不熟而问"这值多少钱"**；直接说"観光で来ました、教えてください"最稳。
