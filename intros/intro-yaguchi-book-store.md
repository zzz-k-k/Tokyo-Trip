- **类型**：书店 / 老铺（古本 / 旧书专门店；戏剧 / 电影 / 日本文学专精）

## 历史与背景

矢口书店创业于 **1918 年（大正 7 年）**，由初代矢口长太郎在神保町开设；恰逢神保町作为「日本人文出版黄金时代」起步的关键节点——**明治末期至大正年间**（1900–1925），神保町因周边明治大学、中央大学、日本大学、专修大学、东京大学（本乡）等高校群密集而催生出"学生城 + 古书街"的双重格局，矢口书店即在此潮流中开店。**1923 年（大正 12 年）9 月 1 日关东大震灾**中，神保町几乎全街烧毁、矢口书店初代店面亦付之一炬；翌年震灾重建潮中，矢口书店在原址重新开张，成为神保町战前老铺中少数延续创业地址的字号。**第二次世界大战末期 1945 年东京大空袭**神保町核心区奇迹般避开焚毁（一说与街区内大量"古书纸箱"被列为"防火资材"优先保护有关），战后矢口书店得以原貌延续；**昭和 20–40 年代** 战后古本街复兴期，二代目矢口家进一步把店铺主营聚焦到「**戏剧（演剧）/ 电影（映画）/ 日本古典文学**」三大专门领域，与隔壁后来成立的「**东京古书会馆**」（1967 年开馆，全日本古书业的总部 + 古书拍卖会场）并列成为神保町最具代表性的两块"百年品牌招牌"。今天矢口书店由**三代目矢口家继承**经营，店头那一连串外推到人行道上、长约 8 米的露天木造书架（"店頭ワゴン棚"）已成为**神保町最易识别的视觉地标之一**——许多 NHK / TBS / 旅行频道介绍神保町古书街的 B-roll 镜头几乎都包含矢口书店的木架特写。店内深进约 12 米的窄长空间里，**戏剧 / 电影专门区** 收藏了大量绝版的日本舞台剧本、能 / 狂言 / 歌舞伎台本、战前 / 战后映画杂志 (例如「キネマ旬報」全卷)、电影海报与剧场节目册；**日本古典文学区** 收藏从万叶集、源氏物语、近世戏作（井原西鹤 / 曲亭马琴）到近代文学（漱石 / 鸥外 / 谷崎 / 三岛）的初版本、限定本与古文书。一句话：矢口书店是"神保町古书街的脸面"，CSV 备注里被特别点出的「标志性的街边书架」即指此外立面。

## 看点

1. **街边木造书架（店頭ワゴン棚）**：店面外推人行道上、长约 8 米的连绵露天木架，约 1,000–1,500 册百元级古本（多数 ¥100–500 / 本）按主题摆放，是矢口书店最易识别的外立面、也是神保町古书街的视觉标志；从对街拍木架与店招合影是经典构图。
2. **戏剧 / 电影专门区**：店内中段约 5 米长的专架，收藏战前 / 战后日本舞台剧本、能 / 狂言 / 歌舞伎台本、映画杂志全套（如「キネマ旬報」从 1919 年创刊以来的部分卷次）、剧场节目册、电影海报缩印本；价位从 ¥800（普通杂志旧号）到 ¥30,000+（绝版剧本初版）。
3. **日本古典 / 近代文学区**：店内深处的窄长书墙，收藏漱石 / 鸥外 / 谷崎 / 三岛 / 川端的初版本、限定本、签名本，以及万叶集 / 源氏物语 / 近世戏作的旧版与古文书；初版漱石《吾辈は猫である》全 3 册曾在此流转，价位 ¥10,000–200,000+。
4. **古本订单流程（注文 / 取置）**：架上没有的特定书目可向店主请「注文」（订单查询）——三代目矢口先生与神保町同业网络可在 1–7 日内调来；店头黑板会写出最近到货的"注目本"清单，是神保町古书业"店主选品 + 同业流转"的活样本。
5. **店招与玻璃窗**：店招"矢口書店"四字为传统招牌字体（**待用户实地确认是否为初代手写**），与玻璃窗内堆叠到天花板的旧书墙构成"昭和时代古本店"原貌——是日剧 / 电影里"神保町场景"的常用取景地之一。

## 推荐玩法

**10:30 开门后立即抵达**（神保町古书店 11:00 后客流上来、矢口书店店内仅约 12 米深的窄长空间易拥挤）→ 先在街对面 / 人行道边拍 1 张木造书架 + 店招的全景外观；→ 翻 30–45 分钟（先翻店头百元木架 → 再进店看戏剧 / 电影专门区 → 末尾扫一眼古典文学墙）；→ **选 1–2 本旧书作纪念**（百元木架的旧映画杂志 ¥100–500 / 本是性价比最高的"神保町手信"，比新书店买图书更有"古书街气息"）；→ 出店后向北沿白山通走 3 分钟到三省堂 / 东京堂书店做 5–10 分钟顺路打卡，再转入咖喱街午饭。**全段在矢口书店内停留 30–45 分钟即可**，不必硬把整条神保町古书街逛完（200 家书店 1 天也逛不完）。

## 注意事项

1. **现金优先**：神保町古书店多数为家族经营、**不一定接电子支付**；矢口书店现场以现金为主，**单笔 ¥10,000+ 的高价绝版书可能可刷 VISA / JCB —— 待用户实地确认**；建议提前在神保町站取 ¥5,000–10,000 现金备用。
2. **不支持免税**：古本（中古书）的特性使其**不在外国游客 Tax Free 免税范畴内**（免税仅适用于"全新未使用消费品"，旧书 / 古书 / 二手商品在日本免税法规下被排除）；购入旧书需按定价含税付现，无免税柜台。
3. **国际寄送**：店家原则上不直接承接 EMS / 国际寄送服务；若购入大量或高价书需国际寄送，建议自行带到神保町邮便局（步行 5 分钟）办 EMS / 海外宅急便，或使用第三方代购服务。
4. **店内空间狭窄**：店内深进约 12 米、宽 3 米的窄长走廊（典型昭和古本店格局），**大型行李 / 雨伞建议留在店外**（但店外人行道上无寄存柜，建议本日轻装前来）；6 月梅雨季湿伞可在店头入口处的伞架放置。
5. **拍照礼仪**：**店头街边木造书架可自由拍照**（这是公共人行道空间，神保町店家普遍接受拍摄外立面）；但**店内拍照需先征得店主许可**（古书店对内部书架拍摄敏感，部分店家明令禁止，矢口书店店内拍摄请先用口型 / 手势询问店头三代目矢口先生）。**禁止把客人 / 店主面部拍入照片**。
6. **周日定休**：本日 6/6 是周六、不冲突；但若因故改期请避开周日；**Golden Week（4/29–5/5）/ 盆休（8/13–16）/ 年末年始（12/29–1/3）** 另有临时定休，**待用户实地确认** 当周营业。
7. **6 月梅雨季**：街边木造书架在中雨 / 大雨日会**收回店内**（露天书会受潮）；若 6/6 当日为大雨日则只能看店内书墙、不可见标志性的街边木架——若希望拍到木架建议看天气预报后选晴天 / 阴天前往，雨天作为遗憾接受即可。
8. **古书店的"专精"性质**：矢口书店是**戏剧 / 电影 / 古典文学专门**店，**不收漫画 / 同人志 / ACG / 实用类 / 商务书**；若想买这些请去附近的「**书泉 GRANDE**」或「**まんだらけ 神保町店**」（如有）；矢口书店是"硬核古本"的代表、不是"什么都卖"的综合书店。
