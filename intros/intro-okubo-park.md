# 大久保公园（大久保公園 / Okubo Park）

- **类型**：公园 / 夜游（歌舞伎町北侧城市公园 + 周边非主流夜街区 + 治安敏感区）
- **位置**：东京都新宿区歌舞伎町 2-43；最近车站 东京 Metro 副都心线 / 都营大江户线「东新宿」站 B1 / A1 出口步行 5 分钟、JR 山手线 / 中央线「新大久保」站步行 7 分钟、JR 新宿站东口步行 12 分钟。
- **历史与背景**：大久保公园所在土地源自江户末至明治初的新宿大久保村边缘地带，1968 年（昭和 43 年）由新宿区正式开园为城市公园。1980–90 年代街区与歌舞伎町同被警视厅列为「夜间治安重点」，公园本体逐步规范化为儿童游具 + 长椅 + 绿地，但公园外侧自 1990 年代起逐步沉淀出新宿独有的「ホスト街道」（牛郎店街道）双重夜街区。2023 年起新宿警视厅与本地町会联合推进「歌舞伎町・大久保公园周辺浄化作戦」，增设临检、巡逻员、夜间监控摄像头，街区秩序明显改善。与歌舞伎町一番街（南）+ Golden-Gai（南东）+ 新大久保韩国城（北）共同构成新宿「三层夜街」生态。
- **看点**：
  1. 公园本体（白天）：儿童游具 + 绿地 + 长椅 + TOHO Cinemas 新宿（公园东南侧）
  2. 东侧「ホスト街道」霓虹立面：沿区役所通向北约 200 米的牛郎店密集街区，21:00 后全街霓虹立面 + 西装迎宾员密集排列
  3. 公园南端 TOHO 哥斯拉头雕像 + 一番街拱门远景
- **推荐玩法**：仅作晚间夜游打卡（21:00–21:30）。从 Golden-Gai 收尾后北行步行 7 分钟 → 公园南端入口 → 沿公园外侧逆时针绕一圈（约 15 分钟）→ 返回新宿三丁目站。全程不进入任何牛郎店 / 风俗店，整段控制在 30 分钟内。
- **票务 / 营业时间**：公园本体 24 小时免费开放、无门禁。
- **交通到达**：
  - 从前一地点 Shinjuku Golden-Gai 出发：Golden-Gai 北端 → 区役所通 → 北行步行 7 分钟即至公园南端（TST ✅ 实付 ¥0）
  - 从两国 COGO RYOGOKU 出发：都营大江户线 两国 → 东新宿（约 30 分钟、¥320、TST ✅）→ A1 出口步行 5 分钟
- **周边联动**：步行 5 分钟内 歌舞伎町（南）、Golden-Gai（南东）、TOHO 新宿 + 哥斯拉雕像（南）、花园神社（东南）
- **消费档次**：纯外侧夜游 ¥0；若途中买便利店饮料 ¥200–500。
- **注意事项**：
  1. 女性单独不建议前往（22:00 后公园外侧醉客密度高）；建议至少 2 人同行 + 22:30 前撤离
  2. 绝不跟随客引き搭话；如被纠缠请走入 TOHO 大厦或便利店
  3. 不在公园北侧深处停留；仅快速通过、不停留拍照
  4. 绝不拍摄牛郎 / 客人面部正面
  5. 22:30 前撤离（末班都营大江户线新宿西口 → 两国约 24:30 前发车）
