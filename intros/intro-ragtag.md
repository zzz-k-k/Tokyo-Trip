- **类型**：购物（中古设计师品牌时装店；4 个补丁外观点之一，按地理纳入 Day 6 上午段）
- **位置**：东京都涩谷区道玄坂 / 公园通沿线 RAGTAG 涩谷店；最近车站 JR 山手线 / 东京 Metro 半蔵门线 / 副都心线 / 银座线「涩谷」站 ハチ公口 / 8 号出口步行 5–8 分钟（TST ✅ Metro 段、TST ❌ JR 段）。具体地址按 RAGTAG 涩谷店为准 —— 待用户实地确认（官网：https://www.ragtag.jp/）。
- **历史与背景**：RAGTAG 是 1990 年代起由 Tinpan-alley 株式会社运营的"设计师品牌中古"专门连锁，主打日本国内外高端设计师 / 高级时装 / 街头潮流的中古商品流通。RAGTAG 的核心定位是"以低于全新一线价 30–60% 的价格收入二手设计师单品 + 经过店内严格的真伪鉴定 + 重新整理后再上架"——经营品类涵盖 Comme des Garçons（川久保玲）+ Yohji Yamamoto（山本耀司）+ Issey Miyake（三宅一生）+ Junya Watanabe（渡边淳弥）+ Maison Margiela + Helmut Lang + Prada + Gucci + Rick Owens + Sacai 等日本国内外设计师品牌。RAGTAG 在东京 / 横滨 / 名古屋 / 大阪 / 福冈 / 仙台 / 京都等城市开设约 15 家直营店；**涩谷店**是 RAGTAG 系统中规模较大的旗舰店之一（数百平方米卖场 + 多层展示），位于涩谷道玄坂 / 公园通一带，是日本设计师品牌中古市场的代表门店。RAGTAG 与日本"中古设计师文化"的兴起密切相关——1990 年代日本经济泡沫破裂后大量高级时装由消费者转入中古市场，催生了 RAGTAG / Komehyo / Brand Off 等中古品牌；RAGTAG 在其中以"设计师品牌专精 + 严格真伪鉴定 + 重新整理"差异化定位，至今仍是日本设计师品牌中古的代名词。
- **看点**：
  1. **Comme des Garçons / Yohji Yamamoto / Issey Miyake「日本三大设计师」中古区**：单价 ¥10,000–80,000，是涩谷一线最大的"日本三大设计师"中古集合。
  2. **欧美高端设计师中古区**：Maison Margiela / Helmut Lang / Rick Owens / Prada / Gucci 等，单价 ¥15,000–150,000。
  3. **「Vintage Archive」专区**：1980–2000 年代的设计师 archive 单品，部分稀有款定价 ¥80,000–500,000。
  4. **店内严格真伪鉴定流程**：每件商品都附 RAGTAG 鉴定卡（含品牌 / 入手时期 / 状态评级 A–D），是中古市场可信度最高的连锁之一。
- **推荐玩法**：16:35 从 LINE CUBE SHIBUYA 沿公园通下行步行 5 分钟到 RAGTAG 涩谷店 → 速逛 15–20 分钟（重点：日本三大设计师 archive 区 + Vintage Archive 区）；非购物日仅外观打卡可跳过深逛。
- **票务 / 营业时间**：
  - 入场料金：免费进入；商品消费另计。
  - 营业时间：12:00–20:00；定休日 不定休（按官网为准）。
  - 6/8（周一）正常营业。票价 / 营业时段如有调整以官网为准 —— 待用户实地确认（官网：https://www.ragtag.jp/）。
- **交通到达**：
  - 从前一地点（LINE CUBE SHIBUYA）：步行 5 分钟（公园通顶端 → 公园通中段道玄坂方向），TST 不适用（步行）。
  - 从两国 COGO RYOGOKU 出发：都营大江户线 → 半蔵门线 涩谷（约 25 分钟、TST ✅）+ 步行 5–8 分钟，0 元票价（TST 已覆盖）。
- **周边联动**：步行 5 分钟可达 **Euro Live**（道玄坂中段，已结束仅外观）；步行 5 分钟可达 **Mandarake Shibuya**（涩谷 BEAM 旁地下店）；步行 5–10 分钟可达 **Tabio / TabioMEN**（涩谷 / 表参道沿线）。
- **消费档次**：日本三大设计师中古 ¥10,000–80,000；欧美高端中古 ¥15,000–150,000；Vintage Archive 稀有款 ¥80,000–500,000；本次预算约 ¥0–50,000（按是否购买 1 件单品）。
- **注意事项**：
  - 店内允许拍商品架，但**部分稀有款 / Vintage Archive 区禁拍**，需店员协助查看；不允许拍他人客户 / 收银台。
  - 试穿允许，但需店员陪同（部分高价款）。
  - 现金 / 主要电子支付（Suica / PASMO / 主要信用卡）皆可。
  - 6 月梅雨季店内全空调；如购买大件 / 高价款建议另行包装防潮。
- **购物类别专属字段**：
  - **主营商品类目**：日本国内外设计师品牌 + 高级时装 + 街头潮流的中古男 / 女装、鞋、包、配饰；含 Comme des Garçons / Yohji Yamamoto / Issey Miyake / Junya Watanabe / Maison Margiela / Helmut Lang / Prada / Gucci / Rick Owens / Sacai 等。
  - **价格区间**：¥10,000–500,000 / 件（多数 ¥15,000–80,000）。
  - **是否支持免税**：单店消费 ¥5,000+ 凭护照可申请免税（消费税约 10%）。
  - **是否支持寄国际**：RAGTAG 支持购物后由店员协助打包 + 用日邮 EMS / 黑猫宅配国际服务自寄国际，运费另计；建议大件 / 高价款用此途径。
