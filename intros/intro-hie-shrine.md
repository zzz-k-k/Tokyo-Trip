- **类型**：神社（江户三大祭主祭社：山王祭；明治维新后官币大社）
- **位置**：东京都千代田区永田町 2-10-5；最近车站 东京 Metro 千代田线「赤坂」站 2 番出口步行约 3 分钟、东京 Metro 南北线 / 银座线「溜池山王」站 7 番出口步行约 3 分钟、东京 Metro 有乐町线 / 半蔵门线 / 南北线「国会议事堂前」站 5 番出口步行约 5 分钟、东京 Metro 银座线 / 丸之内线「赤坂见附」站 11 番出口步行约 8 分钟。任一站均在 TST 覆盖范围内。
- **历史与背景**：日枝神社（Hie-Jinja）的最早记载是文明 10 年（1478 年），太田道灌在筑造江户城时从近江国坂本山王宫（即比睿山日吉大社）勧请其分灵，作为江户城的镇守神供奉于城内。德川家康于 1590 年入主江户后将日枝神社尊为江户城的「产土神 / 镇守神」，1604 年由二代将军德川秀忠将社地扩建至城内一隅；1657 年明历大火后由四代将军德川家纲将神社从江户城内迁至千代田区永田町现址，自此成为江户庶民可正式登拜的「外宫」。1868 年明治维新后被列为东京府社，1882 年升格为「官币中社」，1928 年再升格为「官币大社」，是明治政府重组神社等级时与神田明神、浅草神社并列的「江户大社」之一。1945 年东京大空袭中本殿与社务所被烧毁，现存的本殿 / 拝殿 / 神乐殿是 1958 年战后重建；2002 年完成大规模修缮后新设了西参道（稻荷参道）上的「千本鸟居」连续朱红鸟居群。每年 6 月在此举行的山王祭，与神田祭（神田明神，5 月）、深川祭（富冈八幡宫，8 月）并列「江户三大祭」；山王祭与神田祭因均涉及神舆入皇居参拝（江户时代的「天下祭」资格）地位尤为重要，二者隔年隆重交替（偶数年山王祭、奇数年神田祭为大祭），2026 年（偶数年）正逢山王祭本祭。
- **看点**：
  1. **千本鸟居（稻荷参道）**：从社殿西南侧山门沿石阶向上的连续朱红鸟居群，是日枝神社近年最热门的拍摄机位；与本殿前正参道（南参道）的男坂石阶不同，西参道（稻荷参道）相对清静，6 月晨光透过鸟居间隙打在石阶上是出片机位。
  2. **本殿前广场（拝殿 + 神乐殿）**：山王祭开祭典礼神事的核心舞台，神官 / 巫女在拝殿与神乐殿之间举行「献饌（けんせん，献神食）」「祝词奏上」「玉串奉奠」等神道仪式；可观礼区域为本殿正前方约 20 米的玉砂利广场，无围栏隔离但请勿越过仪式行进路线。
  3. **神猿（神猴像）**：本殿向拝两侧的母猴抱小猴铜像 —— 「猿」与「魔が去る（魔退去）」「勝る（胜过）」谐音，是日枝神社独有的护佑标志；可购入「神猿御守」（¥1,000）作纪念。
  4. **山王祭神舆（神輿）**：祭典期间本殿外侧搭设的神舆台座，可近距离看到江户末期 / 明治时期传承下来的金漆神舆与凤凰装饰；神事结束后的「神幸祭」当天（具体日期以官方公告为准）会以约 500 米的「神幸行列」绕行赤坂—永田町—皇居外苑路线。
  5. **江户三大祭定位**：以「山王祭 / 神田祭 / 深川祭」并列江户三大祭闻名；与浅草神社（三社祭主祭社）、神田明神（神田祭主祭社）并列江户三大社主祭社的横向网络。
- **推荐玩法**：08:30 前从赤坂站 2 番出口出地面，沿外堀通向山门方向步行 3 分钟，从西南侧稻荷参道（千本鸟居）登山进入境内 —— 这条路线相对游客少，可在 08:45 前完成「鸟居 → 本殿」的拍摄与初步参拝。08:50 进入本殿前广场，在右手边「手水舎」做净身（双手 + 漱口）后到正面拝殿前投赛钱（¥5 / ¥50 寓意吉利）、二拝二拍手一拝。09:00 起在本殿前广场观礼山王祭开祭典礼的神道仪式（具体仪式时刻表 —— 献饌 / 祝词 / 玉串 / 神乐演奏，待用户实地确认；以官方公告为准）。仪式结束后（一般约 10:00–10:30）到社务所购入御守 / 朱印帐（朱印 ¥500），再绕到本殿后侧山顶眺望国会议事堂方向，10:30 左右下山接下一站。

- **注意事项**：

  - 着装无强制要求，但山王祭开祭典礼属正式神道仪式，建议避免短裤背心；女性可穿连衣裙；登山道（稻荷参道）有 90 级石阶，建议穿平底鞋。
