- **类型**：购物（中目黑设计师中古 / select 店；山本耀司二手淘货点 / **非强制**）
- **位置**：东京都目黑区上目黑 / 中目黑一带（中目黑站周边）；最近车站 东急东横线 / 东京 Metro 日比谷线「**中目黑**」站步行约 3–8 分钟（东横线 TST ❌、日比谷线 TST ✅）。**确切门牌号请以 Google Maps「Brooch 中目黑」实时定位为准** —— 待用户实地确认。
- **历史与背景**：中目黑（Nakameguro）是东京设计师中古 / 古着 / select 店的高密度街区，沿目黑川两岸聚集大量主打日本设计师品牌（Yohji Yamamoto / Comme des Garçons / Issey Miyake 等）的中古·vintage 小店。Brooch 即为这一带的设计师中古 / select 店之一，主营经过整理 / 鉴定的二手设计师单品，价格通常显著低于主线全新价，是「想要主线那一件、但预算有限」的淘货点。**该店为独立小店，库存为浮动的二手单品**，是否有合适的山本耀司货色取决于到店当日的库存运气。
- **看点**：
  1. **山本耀司 / 日本设计师中古**：可能淘到主线外套 / 衬衫的二手货，价格往往为全新价的 3–6 折。
  2. **店主选品视角**：select 店的库存反映店主审美，常有主流中古连锁（RAGTAG / 2nd STREET）之外的冷门好物。
  3. **目黑川街区氛围**：逛店之余可沿目黑川散步（6 月梅雨季为新绿，非樱花季但氛围清幽）。
- **推荐玩法**：作为**机动 / 非强制**点 —— 若 Day 6 代官山段尚有余裕（代官山 ↔ 中目黑东急东横线一站、约 2 分钟），可顺道下车淘 20–30 分钟；**若当日时间链偏紧（涩谷 SKY 预约 + 歌舞伎座硬锚点），则果断跳过**，不影响主行程。把它当作「捡漏彩蛋」而非「必达点」。
- **票务 / 营业时间**：
  - 入场免费；商品消费另计。
  - 营业时间 / 定休 **以店铺当前公告为准（独立小店营业时间易变动）** —— 出发前用 Google Maps / 店铺 SNS 确认 —— 待用户实地确认。
- **交通到达**：
  - 从代官山：东急东横线 代官山 → 中目黑（一站、约 2 分钟、¥150 前后、TST ❌）。
  - 从涩谷：东急东横线 涩谷 → 中目黑（一站、TST ❌）；或日比谷线（TST ✅）。
- **周边联动**：中目黑目黑川沿岸古着 / 中古店群（FIFTH GENERAL STORE 等）；代官山 T-SITE 蔦屋书店（Day 6 已纳入，东横线一站）。
- **消费档次**：二手设计师单品 ¥8,000–80,000（视品牌 / 年份 / 状态浮动）。
- **注意事项**：
  - **库存随机**：二手店不保证有山本耀司，去之前别抱「必有」预期。
  - **独立小店营业时间不稳定**：务必出发前确认开门。
  - 中古单品多为「一点物」，看中需当场决定；现金更稳妥（小店未必全支持境外卡）。
