# 雷门（雷門 / Kaminarimon Gate，浅草寺总门）

- **类型**：神社 / 寺院 / 商业街（浅草寺总门 + 仲见世通起点 + 大灯笼地标）
- **位置**：东京都台东区浅草 2-3-1（〒111-0032 東京都台東区浅草 2-3-1）；最近车站 东京 Metro 银座线「浅草」站 1 号出口步行约 1 分钟、都营浅草线「浅草」站 A4 出口步行约 3 分钟、东武晴空塔线「浅草」站正面口步行约 5 分钟、つくばエクスプレス（筑波快线）「浅草」站 A1 出口步行约 5 分钟。
- **历史与背景**：雷门（雷門 / Kaminarimon），正式名「**风雷神门**（風雷神門）」，是浅草寺（浅草寺 / Sensō-ji，东京都内最古老的寺院，相传 628 年由渔夫桧前浜成 / 桧前竹成兄弟从隅田川中网得圣观音像后开基）的总门，**最初由武将「平公雅」于天庆 5 年（942 年）作为奉纳建立**，门下供奉风神（風神）与雷神（雷神）二尊，以祈求镇护江户东郊与五谷丰登。建立以来千年间数度毁于火灾——尤以**庆应元年（1865 年）田原町大火**最为彻底，整座门连同两尊神像被焚为灰烬，此后将近 95 年间雷门处于"无门"状态，仅以石碑标记原址。**昭和 35 年（1960 年）**，松下电器（Panasonic）创始人 **松下幸之助**（松下幸之助 / Konosuke Matsushita）因祈愿病愈、发愿向浅草寺奉纳重建：现今所见的钢筋混凝土大门、正中央悬挂的高 3.9 米 / 宽 3.3 米 / 重约 700 公斤的**朱红巨大灯笼**（提灯，大字「雷門」由当时浅草寺贯首・清水谷恭顺亲题、底部刻「松下電器」铭文），以及左右龛中重塑的**风神 / 雷神**（風神・雷神）两尊像，皆是这次松下捐建的产物；之后大灯笼每隔约 10 年由京都「高桥提灯」工房整修、最近一次大整修在 2020 年。雷门作为**浅草寺仲见世通**（仲見世通り，长约 250 米、店家 89 间，江户中期由浅草寺周边居民申请清扫权而获许在参道两侧设店，是日本最古老的商店街之一）的**起点**，每日吸引约 3,000 万人次 / 年的访客，是东京下町最具代表性的地标之一，也是江户三十三观音灵场的第一札所。
- **看点**：
  1. **大灯笼正面「雷門」二字**：朱红巨灯笼是浅草最具识别度的拍照机位 —— 站在灯笼正下方仰拍可入框灯笼底部「松下電器」铭文与编织绳结。
  2. **大灯笼背面「風雷神門」**：从雷门内侧（仲见世通方向）回头可见灯笼背面写的是正式名「風雷神門」，与正面「雷門」呼应。
  3. **左右龛风神 / 雷神像**：进门正面右龛雷神（雷神，背鼓圈）、左龛风神（風神，肩负风袋），均为 1960 年重塑的木像 + 漆金身，龛上装铁栅栏防尘。
  4. **仲见世通起点**：穿过雷门即进入仲见世通 250 米参道，沿途老铺有「人形烧 木村屋」「雷おこし 常盘堂」「雀踊」等百年老店；本日动线只走至宝藏门（仲见世末端）即可，不深入浅草寺本堂以省时间。
  5. **宝藏门与五重塔（远眺）**：仲见世通尽头是宝藏门，门后右手是浅草寺本堂、左手是 1973 年重建的五重塔；从雷门望过去可同框「灯笼 + 仲见世 + 宝藏门 + 五重塔」三层进深，是浅草寺最经典的纵深机位。
- **推荐玩法**：从浅草站 1 号 / A4 口出来直接走到雷门正面，先在 **马路对面**（雷门通南侧人行道）拍一张大灯笼正面广角（避开门前自拍人群），再穿越雷门到达正背面拍「風雷神門」回看机位，然后沿仲见世通慢走 5 分钟（不在沿途店家停留太久），到宝藏门前回身拍一张"灯笼 + 仲见世 + 宝藏门"纵深，整段控制在 30–40 分钟内即返回浅草站。
- **票务 / 营业时间**：
  - **门本身 24 小时开放**（户外结构，无门禁）；浅草寺本堂内殿开放 06:00–17:00（10–3 月 06:30 起）；五重塔不开放参观。
  - 仲见世通各店家营业时间约 09:30–18:30（多数店家 19:00 后关），周三是部分老铺的传统定休日。
  - **免费**：雷门、仲见世通、浅草寺本堂参拜均免费；本堂内殿祈祷 / 御守 / 御朱印另计。
  - **最佳访问时段**：**清晨 07:00–09:30**（开门前 / 商店街尚未开业、人流最少、灯笼光影最佳）或 **18:30 后**（仲见世闭店后大灯笼夜间灯光打亮）。本日动线选 10:30–11:00 时段属次优 —— 比正午 12:00 后的人流峰值好得多但仍有团客。
- **交通到达**：
  - **从前一地点（COGO RYOGOKU 退房后）出发**：步行 5 分钟至两国站 → 都营浅草线（浅草方向）→ 浅草站 1 站 / 约 4 分钟 / ¥180（TST ❌） → 1 号 / A4 出口步行 1–3 分钟。合计约 15 分钟。
  - **从两国 COGO RYOGOKU 出发（同上）**：路径与上同；这是 Day 7 全行程中"住宿 → 浅草核心"的标准段，都营浅草线一线串起 两国—蔵前—浅草桥—浅草 4 站，¥180。
- **周边联动**：步行 10 分钟内可达 仲见世通（沿途名店）、浅草寺本堂、五重塔、宝藏门；步行 5 分钟可达浅草神社（三社祭主场，6 月非祭典期）；步行 15 分钟跨吾妻桥可远眺押上晴空塔（与下午晴空塔段顺路）。
- **消费档次**：**雷门 + 仲见世通 + 浅草寺参拜全程免费**；仲见世名物（人形烧 ¥600–800、雷おこし ¥500–1,500、抹茶冰激凌 ¥400）按当场需求购买，按本日动线建议本段不在仲见世大量消费。
- **注意事项**：
  - **大灯笼正下方人流密集**：自拍杆、正面摆拍队伍排满，建议站在马路对面或靠侧面（左 / 右龛口）拍才能避开人群。
  - **不要在仲见世通边走边吃**：日本商店街礼仪要求边走边吃属"行儀が悪い"，请站在指定的"立食区"或店家门口空场吃完再走。
  - **拍风神 / 雷神像时不要使用闪光灯**：龛内是漆金木像，闪光长期照射会损害涂层。
  - **6 月梅雨季**：雷门是户外结构、无遮蔽；遇雨建议走仲见世通（部分有屋檐）+ 浅草地下街（雷门站 1 号出口连通）。
  - **节假日 / 周末客流**：6/9 为周二非假日属本周最低峰之一，但 11:00 后旅行团仍密集，建议尽量早场。
  - **本日行李提示**：浅草站 1 号 / A4 出口附近投币柜（行李方案 B）周末 10:30 前易满，大型柜尤其稀缺；若行李 > 28 寸建议改方案 A（COGO 寄存）。
- **神社 / 寺院 / 商业街类别专属字段**：
  - **开闭门时间**：雷门本身 24 小时开放、无门禁；浅草寺本堂内殿 06:00–17:00（夏期）/ 06:30–17:00（冬期）；仲见世通各店家约 09:30–18:30 个体营业。
  - **是否收费**：**全程免费**（雷门 + 仲见世通 + 浅草寺本堂参拜）；御守 ¥500–1,500 / 御朱印 ¥500 另计。
  - **最佳季节 / 时段**：**全年皆可**，最佳为**清晨 07:00–09:30**（无人流、灯笼光影、店家未开业）或**夜间 18:30 后**（夜间灯光打亮）；6 月梅雨季有阴湿但游客密度反而比 4 月樱花季 / 11 月红叶季低。
  - **是否允许带饮食 / 摄影**：仲见世通**不允许边走边吃**（站在店门口或立食区吃完再走）；雷门 / 浅草寺**全区允许摄影**但本堂内殿祈祷区禁止摄影、风神 / 雷神龛禁止闪光灯；专业三脚架在仲见世通需事先申请（一般不批）。
