- **类型**：商业街 / 历史街区 / 唐人街（日本最大 / 与神户南京町 / 长崎新地中华街并称「日本三大中华街」）
- **位置**：神奈川县横浜市中区山下町（南北约 500 米 / 东西约 350 米的近正方形街区，由 10 座大小牌楼围合）；最近车站 港未来线「**元町・中華街**」站 **1 号 / 2 号出口**出站即至**善邻门**（南门 / 全街正门），或 JR 京滨东北线 / 根岸线「**石川町**」站北口步行约 5 分钟到**西阳门**（西门），或 港未来线「**日本大通り**」站步行约 5 分钟到**玄武门**（北门）；街区中心 关帝庙 距任一车站均步行 5–7 分钟。

## 历史与背景

横滨中华街的形成可上溯至 **1859 年（安政 6 年）横滨开港**——该年江户幕府按 1858 年日美修好通商条约（与英 / 法 / 俄 / 荷同时签订的安政五国条约）开放横滨港作为对外通商口岸，紧邻外国人居留地（现山手 / 元町地区）的西侧划出一片低洼湿地作为"附属华人居住区"，最初由从广东 / 福建 / 上海跟随欧美洋行来日的第一代华侨（多数为买办 / 翻译 / 厨师 / 裁缝）定居形成聚落，是日本最早的成型唐人街。**1873 年（明治 6 年）**横滨华侨在山下町中心建立**第一代横滨关帝庙**（祀关圣帝君，作为华侨社区精神中心），同年起街区被官方称作"南京町"或"支那町"。**明治后期至大正期（1890–1923）**随横滨港作为日本第一大贸易港的鼎盛，中华街规模迅速扩张到现今约 500 米 × 350 米范围，街区内中医诊所、广东餐馆、福建包子铺、中式裁缝、华人学校（1898 年建立的横浜大同学校）等一应俱全，1893 年第二代关帝庙落成。**1923 年（大正 12 年）9 月 1 日关东大震灾**直接摧毁了几乎整个中华街（含第二代关帝庙），约半数华侨在震灾中罹难或返国，街区一度沦为废墟；震灾后第三代关帝庙于 1925 年重建，但战前的辉煌再未恢复。**二战期间（1937–1945）**因中日交战，街区华侨遭受严密监控，营业大幅萎缩，第三代关帝庙在 1945 年 5 月 29 日横滨大空袭中再次被烧毁。**战后 1950 年代**随华侨人口回流与横滨港复兴，中华街进入第二次黄金期——1955 年首座大型牌楼**「善邻门」（牌楼正门 / 题字"亲仁善邻"，取自《左传》「亲仁善邻，国之宝也」）**于街区南端立起，标志着中华街从"南京町"正式更名为"横浜中華街"并向日本主流社会公开身份。**昭和后期至平成（1970s–2000s）**陆续修建朝阳门（2003 / 东门 / 朝向旭日）、延平门（西门）、玄武门（北门）、市场通门（南门小入口）等大小 10 座牌楼，**1990 年第四代关帝庙重建**（即现存庙宇，从台湾迎请关圣帝君神像、由台湾彩绘师团队完成屋顶剪粘工艺）。**2006 年（平成 18 年）**于关帝庙北侧约 200 米处落成**横滨妈祖庙（横浜媽祖廟）**——这是日本本土第一座正式的妈祖庙，主祀海上守护神妈祖（天后），与关帝庙形成"武圣关公 + 海神妈祖"的双中心格局；建造资金由横浜华侨总会与全日本华侨社团募集，落成时邀请台湾北港朝天宫分灵。**2004 年港未来线开通**后"元町・中華街"站直达街区，客流量再上一台阶；今日横滨中华街以 **250 余间店铺、年来客约 2,000 万人次**稳居**日本最大唐人街**地位，与**神户南京町 / 长崎新地中华街**并称「日本三大中华街」（横滨规模最大，约为神户南京町的 3 倍、长崎新地的 4 倍）。

横滨中华街源于 1859 年横滨开港时第一代华侨随欧美洋行来日定居形成，1873 年第一代关帝庙建立标志精神中心成型；明治后期至大正期为第一次黄金期，规模扩展到约 500m × 350m 的近正方形街区。1923 年关东大震灾几乎摧毁全街、约半数华侨罹难或返国，第二代关帝庙烧毁；1925 年第三代关帝庙重建，1945 年 5 月 29 日横滨大空袭再次烧毁。战后 1950 年代华侨人口回流 + 横滨港复兴，街区进入第二次黄金期，1955 年首座大型牌楼「善邻门」立起，街区正式从"南京町"更名为"横滨中华街"。1990 年第四代关帝庙重建（现存）、2003 年朝阳门立、2006 年妈祖庙落成、按四神兽方位配置 10 座大小牌楼陆续完成；2004 年港未来线开通后客流再上一台阶。今日 250 余间店铺、年来客约 2,000 万人次，是**日本最大唐人街**，与神户南京町 / 长崎新地中华街并称「日本三大中华街」。

## 看点

1. **善邻门（善鄰門 / 牌楼正门，1955 立）**：街区南端入口，1955 年立起的第一座大型牌楼，正面题字「中華街」、背面题字「親仁善鄰」（取自《左传》「亲仁善邻，国之宝也」），是中华街战后再生的标志性符号；港未来线元町・中華街站 1/2 号出口出站即至，最常见的打卡机位。
2. **朝阳门（朝陽門 / 东门，2003 立）**：街区东端朝大海方向的最东大门，朝向旭日初升方向（命名"朝阳"取义"东方旭日"），蓝色琉璃瓦 + 红柱 + 金漆雕饰（青龙象征东方），与西门**延平门**（白虎 / 西）、北门玄武门（玄武 / 北）、南门朱雀门（朱雀 / 南）共同构成中华街按四神兽方位配置的"四方四门"格局——这种把唐人街整体按中国传统四神兽方位规划的做法在世界各国唐人街中绝无仅有，是横滨中华街的独特识别符号。
3. **横滨关帝庙（横浜関帝廟 / 现存第四代庙宇 1990 年重建）**：街区中心，主祀关圣帝君（关羽），是日本最古老 + 规模最大的关帝庙；从 1873 年第一代起经历震灾、空袭、战后再生四次重建，**现存为 1990 年第四代庙宇**——由台湾彩绘师团队完成屋顶剪粘工艺，正殿匾额「萬世人極」「忠義仁勇」，殿内主祀关圣帝君（武圣）+ 副祀观世音 + 福德正神 + 地母娘娘。庙内"求签"（中文 / 日文双语签文，¥100 / 次）是横滨华侨与日本旅人共同的祈愿仪式；每年 6 月 24 日（关圣帝君诞辰）+ 旧历正月十五（元宵节）庙前会举行盛大祭祀与舞龙舞狮。
4. **横滨妈祖庙（横浜媽祖廟 / 2006 年建）**：关帝庙北侧步行约 3 分钟，2006 年建成的日本本土第一座正式妈祖庙，主祀**妈祖（天后 / 海上守护神）**，从台湾北港朝天宫分灵；庙宇风格为闽南剪粘 + 燕尾屋脊，与关帝庙的"武圣"形成"海神"对照，是横滨作为港湾城市的精神象征之一。
5. **横滨大世界（横浜大世界 / Yokohama Daisekai，朝阳门内 8 层综合 entertainment 馆）**：位于朝阳门内侧、街区东南角的 8 层综合性场馆，是中华街最高建筑 + 标志性现代地标——1F 入口 + 大型壁画 + 中华食品店、2–3F 中华美食大厅、4F 趣味艺术博物馆（Trick Art Museum，错视画 + 互动拍照，成人 ¥1,400）、5F 求婚塔 + 望景咖啡、6–7F 杂技 / 演艺剧场（大世界中国舞 / 川剧变脸表演）、8F 楼顶展望平台（部分时段开放 / 免费）。是与中华街"庙堂 + 牌楼"传统侧形成对照的"现代华人娱乐综合体"。
6. **街区主街 中华街大通（中華街大通り）**：从善邻门（南）到玄武门（北）贯通南北约 500 米的主街，沿街 250 余间店铺以"老字号广东菜（聘珍楼 1884 / 华胜楼 / 萬珍樓 / 重慶飯店）+ 上海菜（四五六菜館 1928 / 状元樓）+ 福建包子（江戸清 / 横浜博覽館）+ 中式甜品 / 月饼 / 凤梨酥 / 珍珠奶茶 / 中国茶专卖"等组合呈现；正午 + 黄昏 + 周末为客流峰值，约每 100 米一处大型牌楼或庙宇过渡。

## 推荐玩法

从港未来线馬車道站乘 1 站到元町・中華街站（约 3 分钟、¥190）→ 1/2 号出口出站即至善邻门 → 在善邻门前拍 5 分钟外观 → 沿中华街大通向北走入街区 → **13:00–14:00 选一家老店午餐**（聘珍楼 1884 / 华胜楼 / 四五六菜館 1928 三选一，人均 **¥1,500–3,500**）→ 餐后步行 3 分钟到**关帝庙**（街区中心，¥0 入庙 / 可求签 ¥100）拍 10 分钟正殿与剪粘屋顶 → 步行 3 分钟到**妈祖庙**（关帝庙北侧）拍 5 分钟 → 折向东走至**朝阳门**拍 5 分钟蓝色琉璃瓦 → 朝阳门内即**横滨大世界**1F（免费）穿过中华食品店 → 大世界出口步行 3 分钟到山下公园北端，转入子区域 4。整段约 60 分钟（含午餐 45 分钟），人均 ¥1,500–3,500。

## 注意事项

- **庙宇礼仪**：关帝庙 + 妈祖庙正殿**不允许内部拍照 / 不允许使用闪光灯 / 不允许大声喧哗**；入庙前请脱帽 / 关闭手机声音；上香时先以右手持香、合掌默念心愿、再插入正殿前香炉中；求签后请到副殿一侧的"解签处"找解签师（中文 / 日文双语，免费 / 心意打赏 ¥100–500）。庙门为四方四神兽方位，请勿踩门槛。
- **6 月梅雨季**：中华街街区为开放街道，无拱顶覆盖；雷雨时可临时进入横滨大世界 8 层楼内（1F 入口免费）或选定餐厅躲雨；中华街大通的石板路湿滑，请带防滑鞋。
- **现金 + 电子支付**：老店（聘珍楼 / 华胜楼 / 四五六菜館 / 萬珍樓等）支持主要电子支付（Suica / PASMO / 主要信用卡 / Apple Pay / 微信 / 支付宝）；街区小店与摊位多数仅收现金，建议预留 ¥5,000 现金。
- **老店周二定休风险**：本次行程 6/3 周二，部分老店可能定休；建议出发前先在 Tabelog 或官网确认聘珍楼 / 华胜楼 / 四五六菜館 营业情况，备选店清单：萬珍樓本店（1892 创业 / 山下町 153）/ 重慶飯店本館（1959 / 山下町 164）/ 江户清本店（1894 创业肉包专卖 / 山下町 192，外带午餐 ¥500–800）。
- **拉客 / 套餐避让**：中华街主街（特别是中华街大通中段 + 朝阳门附近）有少数门口拉客的店员手持菜单招呼，多数为新派融合店或自助餐厅；如非首次到访可直接进入预先选好的老店、避免被店外拉客带偏。建议**用日文「ありがとう、決まってます」（谢谢，我已经决定了）**礼貌拒绝即可。
- **关帝诞辰 / 元宵节**：每年 6 月 24 日关圣帝君诞辰 + 旧历正月十五元宵节庙前会举行盛大祭祀与舞龙舞狮，街区客流密度极高；6/3 当日处于平日，无大型节庆。
- **拍照礼仪**：街区允许个人手持摄影（牌楼 + 街景 + 庙宇外观 + 主街），**不允许对店内顾客面部正面拍摄 / 庙宇正殿内部拍摄 / 商业拍摄需向商店街振兴会申请**；老店餐厅店内多数禁止拍照。
