- **类型**：历史建筑 / 广场（重要文化财广场，丸之内中央广场）
- **位置**：东京都千代田区丸之内 1 丁目（东京站丸之内中央口正面 / 东京站丸之内站舍前广场，行幸通起点东端）；最近车站 **JR 东日本「东京」站丸之内中央口直结**（TST ❌ JR 段）；**东京 Metro 丸之内线「东京」站直结**（TST ✅）；东京 Metro 千代田线「二重桥前」站步行 5 分钟（TST ✅）；东京 Metro 半蔵门线 / 都营三田线「大手町」站步行 7 分钟（TST ✅）。
- **历史与背景**：丸之内东京站广场（丸の内東京駅広場 / Marunouchi Tokyo Station Square / Marunouchi Central Square）是 **2007 年起 JR 东日本 + 三菱地所 + 国土交通省 + 千代田区**联合主导的"东京站丸之内站前广场再整备事业"的核心成果——东京站 1914 年（辰野金吾设计）开业以来，站前广场一直作为"皇居前正面玄关轴"（行幸通—皇居—二重桥）的东端节点存在，但 1980–2000 年代广场被巴士站 / 出租车站 / 临时停车场分割破碎。**2014 年 12 月 18 日**广场再整备工程第一期完成，**2017 年 12 月 7 日**第二期完成——把车辆动线移到广场地下化 + 地面层完全步行化 + 设置 4 个对称分布的水景 + 行幸通绿带连续延伸到广场西缘 + 广场两侧丸大楼（丸ビル，三菱地所 2002）+ 新丸大楼（新丸ビル，三菱地所 2007）+ 新东京建物（東京会館，2019）+ 邮便局舍（KITTE，2013）—— 广场作为一个复合的"皇居前正面玄关空间"完整成型。**广场本体作为东京站丸之内站舍（重文）的一体的延伸景观，亦被认定为重要文化财保护范围之一**（实际"重要文化财"指定的是站舍本体，广场作为其前景空间纳入文化保护规划框架）；CSV 用户原始备注"如图所示，是重要文化遗产"指的是**站舍 + 广场组合作为"日本皇居前正面玄关"重要文化景观**的整体。从广场西望"行幸通—皇居正门石桥—二重桥"是东京最具代表性的国家级正面玄关视角；从广场东望则是辰野金吾红砖站舍正面 + 南北圆顶组合。广场夜间打灯到约 22:00，傍晚 19:30–20:30 蓝时段过渡到夜景为最佳拍摄时段。
- **看点**：
  1. **广场中央 4 个对称水景**：2014 / 2017 年再整备的标志元素，对称分布在中央口正面南北两侧。
  2. **东望站舍正面 + 西望行幸通 / 皇居视角**：广场是"东京站红砖站舍 ↔ 行幸通 ↔ 皇居正门石桥 ↔ 二重桥"皇居前正面玄关轴的中心节点，**东望站舍 + 西望皇居**两个视角是东京最具代表性的国家级景观。
  3. **广场南北两侧建筑组合**：丸大楼（丸ビル，三菱地所 2002）+ 新丸大楼（新丸ビル，三菱地所 2007）+ 邮便局舍 KITTE（三菱地所 + 隈研吾改修，2013）+ 新東京会館（2019）四栋，是"丸之内 21 世纪再开发"的代表建筑组合，可一并外观打卡。
  4. **行幸通绿带**：从广场西缘连续向皇居方向延伸约 200 米，是皇室行幸时使用的中央大道（如新天皇即位 / 国葬等仪式），平时为公开步行绿带。
  5. **广场夜景打灯**：傍晚到 22:00 站舍 + 丸大楼 / 新丸大楼 + 行幸通同时打灯，是丸之内夜景的核心节点。
- **推荐玩法**：20:00 从 Tokyo Station 站舍前向广场中央走 → 站到广场中央 4 个水景之间 → 同时东望站舍正面（红砖夜景）+ 西望行幸通 / 皇居方向（远景皇居正门）→ 步行到广场西缘 + 行幸通起点拍 5 分钟 → 回到广场南北两侧外观打卡丸大楼 + 新丸大楼 + KITTE → 20:20 出发去日本桥子区域。
- **票务 / 营业时间**：
  - 入场料金：**免费开放**；广场属公开步行空间，24 小时可进入。
  - 营业时间：广场 24 小时开放；夜间打灯通常到 22:00；周边丸大楼 / 新丸大楼 / KITTE 商业空间 11:00–21:00（KITTE 部分餐饮 23:00），周边各商业楼按其营业时间。
  - 6/8（周一）正常开放。营业时间如有调整以官网为准 —— 待用户实地确认（官网：https://www.tokyostationcity.com/）。
- **交通到达**：
  - 从前一地点（Tokyo Station 站舍）：站舍即在广场东端，**步行 0–1 分钟**（紧邻同访），TST 不适用（步行）。
  - 从两国 COGO RYOGOKU 出发：都营大江户线 两国 → 大门（约 11 分钟、TST ✅）→ 都营三田线 大门 → 大手町（约 8 分钟、TST ✅）+ 步行 5 分钟到广场北缘；合计 ≈ 25 分钟、0 元票价（TST 已覆盖）。
- **周边联动**：紧邻 **Tokyo Station 丸之内站舍**（本子区域第一项）；步行 1 分钟可达丸大楼（丸ビル，三菱地所 2002）+ 新丸大楼（新丸ビル，三菱地所 2007）+ KITTE（隈研吾改修 2013）；步行 5 分钟可达 **Nihonbashi Takashimaya Shopping Center**（日本桥高岛屋，子区域 6 第一项）；步行 8 分钟可达皇居前广场 / 二重桥。
- **消费档次**：广场免费；本日仅外观打卡 + 拍照，预算 ¥0。
- **注意事项**：
  - 广场允许拍照 + 步行，**禁用三脚架 / 自拍杆**（与站舍前广场同规则，避免堵塞通行）。
  - 6 月梅雨季广场无遮挡，请带折伞；夜景拍摄推荐 19:30–20:30 蓝时段过渡到夜景的"魔法时间"。
  - 广场夜间打灯到 22:00；22:00 后仅基础照明，夜景效果减弱。
  - 广场属公开步行空间，**禁止滑板 / 滑行车 / 自行车骑行**；散步 / 拍照 / 静坐 OK。
  - **重要文化财保护**：站舍 + 广场作为重文景观，请勿触碰建筑外墙 / 在广场内野餐 / 留下垃圾；尊重文物保护规则。
- **建筑类别专属字段**：
  - **建筑师 / 设计主持**：**JR 东日本 + 三菱地所 + 国土交通省 + 千代田区**联合主导（"东京站丸之内站前广场再整备事业"，2007–2017）；具体设计由 JR 东日本建筑设计事务所 / 三菱地所设计 + 多家顾问团队组成；周边建筑组合含三菱地所设计的丸大楼（2002）/ 新丸大楼（2007）+ 隈研吾改修的 KITTE（2013）。
  - **建成年份**：**第一期 2014 年 12 月 18 日完成**（站舍前主广场 + 4 个水景）；**第二期 2017 年 12 月 7 日完成**（行幸通绿带连续 + 周边建筑统一）。
  - **设计要点**：车辆动线移至地下化 + 地面层完全步行化 + 4 个对称水景 + 行幸通绿带连续延伸 + 与周边丸大楼 / 新丸大楼 / KITTE / 新東京会館的建筑组合统一；作为东京站丸之内站舍（重文）的延伸景观，是"皇居前正面玄关轴"的核心节点。
  - **可否进入参观**：**免费**进入；24 小时开放；广场属公开步行空间。
