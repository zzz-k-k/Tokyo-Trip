- **类型**：购物（袜子专门店，中性 / 女士主线；4 个补丁外观点之一，按地理纳入 Day 6 上午段）
- **位置**：东京都涩谷区 / 港区表参道沿线 Tabio 主线分店；最近车站 JR 山手线 / 东京 Metro 半蔵门线 / 副都心线 / 银座线「涩谷」站 + 千代田线 / 银座线「表参道」站均可达（TST ✅ Metro 段、TST ❌ JR 段）。地址按当日实际访问店铺为准 —— 待用户实地确认（官网：https://tabio.com/jp/）。
- **历史与背景**：Tabio（タビオ）的整体品牌史与 Tabio MEN 同源（详见 TabioMEN 条），1968 年由越智直正在大阪创立，是日本"袜子专门店"模式的开创者之一。**Tabio 主线**主打中性 + 女士市场，是 Tabio 集团旗下规模最大的产品线，约占总销售额的 60–70%；产品涵盖商务袜（中性灰 / 黑 / 深蓝）+ 时尚袜（图案 / 配色 / 长度多样）+ 浪漫风格 / 复古 / 玫瑰 / 繁花 / 几何花纹（女士专享）+ 编织羊毛 / 棉麻混纺 / 莱卡四色 / 海岛棉等多种材质，价格区间 ¥800–4,000 / 双。Tabio 主线在涩谷 / 表参道 / 新宿 / 银座 / 横滨等核心商圈布点，与 Uniqlo / GU 等快消袜 + ABC-Mart / Footwear Specialists 等鞋袜复合店形成"日本袜子市场三层"中的高品质 / 中价位段。Tabio 的产品质量与日本袜子行业的"奈良县大和高田 / 北葛城"集群（日本袜子产业核心，占全国袜子产量 30%+）紧密相关——大量 Tabio 主线产品在奈良工厂高密度针织 + 双重缝合工艺生产，是"日本制造"袜子的代表之一。
- **看点**：
  1. **女士时尚袜系列**：图案 / 配色 / 长度多样的女士袜，每月新款轮换，是涩谷 / 表参道一线女性消费的小礼物常选。
  2. **奈良产「日本制造」基础款**：印有"Made in Japan"标识的高密度针织 + 双重缝合工艺基础款，¥800–1,500 / 双。
  3. **季节限定款**：6 月梅雨季 + 初夏限定（凉感纤维 / 透气网眼 / 防水 / 防滑）等专项功能袜。
  4. **Tabio 集团旗下「靴下屋（Kutsushita-ya）」联名款**：偶发的「靴下屋」联名（与全国百货合作的儿童 / 礼盒款）。
- **推荐玩法**：16:45 从 Tabio MEN 步行 1 分钟到 Tabio 主线 → 速逛 10 分钟（重点：女士时尚袜 / 季节限定款）；与 TabioMEN 一并外观打卡可跳过深逛。
- **票务 / 营业时间**：
  - 入场料金：免费进入；商品消费另计。
  - 营业时间：11:00–20:00（部分店延长至 21:00）；定休日 不定休。
  - 6/8（周一）正常营业。票价 / 营业时段如有调整以官网为准 —— 待用户实地确认（官网：https://tabio.com/jp/）。
- **交通到达**：
  - 从前一地点（Tabio MEN）：步行 1–5 分钟（同店或临近店），TST 不适用（步行）。
  - 从两国 COGO RYOGOKU 出发：都营大江户线 → 半蔵门线 涩谷 / 表参道（约 25 分钟、TST ✅）+ 步行 5–10 分钟，0 元票价（TST 已覆盖）。
- **周边联动**：步行 1 分钟可达 **Tabio MEN**（男士线）；步行 5 分钟内可达 **RAGTAG**（涩谷店设计师品牌中古）；步行 8 分钟可达 **Euro Live**（道玄坂中段，已结束仅外观）。
- **消费档次**：基础款 ¥800–1,500 / 双；时尚袜 / 季节限定 ¥1,200–2,500 / 双；高端 / 联名 ¥2,500–4,000 / 双；本次预算约 ¥0–3,000（按机动选购 1–2 双）。
- **注意事项**：
  - 店内允许拍商品架 / 单品，但不允许拍他人客户 / 收银台。
  - 现金 / 主要电子支付（Suica / PASMO / 主要信用卡）皆可。
  - 6 月梅雨季店内全空调，无湿度问题。
- **购物类别专属字段**：
  - **主营商品类目**：女士时尚袜 + 中性 / 商务基础款 + 季节限定（凉感 / 防水 / 防滑）+ 高端 / 联名款。
  - **价格区间**：¥800–4,000 / 双（基础 ¥800–1,500、时尚 ¥1,200–2,500、限定 / 联名 ¥2,500–4,000）。
  - **是否支持免税**：单店消费 ¥5,000+ 凭护照可申请免税（消费税约 10%）。
  - **是否支持寄国际**：店内不直接寄国际；可通过 Tabio 官网海外发送，建议大件用此途径。
