- **类型**：商业街 / 历史巷弄（江户文人居住地 + 战后法日交流密集区 + "东京小巴黎"）
- **位置**：东京都新宿区神乐坂 1–6 丁目（神乐坂坂道沿线 + 东西延伸的巷弄群：兵库横丁 / 芸者新道 / 轻子坂 / 见番横丁 / 三身寺横丁）；最近车站 东京 Metro **东西线「神乐坂」站 C09 神楽坂口出口**步行约 1 分钟（出口即神乐坂坂道北端）+ 东京 Metro 有乐町线 / 南北线 / 东西线 / 都营大江户线「**饭田桥**」站 B3 出口步行约 5 分钟（坂道南端 / 牛込桥 + 神楽坂下交差点方向）+ JR 总武线 / 中央线「**饭田桥**」站东口步行约 6 分钟。任一站均在 TST 覆盖范围内（JR 段除外）。
- **历史与背景**：神乐坂的地名相传源自江户初期此地的「神楽坂下」即筑土八幡宫（つくどはちまんぐう）+ 善国寺（後述）举行神乐演奏的小坂道，坂道因每逢祭礼时神乐声从山下传至山上，"神乐"声漫坂而上，故名「神楽坂」。江户时代神乐坂一带是**牛込御门（外郭见附之一）**的内侧，沿坂道两侧分布着大量旗本 / 御家人的武家屋敷与少量町家；坂上的若宫八幡宫与坂下的善国寺（毘沙门天）是神乐坂街区的精神支柱，至今仍由本地居民每年定期祭祀。明治维新后武家屋敷被拆除，神乐坂在 1880–1900 年代重组为商业街，因毗邻早稻田大学（1882 年由大隈重信创立）+ 学习院 + 法政大学等近代教育机构，吸引了大量学者 / 文人 / 出版人入住，**1910–1930 年代被誉为"东京小巴黎"**：当时神乐坂是东京最早一片有"洋灯街灯 + 法语补习班 + 法日交流沙龙 + 法国料理店 + 高级料亭"的街区，与同期作为政界 / 商界料亭街的赤坂、作为艺者花柳街的浅草六区共同构成"江户三大花柳街"，但神乐坂以**文学 + 学问气息更浓**区别于另两者。**三岛由纪夫**（青年时代）+ **谷崎润一郎**（早稻田 / 牛込居住期）+ **泉镜花**（一生定居神乐坂）+ **夏目漱石**（早稻田南町 / 现夏目坂）+ **尾崎红叶**（横寺町 / 神乐坂坂上）等大量明治–大正–昭和文豪曾居住或频繁往来神乐坂；该街区因此积淀了东京少有的"巷弄文学"基因。1923 年关东大震灾时神乐坂未受重大破坏，1945 年东京大空袭中坂下段被烧毁但**坂上的兵库横丁（兵庫横丁）+ 芸者新道（芸者新道）+ 见番横丁等核心巷弄群幸免于难**，是东京中心区少有的保留了明治末期 - 大正期町割（街区划分）+ 黑塀（黑色木围墙）+ 石板路 + 小料亭格局的"江户–大正"街区现场样本。1950–60 年代神乐坂作为战后小料亭 / 高级割烹密集区延续了花柳街余韵，1970 年代起随东京中心商务化转型，神乐坂的料亭逐步减少但**法语补习班 / 日仏学院（Institut français du Japon）+ 法日交流沙龙 + 法国料理店 / Bistro / Patisserie**密度反而上升 —— 现新宿区神乐坂 4 丁目设有「Institut français du Japon - Tokyo」（日仏学院东京），是日本国内最古老的法语教学 + 法国文化交流机构（1952 年开设），周边密集分布的法日双语书店、法国 Bistro、法日交流酒吧使神乐坂成为东京"**最具法语气息的非外国人街区**"。CSV 备注「江户时代的文人聚居地 + 现在巷弄保留极高私密感 + 适合探险」即指本街区的"巷弄文学"基因 + 兵库横丁 / 芸者新道的私密石板巷弄探险价值，是 Day 5 隈研吾主轴傍晚段的"江户–大正–战后法日交流"叙事支点。
- **看点**：
  1. **神乐坂坂道（坂上 → 坂下，全长约 600 m）**：从东京 Metro 神乐坂站「神楽坂口」出地面即坂上端，沿坂道向南漫游至坂下「神楽坂下交差点」+ 牛込桥（外堀通对岸即饭田桥站）；坂道两侧密集分布和菓子老铺（梅花亭 / 五十铃 / 紀の善）+ 法国 Bistro + Patisserie + 法日双语书店 + 高级割烹小料亭；6 月梅雨季傍晚坂道亮起暖黄色街灯时是东京少有的"小巴黎"夜景。
  2. **兵库横丁（兵庫横丁 / Hyōgo Yokochō）**：从神乐坂坂中段（约神楽坂 4 丁目）东向分支的细长石板巷弄，全长约 80 m，两侧黑塀（黑色木围墙）+ 石板路 + 小型割烹 / 料亭 / 隐藏式日式酒吧；是东京中心区**保留最完整的明治末期–大正期町割现场**，被称为"东京最具江户末期巷弄风情的小街"，6 月梅雨季黑塀映青苔 + 石板反光的画面感是出片机位。
  3. **毘沙门天善国寺（毘沙門天 善国寺 / Zenkoku-ji）**：神乐坂坂中段东侧的小型佛寺（日莲宗），创建于 1595 年（文禄 4 年）安土桃山时代，原本位于日本桥馬喰町，1793 年迁至神乐坂现址。本尊是镰仓时代雕刻的「**毘沙门天像**」（神乐坂的别称"毘沙门天的街"即源于此寺），是江户「七福神巡」之一的毘沙门天担当寺，每年 1 月 / 5 月 / 9 月例祭日（具体日期以官网为准）举行寅之日大祭。**6/7 当日非例祭日**，但寺院境内 24 小时开放（拝殿至 17:00），可在傍晚段做简短参拝（投赛钱 ¥5–50 + 二拝二拍手一拝）+ 拍摄朱红色山门 + 寺前石灯笼。
  4. **芸者新道（芸者新道 / Geisha Shindō）+ 见番横丁（見番横丁）**：兵库横丁向北分支的另一条细巷弄，明治期是神乐坂芸者町的核心通道（"见番"即芸者管理事务所），现仍保留少量黑塀 + 石板路 + 隐藏式割烹，是神乐坂"巷弄文学"基因的另一处现场。
  5. **梅花亭（梅花亭 神楽坂本店）+ 五十铃（五十鈴）+ 紀の善（紀の善）等和菓子老铺**：神乐坂坂道两侧分布的明治 / 大正期创业的和菓子老铺，与赤坂青野（明治 32 年创业）同属"东京老铺和菓子谱系"；其中**紀の善**的"抹茶 babaroa（バババロア）"是神乐坂限定甜点；可在巷弄夜游时购入 1–2 件作宵夜 / 第二天早茶。
  6. **法语气息看点：日仏学院（Institut français du Japon - Tokyo）+ Bistro / Patisserie 群**：神乐坂 4 丁目的日仏学院是日本最古老的法语教学机构（1952 年开设），周边密集分布的法国 Bistro（如「La Bretagne」可丽饼店、「Brasserie Paul Bocuse Le Musée」等）+ Patisserie + 法日双语书店共同构成神乐坂"东京最法国气息的非外国人街区"；本次行程仅作过道阶段感受，不深入用餐（晚餐留给神乐坂坂道老铺餐饮）。
- **推荐玩法**：从根津美术馆 / 表参道乘东京 Metro 东西线（饭田桥换乘）抵达东京 Metro 神乐坂站后，从「神楽坂口」（C09 出口）出地面即坂上端 —— 沿神乐坂坂道向南漫游约 30 分钟，途中**重点拐入兵库横丁**：从坂中段（约神楽坂 4 丁目「龙公亭」标识附近）东向分支进入兵库横丁石板巷弄，缓慢走完 80 m 全长，在巷弄中段 / 出口处拍 1 张黑塀 + 石板路机位（傍晚 18:00–18:30 街灯亮起后画面感最佳）→ 出兵库横丁后回到神乐坂主坂道，继续向南步行 5 分钟到**毘沙门天善国寺**（坂中段东侧），简短参拝 + 拍摄朱红山门 → 再向南步行 10 分钟到**坂下「神楽坂下交差点」+ 牛込桥**，在外堀通对岸看一眼饭田桥站夜景。沿途择一家和菓子老铺（推荐**紀の善**的抹茶 babaroa 或**梅花亭**的どら焼）购入 1–2 件作宵夜带回。**全段神乐坂停留约 60 分钟**（17:30–18:30 是兵库横丁灯笼亮起的黄金时段）。完成后从神乐坂坂上端原路返回到东京 Metro 神乐坂站，乘东西线一站到早稻田站接早稻田大学 4 号馆段；亦可直接从神乐坂坂道南端步行约 15 分钟穿过早稻田通到达早稻田大学西门（步行约 1.2 km，巷弄夜游 + 早稻田南町夏目漱石旧居方向，沿途可经过夏目坂 + 漱石公园外观打卡）。
- **票务 / 营业时间**：
  - 神乐坂街区 / 巷弄：24 小时开放、免费；街灯 / 街区夜景至 23:00 后明显冷清。
  - **毘沙门天善国寺**：境内 24 小时开放、免费；拝殿 06:00–17:00（朱印 / 御守 09:00–16:30）；定休按例祭日调整。
  - 各和菓子老铺营业时间约 09:00–19:00（紀の善因人气极高 17:00 售罄常见）；定休按各店家（多为周一 / 周二）。
  - **日仏学院（参考、本次仅外观）**：09:00–20:00（事务局周一闭）；不需入馆。
  - 营业时间 / 定休如有调整以官网或店头为准 —— 待用户实地确认。

- **注意事项**：
  - **巷弄需穿平底鞋**：兵库横丁 / 芸者新道 / 见番横丁等核心巷弄为细长石板路 + 小台阶 + 玄关阶差 + 部分湿滑黑塀基础；6 月梅雨季石板路湿滑率高，**禁穿高跟鞋 / 一字带凉鞋**，建议运动鞋 / 平底乐福鞋。
  - **巷弄白天 / 夜晚两面性**：白天巷弄空旷、看建筑 + 黑塀 + 石板纹理为主；傍晚 18:00 街灯亮起后转为暖黄街灯 + 小料亭 / 割烹门口红灯笼 + 偶有芸者来去（神乐坂仍有约 25 名现役艺者）的"小巴黎夜景"，是本街区的精华时段。**6/7 周日傍晚兵库横丁 / 芸者新道亮灯偏晚**（部分小料亭周日定休不点灯笼），需保留期待管理。
  - **巷弄拍摄礼仪**：兵库横丁 / 芸者新道两侧的小料亭 / 割烹是私人营业场所，**严禁对着店内 / 玄关 / 芸者特写拍摄**；拍黑塀 / 石板路 / 远景灯笼无问题，但勿用三脚架阻挡通行。
  - **6 月梅雨季巷弄无遮蔽**：兵库横丁 / 芸者新道两侧黑塀高 1.8–2.2 m，巷弄上空无遮挡，雨天需带折伞；坂道部分有商店遮阳棚但巷弄完全裸露。
