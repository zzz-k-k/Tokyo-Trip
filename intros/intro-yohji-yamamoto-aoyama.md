- **类型**：购物（山本耀司主线旗舰店 / 朝圣点；本次行程「就算不买也要去」的核心增量地点）
- **位置**：东京都港区南青山 5 丁目一带（青山 / 表参道商圈）；最近车站 东京 Metro 银座线 / 千代田线 / 半蔵门线「**表参道**」站 **B1 / A5 出口**步行约 3–7 分钟，或银座线「**外苑前**」站步行约 8 分钟（表参道站为 TST ✅）。**具体门牌号 / 当日营业以官网 SHOP INFO 为准**（官网：https://www.yohjiyamamoto.co.jp/）—— 待用户实地确认。
- **历史与背景**：山本耀司（Yamamoto Yōji，1943– ）是与川久保玲并肩的日本前卫时装大师，1981 年携 Yohji Yamamoto 登陆巴黎时装周，以解构剪裁 + 不对称 + 全黑廓形掀起被西方媒体称为「褴褛美学」的时装革命。青山 / 表参道一带是 Yohji Yamamoto 在东京的大本营 —— 这里聚集了主线 **Yohji Yamamoto POUR HOMME（男装）/ femme（女装）/ REGULATION** 的旗舰零售空间，店内空间多由山本本人参与设计，混凝土 + 黑铁 + 留白的极简构造本身就是其「间（ma）」美学的物理呈现。
- **重要变更提示（2026 最新）**：青山原「**Yohji Yamamoto 2**」已于 **2026 年 1 月 16 日改名为「LIMI feu 青山店」**（LIMI feu 是山本耀司之女山本里美的品牌）—— 即原本的二号店已转为 LIMI feu 专门店。**主线 Yohji Yamamoto 本店仍在青山 / 表参道一带运营**；前往前**务必用官网 SHOP INFO 或 Google Maps 确认当前主线本店的确切地址与营业状态**，避免走错到已改名的 LIMI feu 店。
- **看点**：
  1. **主线男女装秀场款全陈列**：POUR HOMME（男）/ femme（女）当季主线 —— 全黑廓形、解构剪裁、不对称下摆、超长大衣，是「秀场血统的那一件」的源头。
  2. **店铺空间本身**：山本参与设计的零售空间，混凝土 + 黑色 + 大量留白，是体验其「间」与「禅」美学的现场，**就算不买也值得进去站一会儿**。
  3. **REGULATION 线**：主线中相对基础 / 可日常的延伸，剪裁经典、装饰克制，价格门槛略低于纯秀场款。
  4. **店员的专业度**：青山本店店员对剪裁、面料、系列脉络如数家珍，可请其讲解一件衣服的设计语言 —— 这本身是一次「设计师态度」的近距离体验。
- **推荐玩法**：作为「朝圣 + 试穿」点，建议预留 30–45 分钟。进店后先整体感受空间 → 请店员介绍当季主线主推款 → 试穿 1–2 件外套 / 衬衫感受廓形与「间」→ 即便预算不允许买主线，也可考虑入手单价较低的衬衫 / 针织 / 配饰（discord 包 / 小物）作为「源头纪念」；或记下喜欢的款式，转去二手店（RAGTAG / Brooch）以更低价格寻找同款。**详细的产品线分梯队解析 + 退一步预算方案见专题文档**：[山本耀司：黑色美学与东京购物攻略](../山本耀司.md)。
- **票务 / 营业时间**：
  - 入场料金：免费进入；商品消费另计（主线外套 ¥150,000–400,000+ / 衬衫·针织 ¥50,000–120,000 / REGULATION 略低）。
  - 营业时间：青山区域门店多为 **11:00–20:00**（具体当日营业 / 定休按官网 SHOP INFO 为准）。
  - 免税：外国护照单店消费 ¥5,000+ 可办免税（消费税约 10%）。
- **交通到达**：
  - 从「表参道」站 B1 / A5 出口步行 3–7 分钟（TST ✅）。
  - 从两国 COGO RYOGOKU 出发参考：都营大江户线 → 青山一丁目 → 银座线一站到表参道（约 35 分钟、TST ✅）。
- **周边联动**：步行 8 分钟可达 **根津美术馆**（南青山 6-5-1，隈研吾设计，Day 5 已纳入）；步行可达 Comme des Garçons 青山 / 表参道 Hills（安藤忠雄）/ Prada 表参道（Herzog & de Meuron）等表参道建筑·时装群；整个南青山段可与 Day 5 的根津美术馆串成「青山时装 + 建筑」轴。
- **注意事项**：
  - **务必先确认主线本店地址**（区别于已改名的 LIMI feu 青山店）。
  - 主线价格门槛高；若预算有限，可只朝圣 + 试穿，转 Y's / Ground Y（涩谷 PARCO）或二手店（RAGTAG / Brooch）实际下单 —— 见专题攻略。
  - 部分门店对试穿主线高价款需店员陪同；店内一般允许拍空间但拍商品 / 他人需先询问。
  - 现金 / 主要电子支付（Suica / Visa / Mastercard / JCB / Apple Pay）皆可。
