- **类型**：商业街 / 御宅 / 电器
- **位置**：东京都千代田区外神田 1 丁目（JR 秋叶原站电器街口正面，中央通南端 + 昌平桥通沿线，约 200 × 200 米核心街区）；最近车站 JR 山手线 / 京浜东北线 / 总武线「秋叶原」站电器街口出站即入（步行 0 分钟）、东京 Metro 日比谷线「秋叶原」站步行 1 分钟、都营新宿线「岩本町」站步行 5 分钟。

## 历史与背景

秋叶原电器街的物质起点是 1949 年（昭和 24 年）11 月开业的"秋叶原ラジオセンター"——战后初期 GHQ 对市场实施严格管制，神田青果市场旁的 JR 秋叶原站高架下空间被改造为收音机零件 / 真空管 / 二手电气器材的露天集散地，由旧"东京电机大学"（前身电气工业讲习所）的师生 + 神田周边小商人共同经营。1950 年代随着家庭电气化（白电 = 电冰箱 / 洗衣机 / 电视机；黑电 = 收音机 / 唱机），电器街从"零件 + 真空管"扩张为"白电 + 黑电 + 零件"全谱系市场，1962 年（昭和 37 年）"ラジオ会館"以 8 层钢筋混凝土楼建成，是当时秋叶原最高的电器楼，与 1949 年的ラジオセンター + 1950 年的"东京ラジオデパート"合称"秋叶原三大无线塔"，至今仍在运营（2014 年ラジオ会館完成全栋翻新，1F-9F 几乎全为玩具 / 模型 / 中古玩具店）。1970–80 年代秋叶原电器街是日本"白电黑电"消费爆炸期的核心市场，年营业额一度占全日本家电零售的 10% 以上。1990 年代 PC 自作热潮使电器街从"白电"快速转型为"电脑 + 半导体 + 零件"，"あきばお〜""じゃんぱら""ソフマップ"等 PC 中古连锁在 1990 年代后期纷纷进入。2000 年代后半 Yodobashi-Akiba（2005 年开业，神田青果市场旧址，地上 9F + 地下 6F，约 27,000㎡，**日本单店面积最大的家电量贩**）的开业使老电器街业态进一步分化：老街区（中央通 + 昌平桥通）转向"零件 / 二手 / 御宅向器材"的细分长尾市场；Yodobashi 占据"主流家电 / 摄影 / 大件"流量。今日的秋叶原电器街仍保留 1950 年代以来的"高架下 / 楼内分割小商铺"独特业态，是全球最具识别度的"电子零件长尾市场"。

## 看点

1. **ラジオ会館（1962 / 2014 重建，9 层）**：秋叶原最具识别度的电器楼，1F-9F 几乎全为玩具 / 模型 / 中古玩具 / 海洋堂直营 / 卡牌店；北端神田明神方向通透；电梯口 / 阶梯标识"ラジオ会館"四字是经典拍照位。
2. **ラジオセンター（1949，高架下）**：JR 秋叶原站高架下东西约 60 米的露天 + 半室内零件商店群，约 30 家小店铺集中卖真空管 / 电阻 / 电容 / 焊接器 / 二手收音机；秋叶原电器街的"创世段"，今天仍以原始业态运营，是御宅 + 工程师双圈层"朝圣地"。
3. **东京ラジオデパート（1950，4 层）**：与ラジオセンター隔街相对，4 层独栋楼内集合 ~ 60 家小店；2F-3F 集中在电子零件 + LED + 模拟电路；4F 二手 / 复古元件；与ラジオセンター + ラジオ会館共构成"三大无线塔"。
4. **Yodobashi-Akiba（2005，地上 9F + 地下 6F）**：日本单店面积最大的家电量贩，**电器街口正东 100 米**；1F-2F 智能手机 + 笔记本 + 摄影器材；3F 数码相机 + 镜头 + 配件；4F-6F 白电 + 玩具 + 模型；7F-8F 餐厅 + 高尔夫 + 自行车；**国际游客凭护照 ¥5,500（含税）以上消费可免税**。
5. **Sofmap 总店 + Mandarake 总店 + Animate 秋叶原 + Don Quijote 秋叶原**：电器街边缘的 4 大御宅 / 中古零售连锁旗舰；与电器街共构成"PC + 模型 + 动漫"立体复合零售带。
6. **二手 / 中古机件店群**：「あきばお〜」「じゃんぱら」「ソフマップ中古」「BOOK・OFF 秋叶原店」—— 中古笔记本 + 中古单反 + 二手游戏机原装卡带的密集集散地。

## 推荐玩法

从 JR 秋叶原站电器街口出闸，正面 30 米即ラジオ会館入口，**先垂直爬完 ラジオ会館 1F-9F**（约 30 分钟，电梯上 + 楼梯下，每层快速过一遍）；下楼出门后转身南走 50 米过昌平桥通即到ラジオセンター（高架下）+ 东京ラジオデパート（隔街），两栋按"逛旧物市场"节奏看 30–45 分钟；最后进 Yodobashi-Akiba（电器街口正东 100 米）按需 30–60 分钟。**电器街整段最佳停留 90 分钟**，时间不足可省略 Yodobashi（与三大无线塔的"长尾御宅向"业态重合度低，不必非看）。

## 票务 / 营业时间

街区本身全天开放、全年免费、无入场限制；**ラジオ会館** 10:00–20:00（部分店 11:00 开业，全年无休）；**ラジオセンター** 10:00–19:00（部分店周日休 / 元旦休）；**东京ラジオデパート** 10:00–19:00（部分店周日休 / 元旦休）；**Yodobashi-Akiba** 09:30–22:00（全年无休）。

## 交通到达

- 从前一地点 秋叶原中央通：步行 0–3 分钟（与中央通南端连通）。
- 从两国 COGO RYOGOKU 出发：JR 总武线 两国 → 秋叶原（约 5 分钟、¥160、TST ❌）+ 步行 0 分钟（电器街口下车即到）；或都营大江户线 两国 → 上野御徒町 → 换 JR 山手线一站到秋叶原（共约 15 分钟、¥370、大江户线段 TST ✅ + JR 段 ❌）。

## 周边联动

步行 5 分钟内 `Akihabara`（中央通，本身覆盖电器街）、`Mandarake Complex`（北行 6 分钟）、`Pop Life Department M's`（电器街口正南步行 2 分钟）；构成"秋叶原电器 + 御宅"垂直街区。

## 消费档次

电器街本身免费；ラジオ会館内手办 / 模型 ¥1,500–50,000 / 件；ラジオセンター零件 ¥100–5,000 / 件；Yodobashi-Akiba 数码相机 ¥80,000–500,000 / 台；中古笔记本 ¥30,000–150,000 / 台；中古游戏卡带 ¥500–30,000 / 件；整段（仅看不买）¥0；如购入手办 / 中古机件人均 ¥10,000–100,000+。

## 注意事项

- **现金为主**：ラジオセンター + 东京ラジオデパート 内的小店多数仅收现金；ラジオ会館 + Yodobashi 全面支持信用卡 / Suica / Apple Pay。
- **价格谈判**：零件店 + 中古机件店允许小幅议价（约 5–10%），**整数额或捆包购买更易得到议价空间**；高价新品店（Yodobashi）价格固定不可议价。
- **闪光灯禁止**：ラジオ会館 + 东京ラジオデパート 内大多数店铺禁止闪光灯 + 禁止商品近距离拍摄（防仿冒 + 保护版权），拍店内全景前最好询问店员。
- **国际寄送**：Yodobashi-Akiba 提供国际邮寄服务（EMS / DHL，1F 服务台办理），结账时勾选"免税 + 海外寄送"；中小店一般不提供国际寄送，**需自己打包带回或委托当地集运**。
- **6 月梅雨季**：电器街老楼内地面易潮湿，**穿防滑鞋**；ラジオセンター 是高架下半室外，雨天会有少量飘雨。

## 购物专属字段

- **主营商品类目**：电子零件（真空管 / 电阻 / 电容 / IC / LED）+ 二手电脑（笔记本 / 主板 / GPU / 显卡）+ 摄影器材（数码相机 / 镜头 / 三脚架）+ 半导体 + 中古游戏机（任天堂 / 索尼 / 世嘉历代主机 + 卡带）+ 玩具模型（GK / 手办 / 假面骑士 / 战队腰带）+ 卡牌 + 海洋堂周边。
- **价格区间**：零件 ¥100–5,000 / 件；中古笔记本 ¥30,000–150,000 / 台；新数码相机 ¥80,000–500,000 / 台；中古游戏卡带 ¥500–30,000 / 件；手办 / 模型 ¥1,500–50,000 / 件。
- **是否支持免税**：**Yodobashi-Akiba** 支持（凭护照 ¥5,500+ 即时退税）；**ラジオ会館 内连锁店**（如海洋堂直营 / Animate 系）多数支持免税；**ラジオセンター + 东京ラジオデパート 内小店一般不支持免税**（小店未注册免税商户 / 不接受护照核验）。
- **是否支持寄国际**：**Yodobashi-Akiba** 支持 EMS / DHL 国际邮寄（1F 服务台办理）；ラジオ会館 内**部分连锁店**（海洋堂 / Animate / Surugaya）支持；ラジオセンター + 东京ラジオデパート 内小店**绝大多数不支持国际寄送**，需自带或委托集运。
