- **类型**：餐饮（**江户荞麦 / "薮系"老铺，1913 年创业**）

- **历史与背景**：並木藪蕎麦（Namiki Yabusoba）是 **1913 年（大正 2 年）由堀田胤雄（後の藪一族）**在浅草雷门西侧並木通创业的"**江户三大薮系（藪 / Yabu）老铺**"之一，与 神田藪蕎麦（神田淡路町，1880 年创业）、池之端藪蕎麦（上野，1953 年从分家独立）合称"**江户三大薮**"——这"三大薮"都源自 19 世纪江户期(江戸時代)後期的"**團子坂藪蕎麦（だんござかやぶそば）**"（团子坂的藪），即东京最早的"薮系"江户荞麦本家；19 世纪末该本家分家后，并木薮（堀田家）+ 神田薮（堀田家分家）+ 池之端薮（神田薮分家）三家在东京形成了"薮系"在地化网络。"**薮系（Yabu / 藪）**"作为江户三大荞麦流派之一（**砂场（Sunaba）/ 更科（Sarashina）/ 薮（Yabu）**），其特色是**深黑色 + 极辛辣 + 极浓郁的蕎麦汁（Tsuyu / つゆ）**——薮系的汁底以**酱油 + 味醂 + 鲣鱼出汁 + 砂糖**长时间熬制成深黑色，盐分 + 旨味 + 咸辣浓度都比"砂场系"或"更科系"高一倍以上，因此**薮系荞麦的食用礼仪要求"只蘸蘸 15–20%"**——把面条夹起后**只把面条底部 15–20% 的部分轻蘸蘸汁就迅速送入口中**，**严禁把整根面条浸入蘸汁**——如果整根面浸入，盐分 + 浓度会**完全压过荞麦本身的清香味，"系统平衡"会被打破**。这种"克制的食用协议"是江户老铺薮系传承百年的核心仪式之一，也是江户(下町)文化"侘び寂び（侘寂）+ 食物自尊"的代表表达。並木藪蕎麦的店铺由堀田家从 1913 年延续至今约 110 年，店内仍保留大正期 + 昭和初期的木造店铺氛围（榻榻米 + 矮桌 + 暖簾 / Noren / のれん），是浅草雷门一带"江户老铺集中区"的代表小铺。代表餐品 **ざるそば（Zaru Soba / 竹筛凉面）¥800–1,000**、**もりそば（Mori Soba）¥800**、**かけそば（Kake Soba 热汁面）¥800**、**天ぷらそば（天妇罗荞麦）¥1,800–2,000**——分量极简（**老江户人习惯把它当作下午茶 / 酒后点心**），即使点上两份预算也只要 ¥2,000 左右。
- **看点**：
  1. **1913 年大正期创业的店铺**：浅草雷门西侧並木通沿线，木造店面 + 暖簾 + 榻榻米 + 矮桌的江户老铺氛围。
  2. **深黑色极辛辣极浓郁的"薮系蘸汁（Tsuyu）"**：江户三大薮的代表汁底，浓度比一般荞麦店高一倍以上。
  3. **「只蘸 15–20%」食用礼仪**：百年传承的食用协议，把面条底部 15–20% 轻蘸即送口；禁止整根浸入。
  4. **基础 ざるそば（Zaru Soba 竹筛凉面）¥800–1,000**：代表餐品，分量极简。
  5. **天ぷらそば（天妇罗荞麦）¥1,800–2,000**：江户三大薮的另一代表餐品，加炸虾天妇罗。
  6. **浅草雷门西侧地理位置**：从店铺步行 1 分钟即雷门 / 仲见世通起点，方便结合浅草寺其他保留地点（已纳入 Day 7 / Day 4 等其他日）。
