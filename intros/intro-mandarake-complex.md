- **类型**：购物 / 御宅（手办 + 同人志 + 中古 CD/DVD + 偶像周边的 8 层综合店）
- **位置**：东京都千代田区外神田 3-11-12（中央通北段，距 JR 秋叶原电器街口步行 6 分钟、东京 Metro 银座线「末广町」站 3 番出口步行 1 分钟、JR「御茶水」站圣桥口步行 8 分钟）；建筑 8 层（地上 8F + B1）+ 漆黑外墙是中央通最具识别度的地标之一。

## 历史与背景

Mandarake（まんだらけ）创业于 1980 年（昭和 55 年）2 月，由古川益藏（社长，本人也是漫画收藏家 + 同人志研究者）在中野百老汇 4F 开第一家中古漫画书店—— 这家店至今仍在中野百老汇 2F-4F 散布的 30 余家"门""大怪兽""Galaxy"等 Mandarake 系主题分店组成（详见 Day 4 中野段 `Mandarake Nakano`）。1990 年代随着同人志 + 中古玩具 + 御宅消费爆炸，Mandarake 从中野扩张到涩谷（1994）、福冈（1995）、名古屋（1997）、大阪（1998）、札幌（2002）等城市，2005 年股票上市（东证 JASDAQ），是日本第一家上市的"御宅 / 中古"专业连锁。秋叶原 Complex 总店于 **2007 年 9 月开业**，地址是中央通北段外神田 3-11-12 的 8 层独栋楼，**整栋外墙涂满漆黑色 + Mandarake 红字招牌**，是中央通最具识别度的"御宅地标"。秋叶原店与中野店在选品上有意分化：**中野店保留了"古董玩具 + 怪兽软胶 + 怀旧游戏卡带 + 复古漫画"的复古御宅取向**（适合 30 岁以上的怀旧御宅）；**秋叶原 Complex 总店则更偏向"现代御宅族 + 国际游客"** —— 低楼层（B1-2F）是手办 / 超合金 / 假面骑士 / 战队变身腰带，中楼层（3-5F）是同人志 + 中古漫画，高楼层（6-8F）则逐渐深入到中古 CD/DVD / 偶像周边 / 同人音声 / 贩售コーナー。"垂直电梯一上一下，就是两个完全不同的世界"是 Mandarake 秋叶原 Complex 的设计语言。今日 Mandarake 秋叶原 Complex 是日本中古御宅市场的最高单店，年访客量百万级，国际游客（含中国大陆 / 香港 / 台湾 / 北美 / 欧洲）占比约 40%。

## 看点

1. **漆黑外墙 + Mandarake 红字招牌**：建筑外观本身是中央通最具识别度的地标，**8 层涂黑 + 大窗 + 红字"まんだらけ"** 在白天 / 夜间都极易辨认，是合影必拍点。
2. **B1-2F 手办 / 超合金 / 假面骑士 / 变身腰带**：中古手办（古明地觉 / 高达 / Re:Zero / 鬼灭等）+ 玩具（万代 / 海洋堂 / Goodsmile）+ 假面骑士 / 战队的变身腰带与小道具。
3. **3-5F 中古漫画 + 同人志 + 画集**：同人志（C 系列各次 Comiket 现场刊）+ 中古单行本 + 复刻版漫画 + 设定集 + 画集；3F"少年漫画"4F"少女 / 青年漫画"5F"同人志"分区。
4. **6-7F 中古 CD/DVD / 偶像周边 / Hello! Project / AKB48 / 杰尼斯**：偶像团体的演唱会 DVD / 限定 CD / 周边 / 写真集 / 复古 7 寸黑胶。
5. **8F 透明天井 + Mandarake 商标 + 偶像 / 同人音声 / 贩售コーナー**：8F 顶楼有玻璃天井 + Mandarake 商标 + 部分小众选品（同人音声 / Voice Drama / 18 禁选品集中区）；从 8F 透过玻璃天井可见中央通天空。

## 推荐玩法

从 JR 秋叶原电器街口北行约 6 分钟（沿中央通），到达外神田 3-11-12 的漆黑外墙建筑—— **先在外墙正面拍合影**（最佳机位是从中央通对面斜 45 度拍，能完整看到 8 层立面 + Mandarake 红字招牌）；进店后**搭电梯直上 8F → 楼梯逐层下行**（这是 Mandarake 推荐的标准动线，从最小众到最大众，避免在低楼层就买完手办之后没力气上楼）；每层 5–10 分钟快速过一遍，遇到心仪商品再回头细看；**总停留 60–90 分钟**（深度逛可 ≥ 2 小时）；结账在 1F 或 B1F 收银台，**国际游客凭护照可免税**。

## 票务 / 营业时间

入店免费、无入场限制；**营业时间 12:00–20:00**（**注意：Mandarake 全连锁中午 12 点才开店**，比秋叶原其他御宅店晚 1–2 小时，不要 11:00 就来扑空）；**全年无休**（仅 1/1 元旦休业）；**部分商品限定贩售期 / 抽选购买**会另行告示。

## 交通到达

- 从前一地点 秋叶原电器街：沿中央通北行步行 6 分钟，免费；或东京 Metro 银座线 末广町站 3 番出口直接出站步行 1 分钟即到。
- 从两国 COGO RYOGOKU 出发：JR 总武线 两国 → 秋叶原（约 5 分钟、¥160、TST ❌）+ 沿中央通北行步行 6 分钟；或都营大江户线 两国 → 上野御徒町 → 换东京 Metro 银座线一站到末广町（共约 12 分钟、¥390、大江户线段 ✅ + 银座线段 ✅，**TST 全覆盖**）+ 步行 1 分钟。**推荐 TST 全覆盖路线**。

## 周边联动

步行 5 分钟内 `Akihabara`（中央通，本身覆盖 Mandarake）、`Akihabara Electric Town`（电器街口南行 6 分钟）、`Pop Life Department M's`（电器街口正南步行 8 分钟）。

## 消费档次

入店免费；中古手办 ¥1,500–80,000 / 件；中古漫画 ¥100–3,000 / 本；同人志 ¥300–5,000 / 本；偶像周边 ¥500–30,000 / 件；中古 CD/DVD ¥500–10,000 / 件；整段（仅看不买）¥0；如深度购入人均 ¥5,000–80,000。

## 注意事项

- **店内禁止拍照 / 录像**：B1-8F 全栋禁止对商品 / 货架 / 收银 / 其他客人的拍摄（防止仿冒 + 保护版权 + 保护客人隐私）；建筑外墙合影 OK，**进入店门后请把手机相机收起**。
- **结账规则**：选品后到 1F 或 B1F 主收银台结账（每层不设独立收银台）；现金 / 信用卡 / Suica / Apple Pay 均支持；**护照在结账时一次性出示**，免税商品现场扣税开免税单。
- **18 禁选品 / R-18 同人志**：8F 部分区域为 18 禁，入口有清晰告示，**未成年人禁止进入**；进入需自助确认 18+。
- **国际游客寄送**：Mandarake 提供"店铺直邮国际"服务（EMS / DHL），结账时勾选并填写收件地址，免税商品需带回原国 / 地区 30 天内完成报关；**寄送费 + 包装费另收**（小件 ¥3,000–5,000、大件 ¥8,000–15,000）。
- **价格谈判 / 议价**：中古商品价格已含书 + 标签，不接受议价；但偶有"30% off / 50% off"促销贴纸，可向店员询问。
- **6 月梅雨季 + 雨天**：店内有空调 + 干爽，店外漆黑外墙 + 黑色不锈钢台阶在雨天易反光易滑，**穿防滑鞋 + 进店前在门口刮鞋**。

## 购物专属字段

- **主营商品类目**：中古手办 + 中古漫画 + 同人志 + 偶像周边（AKB48 / 杰尼斯 / Hello! Project 等）+ 中古 CD/DVD + 中古玩具（假面骑士 / 战队变身腰带）+ 18 禁选品（8F 限定）。
- **8 层垂直分区**：B1-2F 手办 / 玩具 / 变身腰带；3F 少年漫画 + 中古单行本；4F 少女 / 青年漫画；5F 同人志；6F 中古 CD/DVD；7F 偶像周边 + Hello! Project / AKB48 / 杰尼斯；8F 同人音声 + 18 禁选品 + 顶楼玻璃天井。
- **价格区间**：中古手办 ¥1,500–80,000 / 件；中古漫画 ¥100–3,000 / 本；同人志 ¥300–5,000 / 本；偶像周边 ¥500–30,000 / 件；中古 CD/DVD ¥500–10,000 / 件。
- **是否支持免税**：**支持**——国际游客凭护照、当日同店消费 ≥ ¥5,500（含税）即时退税；现场扣税。
- **是否支持寄国际**：**支持**——Mandarake 直邮国际（EMS / DHL），结账时填写收件地址；寄送费 + 包装费另收（小件 ¥3,000–5,000、大件 ¥8,000–15,000）。
