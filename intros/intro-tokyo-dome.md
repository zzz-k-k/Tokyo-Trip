- **类型**：体育场 / 演出（日本第一座空气支撑式屋顶球场，读卖巨人主场，容量 4.6 万）
- **位置**：东京都文京区后乐 1-3-61（Tokyo Dome City 内，水道桥 / 后乐园站之间）；最近车站 都营三田线「水道桥」站 A1 / A2 出口步行 3 分钟、JR 总武线 / 中央线（各停）「水道桥」站东口步行 5 分钟、东京 Metro 丸之内线 / 南北线「后乐园」站 2 番 / 5 番出口步行 5 分钟、都营大江户线「后乐园」站 5 / 6 番出口步行 5 分钟、都营大江户线 / 都营三田线「春日」站步行 7 分钟。

## 历史与背景

东京巨蛋（俗称"BIG EGG"或"东京ドーム"）于 1988 年（昭和 63 年）3 月 17 日开馆，由 株式会社东京巨蛋（东京ドーム株式会社）运营，与同年同期开馆的"东京巨蛋城（Tokyo Dome City）"游乐设施 + LaQua 温泉 spa 综合体共构"日本第一座大型室内复合娱乐设施"。东京巨蛋接替的是 1937 年（昭和 12 年）开馆的"后乐园球场"——后乐园球场是日本职棒草创期的圣地，读卖巨人队从 1937 年起即以后乐园为主场，1950 年首届"日本シリーズ"（日本系列赛）也在此举办；1987 年后乐园球场拆除，1988 年东京巨蛋在毗邻用地开馆，**巨人队主场身份延续至今未中断**。东京巨蛋是**日本第一座空气支撑式屋顶（Air-Supported Roof）球场**，屋顶由 0.8mm 厚的 Teflon 涂层玻璃纤维布构成、外部覆土 + 内部以 0.3% 大气压差吹胀维持形态，无任何梁柱支撑，跨度 201m × 144m × 高 56m，是当时世界最大跨度的"气垫顶"，技术参考自美国密歇根 Pontiac Silverdome（1975）但跨度更大。容量约 4.6 万人（棒球赛 46,000 / 演唱会模式 55,000）。1988 年开馆同年还落成"巨人队球员通道（Players Tunnel）+ V9 通道（V9 = 1965–1973 年巨人队连续 9 年日本一）+ 长嶋茂雄 / 王贞治名球会博物馆"，把 1950 年代后楽園期的巨人队历史完整迁入。**6 月 5 日（周五）16:00 开场、18:00 比赛开始的「读卖巨人 vs 千叶罗德海洋」**是日本职棒**セ・パ交流戦**（中央联盟 vs 太平洋联盟跨联盟交流赛，2005 年起每年 5 月底–6 月中举办）的固定赛事之一——这场对决是巨人队（中央联盟，东京）对阵千叶罗德海洋队（太平洋联盟，千叶），交流戦战绩计入各队联盟排名；本场的票务 / 入场口分配按"内野指定 11–18 号 + 外野自由 22 号"标准分配。东京巨蛋之外，整片"东京巨蛋城"还包括 Tokyo Dome Hotel（1996，丹下健三设计）+ Thunder Dolphin 过山车 + Big O 摩天轮（中心无轴的世界第一款）+ LaQua 温泉 spa + Hot Dog Press（1988 年同期开业的复合美食街，已部分翻新），构成 1988–今日的"巨蛋城娱乐综合体"。

## 看点

1. **空气支撑屋顶（Air-Supported Roof）**：日本第一座、世界第二大跨度的气垫顶（仅次于 Pontiac Silverdome），从外野座席仰头看屋顶光线透过半透明玻璃纤维布的视觉效果是东京巨蛋独有的"室内白昼感"；雨天屋顶不受影响，6 月梅雨季外面下雨内部完全无影响。
2. **球场内饮食（巨蛋名物）**：**ヨネポンチキン（米粉炸鸡）**¥800–1,200 / 份是巨人队主场限定名物、**巨蛋啤酒（生ビール）**¥800 / 杯由"啤酒妹"（売り子）背着背筒巡回兜售（日本职棒文化标志，看一场比赛建议至少买 1 杯支持）、其他还有"巨人队限定弁当 ¥1,500"、"东京ドームバーガー ¥900"、"啤酒妹特别版 + 巨人队应援毛巾 ¥1,500 套餐"。
3. **巨人队应援团（外野指定席）**：外野席（含 22 号入场口对应的外野自由席）由"巨人軍応援団"（巨人队应援团）领衔，全场齐声唱《闘魂こめて》（巨人队队歌）+ 球员个人应援曲（如冈本和真应援曲）+ 7 回攻防时齐放气球（ラッキーセブン），**这是日本职棒文化的核心场景**，外野自由席的应援氛围是内野指定席无法替代的。
4. **7th Inning Stretch（七回伸展时段）**：日本职棒沿袭美职棒的"七回攻防开始前观众起立伸展"传统；东京巨蛋在 7 回里头放气球（黄色橡胶气球，1 杯啤酒 ¥800 附 1 个，散场后清扫人员收回 5 万个气球是巨蛋的标志性画面）；建议保留 1 个气球做纪念。
5. **巨人队博物馆 + V9 通道（22 号入场口前侧厅）**：22 号入场口前侧厅设小型展厅，展长嶋茂雄 / 王贞治 1965–1973 V9 时期的纪念球衣 + 1958 年长嶋茂雄入团仪式照片 + 王贞治第 868 本垒打用球（1977 年），赛前抵达后可顺道参观 5–10 分钟。
6. **Tokyo Dome City 复合体（散场前后顺路）**：Big O 中心无轴摩天轮 + Thunder Dolphin 过山车从巨蛋外野席外环可见、夜间灯光秀 18:00–22:00；LaQua 温泉 spa（24 小时营业）就在巨蛋北侧，散场后想泡温泉缓解一天疲劳可顺道入浴 ¥3,230 / 人（深夜加成另算）。

## 推荐玩法

- **16:00 开场即入场**：从 22 号入场口检票入内，先在外野自由席外环找到自己的入场口对应区域（22 号入场口对应外野左 / 右翼自由席），把背包放好；接着去 1F 食品摊买 1 份ヨネポンチキン + 1 杯巨蛋啤酒（共约 ¥1,800），回到外野席座位边吃边等。
- **16:30 拍场内全景 + 屋顶**：吃完后趁人不多去外野席最高点拍内野方向全景（从外野往本垒方向看，正面是巨人队应援团 + 大屏），同时抬头拍空气支撑屋顶的半透明光线效果。
- **17:00 外野自由席占位**：外野自由席（22 号入场口对应）**先到先得**，**17:00 前必须返回座席占位**——17:00 后应援团核心区前排会被巨人軍応援団先占，外野其他位置仍可坐。
- **17:55 国歌**：18:00 开场前 5 分钟全场起立唱日本国歌《君が代》，外国游客可保持安静站立致意（不必跟唱）。
- **18:00 开打**：巨人队主场（先攻 / 后攻按主客场规则，主场巨人是后攻）；7 回上半结束、7 回下半开始前是"7th Inning Stretch + 气球放飞"——保留 1 个气球做纪念。
- **21:00 末半**：约 21:00 进入 9 回（最终回），21:30 前后比赛结束（9 局制 + 棒球赛通常 3 小时整）；如有平局则进入"延长（最多 12 回）"，但交流戦自 2018 年起取消延长（12 回打完按平局结束），所以**21:30 散场是稳定时刻**。
- **21:30 散场**：散场后**避开主流人潮**——主流人潮（约 70%）走 JR 水道桥东口 + 都营三田线水道桥 A1 出口，**反方向走北侧 LaQua → 都营大江户线后乐园站**可大幅减少排队（详见上文段内导览的回程指引）。

## 票务 / 营业时间

- **6 月 5 日比赛日开场时间**：**16:00 开场（开门 / 入场口）**、**18:00 比赛开始**、**约 21:30 散场**；外野自由席先到先得不指定座位，内野指定 / 特别席按票面分配。
- **票价区间**（巨人主场标准价；具体场次以官网为准）：
  - **外野自由席（自由席）**：¥2,000 / 张；先到先得不指定座位，**对应入场口 22 号 + 23 号**；6/5（周五）交流戦的外野自由席约 17:00 满席。
  - **内野指定席（B 席 / C 席）**：¥4,500–6,500 / 张；分内野 1 / 2 / 3 垒侧；**对应入场口 11–18 号**（按票面 11 / 12 / 13 / 14 / 15 / 16 / 17 / 18 号分配）。
  - **特别席（SS 席 / S 席 / プレミアム席）**：¥10,000–15,000+ / 张；本垒后侧 + 1 垒后侧最佳视角；**对应入场口 11–13 号 + 后侧 VIP 通道**。
- **票务网址**：
  - **巨人公式（Giants Ticket）**：[https://www.giants.jp/](https://www.giants.jp/) —— 读卖巨人队官方售票（提前 1 个月开售，会员优先）；中文 / 英文界面有限，建议借助翻译插件。
  - **チケットぴあ（Ticket Pia）**：[https://t.pia.jp/](https://t.pia.jp/) —— 第三方综合售票，巨人队主场比赛同步上架。
  - **东京ドーム 官方**：[https://www.tokyo-dome.co.jp/](https://www.tokyo-dome.co.jp/) —— 含场馆信息 + 当日座席图。
  - **票务以官网为准 —— 待用户实地确认 / 提前 1 个月购票**。
- **入场口（按票面分配，共 25 个入场口）**：
  - **22 号入场口（外野自由席专用）**：本次推荐入场口；位于巨蛋西侧，从都营三田线 A1 出口或 JR 水道桥东口最快到达。
  - **11–18 号入场口（内野指定席）**：按票面具体号分配；11–13 号在本垒后侧 + 1 垒侧；14–15 号在 1 垒外野侧；16–17 号在 3 垒侧；18 号在 3 垒外野侧；从都营三田线 A1 出口入场后沿外环顺时针 / 逆时针走到对应号。
  - **21–25 号入场口（外野指定席 + 自由席分配）**：21 号外野 1 垒侧、22 号外野自由（西）、23 号外野自由（东）、24–25 号外野 3 垒侧指定。
  - **VIP / 特别席通道**：与主流入场口分离，凭 VIP 票从 11 号或后侧专用门入场。
- **检票流程**：
  1. **电子票（QR 码）**：手机出示 QR 码 → 入场口扫码闸机 → 扫码成功后过闸 → **同时进行随身物品安检**（小背包打开示意 + 大背包过 X 光台）；电子票 1 张仅可 1 次过闸，不可分发给同行人。
  2. **纸票**：纸票 → 闸机插入 / 工作人员人工检票 → 票面被打孔 / 撕角 → **同时进行安检**；纸票丢失不补办，请妥善保管。
  3. **安检**（与检票同步进行）：所有入场者**禁带物品需先在闸机外丢弃 / 退场寄存**（详见下文）；金属 / 大型水瓶 / 玻璃容器会被当场没收。
- **禁带物品**（违反将被当场没收 / 退场）：
  1. **500 ml 以上瓶装饮料**：所有 500 ml 以上的瓶装水 / 茶 / 饮料**禁止携入**；500 ml 以下的塑料瓶水 / 茶可入（但**玻璃瓶 / 金属罐绝对不可**——即使 350 ml 也不行）；解决方法：场内有自动售货机 + 啤酒妹 + 食品摊；或在入场前喝完。
  2. **玻璃瓶 / 金属罐**：所有玻璃容器（红酒 / 啤酒玻璃瓶）+ 金属罐（罐装啤酒 / 可乐）**绝对不可入场**——违反者球场观众席投掷玻璃瓶有伤人致死的历史先例，全日本职棒球场严格执行。
  3. **自拍杆 / 三脚架 / 大型摄影器材**：自拍杆 + 三脚架 + 大型变焦镜头（焦距 ≥ 200mm）**禁止携入**；普通手机 / 小型相机 / 短焦镜头可入，**但禁用闪光灯**（赛中干扰球员）。
  4. **大型背包 / 行李箱**：超过 50cm × 40cm × 30cm 的背包 / 行李箱**禁止携入**——可寄存到 Tokyo Dome 周边的「水道桥站投币柜」（¥400–700 / 日）或东京 Dome City 内的人工寄存（¥800 / 日）。
  5. **横幅 / 应援道具未事先申请**：自制横幅 / 大型应援旗（≥ A2 尺寸）+ 大型加油棒**需赛前 7 日向巨人队官方申请**（[https://www.giants.jp/](https://www.giants.jp/) 相关流程）；未申请的不得入场。**普通的应援毛巾 / 巨人帽 / 球员名字毛巾在场内随意 OK**。
  6. **烟花 / 爆竹 / 危险物品**：所有烟花 / 爆竹 / 焰火 / 棒球之外的金属棒**绝对禁止**。
  7. **录音录像设备非新闻持有者**：专业录音笔 / 摄像机（≥ 手持摄像机级别）**仅限持新闻 / 媒体记者证的持有者**；普通手机录像 / 30 秒以内 SNS 短视频在不影响其他观众的前提下默许，但**禁止全场录像** + 禁止上传赛事完整影像（巨人队 / NPB 版权）。

## 交通到达

- 从前一地点 Pop Life Department M's（秋叶原电器街口南）：步行 5 分钟到都营新宿线「岩本町」站 A3 出口 → 都营新宿线 岩本町 → 神保町（约 4 分钟）→ 同站换都营三田线 神保町 → 水道桥（约 3 分钟）→ A1 出口步行 3–5 分钟即到 22 号入场口；全程 ~ 12 分钟、票价 ¥220（**TST 全覆盖 ✅**，**推荐**）；或 JR 总武线 + 中央线（御茶水换乘）从秋叶原 → 水道桥（约 6 分钟、¥150、**TST ❌**）。
- 从两国 COGO RYOGOKU 出发：都营大江户线 两国 → 上野御徒町 → 步行 / 换乘到秋叶原 → 都营新宿线 + 都营三田线如上；或都营大江户线 两国 → 后乐园（约 12 分钟、¥220、TST ✅）+ 步行 5 分钟到 22 号入场口（**最直接路线**——若 Day 3 因故跳过子区域 1 / 2，可直接走此线 16:00 抵达巨蛋）。**散场后回程**：都营大江户线 后乐园 → 两国（约 12 分钟、¥220、TST ✅）。

## 注意事项

- **雨天屋顶不受影响 / 6 月梅雨季外面下雨内部完全无影响**：东京巨蛋是空气支撑式屋顶，全封闭室内空间，雨 / 雷电 / 台风 / 暴雨**对赛事零影响**；6 月梅雨季外面下雨可能影响入场前的步行段，**带折叠伞 + 防水鞋**，但入场后完全干爽。
- **散场后人潮 30 分钟**：21:30 散场后，水道桥 / 后乐园 / 春日 三个地铁站会同时涌入约 4 万人，**水道桥 JR 东口 + 都营三田线 A1 出口的安检 + 闸机排队约 15–25 分钟**；**反方向走北侧大江户线后乐园站可避开 70% 人潮**（前文已详述），是本日散场归途的关键技巧。
- **应援文化基本礼仪**：巨人队主场观众席的"巨人軍応援団"齐声应援是核心文化场景，**外国游客可跟节奏鼓掌 / 摇毛巾 / 跟唱英文部分**，但**不要在 7 回攻防的"巨人队进攻段"喊罗德队球员名字 / 反方应援**——主场氛围会被打断。
- **拍照 / 录像基本规则**：场内随意拍照 OK（**禁用闪光灯**——赛中干扰球员、夜间影响转播灯光）；30 秒以内 SNS 短视频 OK；**严禁全场录像 + 严禁上传赛事完整影像**（NPB / 巨人队版权）；7 回气球放飞时举手机录像 OK。
- **周一定休 / 比赛日规则**：东京巨蛋本身**全年无休**（每天都有比赛 / 演唱会 / 游乐园营业），但**比赛日（如 6/5）部分外延设施 LaQua / Big O 摩天轮可能因人流管制部分时段限流**；散场后 21:30 的 LaQua 温泉仍正常营业（24 小时）。
